## 28.1 甲狀腺切除術

| 項目 | 內容 |
|-----|------|
| 程序 | 甲狀腺全部或部分切除 |
| 時間 | 1-2小時（取決於複雜程度） |
| 疼痛 | +/++ |
| 體位 | 肩枕和頭環；頭高腳低位 |
| 失血 | 通常最少；如果延伸至胸骨後則可能大量 |
| 實用技術 | IPPV + 加強型ETT |

### 一般考量

| 考量 | 內容 |
|-----|------|
| 複雜程度 | 從甲狀腺結節移除到解除氣管壓迫的胸骨後甲狀腺切除 |
| 胸骨後甲狀腺腫 | 通常通過標準切口切除，但偶爾需要胸骨切開術 |
| 損傷風險 | 喉返神經和甲狀旁腺可能受損或被切除 |

### 術前考量

| 考量 | 內容 |
|-----|------|
| 甲狀腺功能 | 確保患者盡可能接近甲狀腺功能正常 |
| 甲亢並發症 | AF、心動過速、眼球突出 |
| 急性準備 | 包括碘和糖皮質激素（抑制T4轉化為T3） |
| 氣道評估 | 檢查氣管偏移和狹窄；CT掃描準確描述氣道侵犯程度 |

### 氣道規劃選項

| 選項 | 內容 |
|-----|------|
| 吸入誘導 | 七氟醚（用於有喘鳴和疑似困難上氣道的患者） |
| 光纖插管 | （見相關頁面） |
| SGA | 在有喉部偏移的患者可能難以放置 |
| 氣管切開術 | 在LA下（僅在易於在阻塞水平以下進行時可能） |
| 硬支氣管鏡 | 通過嘗試通過ETT失敗時的備份選項 |

### 圍手術期考量

| 考量 | 內容 |
|-----|------|
| 眼保護 | 墊和潤滑尤其重要（如果有眼球突出） |
| 肌肉鬆弛 | 手術期間氣管操作可能非常刺激；使用充分鬆弛或remifentanil |
| 監測 | 喉返神經的電生理監測越來越常見（使用專用加強型ETT） |
| 固定 | 用膠帶牢固固定ETT（避免頸部綁帶） |
| 拔管 | 患者坐起以減少靜脈壓迫；使用盡量減少咳嗽的拔管技術 |

### 術後併發症

| 併發症 | 內容 |
|--------|------|
| 術後喘鳴 | 出血（頸部緊張腫脹）、氣管軟化、雙側喉返神經麻痹 |
| 低鈣血症 | 甲狀旁腺切除術後（見章節27-713） |
| 甲狀腺危象 | （見相關頁面） |
| 氣胸 | 如果有胸骨後解剖可能 |

## 28.2 甲狀旁腺切除術

| 項目 | 內容 |
|-----|------|
| 程序 | 單個腺瘤或四個腺體切除（增生） |
| 時間 | 1-3小時 |
| 疼痛 | +/++ |
| 體位 | 肩枕和頭環；頭高腳低位 |
| 失血 | 通常最少 |
| 實用技術 | IPPV + ETT |

### 一般考量

| 考量 | 內容 |
|-----|------|
| 適應症 | 原發性甲狀旁腺功能亢進（來自甲狀旁腺腺瘤） |
| 高鈣血症 | 通常；中度升高需在術前糾正 |

### 高鈣血症的管理

| 水平 | 管理 |
|-----|------|
| >3 mmol/L | 術前糾正 |
| 液體 | 1小時內1 L 0.9%氯化鈉，然後24小時4-6 L |
| 帕米膦酸鹽 | 60 mg在500 mL 0.9%氯化鈉中超過4小時 |

### 圍手術期考量

| 考量 | 內容 |
|-----|------|
| 電解質 | 監測鎂²⁺、磷酸鹽和K⁺ |
| 鎮痛 | 不嚴重（特別是有LA浸潤或淺表頸叢阻滯） |

### 術後考量

| 考量 | 內容 |
|-----|------|
| 鈣離子 | 6小時和24小時檢查 |
| 低鈣血症 | 可能發生（見前述甲狀腺切除術部分） |
| 喉返神經損傷 | 如有懷疑進行光纖鼻內鏡檢查 |

## 28.3 嗜鉻細胞瘤

| 項目 | 內容 |
|-----|------|
| 程序 | 切除一個或兩個腎上腺或腎外腫瘤 |
| 時間 | 1-2小時開放；腹腔鏡可能更長 |
| 疼痛 | +/++（取決於開放或腹腔鏡） |
| 體位 | 開放為仰臥或側臥；腹腔鏡為側臥 |
| 失血 | 可變 |
| 實用技術 | IPPV + ETT；arterial和CVP線± CO監測 |

### 腫瘤特點

| 特點 | 內容 |
|-----|------|
| 分泌 | 去甲腎上腺素（常見）、腎上腺素或多巴胺（最少見） |
| 位置 | 99%在腎上腺；10%為雙側；可沿交感神經鏈任何位置 |
| 遺傳關聯 | MEN2A、MEN2B（甲狀腺髓樣癌、副甲狀旁腺腺瘤） |

### 呈現

| 症狀 | 內容 |
|-----|------|
| 高血壓 | 持續、間歇或不明顯 |
| 典型三元組 | 心悸、出汗、頭痛和高血壓 |
| 急性呈現 | 肺水腫、心肌梗塞、腦血管事件 |

### 術前準備

| 準備 | 內容 |
|-----|------|
| α-阻斷 | 通常先使用，然後如有需要使用β-阻斷劑 |
| 藥物選擇 | 苯氧苯扎明（非競爭性、非選擇性α-阻斷劑）；然後使用阿替洛爾/普萘洛爾/美托洛爾 |
| 術前阻斷目標 | 血壓 <140/90 mmHg，心率 <100 bpm |

### 圍手術期管理

| 階段 | 管理 |
|-----|------|
| 誘導 | 避免釋放組胺的藥物（使用propofol、alfentanil/remifentanil和vecuronium/rocuronium） |
| 低血壓 | 可能需要麻黃鹼、間羥胺或苯腎上腺素 |
| 高血壓 | 可用鎂²⁺（2-4 g bolus）、remifentanil、酚妥拉明、硝普鈉或拉貝他洛爾 |
| 腫瘤切除後 | 血壓下降；需要維持高CVP（10-15 mmHg）；可能需要幾升晶體 |

### 術後考量

| 考量 | 內容 |
|-----|------|
| 監護 | ICU/HDU 12小時 |
| 血糖 | 監測（兒茶酚胺撤離可導致嚴重低血糖） |
| 類固醇 | 如果雙側切除需要立即類固醇支持 |

## 28.4 類癌腫瘤

### 特點

| 特點 | 內容 |
|-----|------|
| 起源 | 嗜銀細胞 |
| 位置 | GI tract（75%）、支氣管、胰腺和性腺 |
| 分泌 | 肽和胺 |

### 麻醉考量

| 考量 | 內容 |
|-----|------|
| 術前 | 症狀治療（抗腹瀉藥、支氣管擴張劑、糾正脫水/電解質失衡、心衰竭治療） |
| 奧曲肽 | 術前連續輸注或SC |
| 避免 | 組胺釋放藥物（如嗎啡）、促效應 |
| 監測 | 侵入性BP、CVP、血糖、血氣 |

### 圍手術期管理

| 階段 | 管理 |
|-----|------|
| 誘導 | 預防性給予奧曲肽100 mcg IV稀釋至10 mcg/mL |
| 維持 | TIVA或吸入技術 |
| 低血壓 | 奧曲肽10-20 mcg bolus |
| 高血壓 | 拉貝他洛爾、艾司洛爾或酮色林 |

### 術後考量

| 考量 | 內容 |
|-----|------|
|甦醒 | 可能非常緩慢（由於血清素） |
| 低血壓發作 | 手術可能只是減少而非消除腫瘤；需要進一步IV奧曲肽bolus |
| 奧曲肽撤離 | 在腫瘤切除術後7-10天內逐漸減量 |
