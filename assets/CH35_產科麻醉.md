## 生理學與藥理學（Physiology and Pharmacology）

### 心血管系統

| 項目 | 變化 |
|------|------|
| 心輸出量 | 增加 750% |
| 血壓 | 舒張壓早期懷孕下降，20 週時最低，足月時回至孕前水平 |
| 子宮胎盤血流 | 無自我調節，依賴，子宮动脉血壓 |
| 主腔靜脈阻塞 | 仰臥位幾乎普遍發生，10% 發生明顯仰臥低血壓症候群 |

### 血液系統

- 血漿量增加 50%，紅血球質量僅增加 30%
- 懷孕早期即處於高凝狀態
- 所有孕婦手術前應評估靜脈血栓栓塞風險

### 呼吸系統

- PaCO2 早期懷孕下降至 4.0 kPa (30 mmHg)
- 功能殘氣量足月時減少 20%
- 50% 仰臥女性發生氣道關閉
- 氧消耗增加 60%

### 胃腸系統

- 胃排空在產程中減慢，全身鴉片類藥物幾乎停止胃排空
- 食道下括約肌壓力降低
- 產後 48 小時，所有參數回覆至非懷孕對照組

### 其他系統

| 系統 | 變化 |
|------|------|
| 腎血流 | 增加 75%，腎絲球濾過率增加 50% |
| 神經組織 | 對局部麻醉藥敏感性增加，MAC 降低 |
| 分布容積 | 增加 5 L，影響主要為極性藥物 |

## 產程止痛（Analgesia for Labour）

### 止痛方法比較

| 方法 | 優點 | 缺點 |
|------|------|------|
| 笑氣/氧氣 (Entonox) | 最常用，比哌替啶更有效安全 | 從不達到完全止痛 |
| 鴉片類 (Pethidine) | 全球最常用 | 胎兒半衰期長 (18-23h)，影響新生兒神經行為和母乳哺育 |
| 脊椎硬膜外聯合 (CSE) | 最有效止痛 | 需技術操作 |

### 脊椎止痛適應症

| 適應症 | 說明 |
|--------|------|
| 母親要求 | 自願止痛 |
| 預期手術分娩 | 備用麻醉 |
| 可能使全身麻醉困難的情況 | 如病態性肥胖、困難氣道 |
| 母體共病症 | 心血管、呼吸、神經疾病 |

### 禁忌症

| 類型 | 說明 |
|------|------|
| 絕對禁忌 | 母親拒絕、過敏、局部或全身感染、未糾正低血容量、凝血病、顱內壓升高 |
| 相對禁忌 | 預期大量出血、未治療全身感染、特定心臟病 |

## 脊椎硬膜外阻斷技術（Cental Neuraxial Blocks）

### 硬膜外止痛技術

**步驟：**
1. 確保訓練有素的助產士提供一對一護理
2. 建立靜脈通路
3. 無菌技術：口罩、帽子、隔離衣、手套
4. 位置：側臥或坐姿
5. 記錄胎兒心率
6. 皮膚消毒（0.5% 氯己定）
7. 以 Tuohy 針定位硬膜外腔
8. 導管留置 3.5-5 cm
9. 抽吸檢查血液或腦脊髓液
10. 給予測試劑量
11. 負載劑量建立止痛

**維持方式：**
- PCEA（患者自控硬膜外止痛）
- 連續輸注
- 間歇追加
- 程控間歇性追加

### CSE（脊椎硬膜外聯合）

| 優點 | 缺點 |
|------|------|
| 快速 onset | 血壓和心輸出量快速變化 |
| 良好止痛品質 | 技術更困難 |
| 可術中追加 | 脊椎失敗率更高 |
| 術後可持續止痛 | 導管未測試 |

## CNB 併發症（Complications of CNB）

### 低血壓

**預防和治療：**
- 維持收縮壓 >90% 基線
- 避免主腔靜脈阻塞（左側傾斜）
- 晶體液負荷（非必要時常規輸液）
- 血管收縮劑：Phenylephrine 50-100 mcg 或 Ephedrine 6 mg

### 脊椎阻斷

| 情況 | 處理 |
|------|------|
| 高脊椎/全脊椎 | 頭高位（如為高比重局部麻醉藥） |
| 呼吸衰竭 | 呼叫幫助，ABC 原則，維持氣道和換氣 |
| 心血管抑制 | 支持治療 |

### 意外靜脈注射局部麻醉藥

**預防原則：**
- 始終檢查導管中的血液
- 每次劑量時警惕靜脈注射症狀
- 將所有大劑量分成小等分
- 使用非可互換 CNB 連接器（NRFit 系統）

### 神經損傷

| 損傷類型 | 發生率 |
|----------|---------|
| 暫時性神經功能障礙 | 約 1:3000 |
| 延長性神經功能障礙 | 更罕見 (1:15,000) |
| 永久性重大神經損傷 | <1:80,000 |

## 硬膜穿刺和術後硬膜穿刺頭痛（Dural Puncture and PDPH）

### PDPH 特徵

- 硬膜穿刺後 24-48 小時發病，可持續達 5 天
- 站立時頭痛惡化
- 通常為額枕部頭痛
- 可伴有頸部強直、聽力減退、耳鳴
- 第六對顱神經麻痺伴複視

### 處理

| 方式 | 說明 |
|------|------|
| 保守治療 | 臥床休息、充足液體攝入、簡單止痛藥、咖啡因 |
| 血液貼片 | 明確治療方法，成功率 50-80% |

### 血液貼片技術

- 書面同意
- 兩名操作者
- 患者側臥位
- 硬膜外注射 20 mL 血液
- 術後臥床 2 小時

## 產科手術麻醉（Caesarean Section）

### 緊急程度分類

| 等級 | 定義 |
|------|------|
| 等級 1 | 威脅母親或胎兒生命的緊急威脅 |
| 等級 2 | 非立即危及生命的母親或胎兒損害 |
| 等級 3 | 無母親或胎兒損害但需早期分娩 |
| 等級 4 | 無母親或胎兒損害，按母親和產科服務時間安排 |

### CNB 優點

- 母親和伴侶可在場
- 吸入風險低、過敏反應風險低
- 新生兒更警覺，促進早期連結和母乳哺育
- 術後止痛更好、早期活動

### 脊椎麻醉技術

**劑量範例：**
- 2.5 mL 0.5% 高比重 Bupivacaine
- 加 300 mcg Diamorphine 或 15 mcg Fentanyl

**術後止痛：**
- 脊椎 Diamorphine 300-400 mcg：提供 6-18 小時止痛
- 硬膜外 Diamorphine 2.5 mg：提供 6-10 小時止痛

### 低血壓預防

| 方法 | 說明 |
|------|------|
| α-致效劑 | Phenylephrine 或 Metaraminol 為首選 |
| 輸液 | 共同負荷 10-15 mL/kg 晶體液 |
| 體位 | 左側傾斜避免主腔靜脈阻塞 |

### 全身麻醉適應症

- 母親要求
- 等級 1 剖腹產
- CNB 失敗或禁忌
- 計劃較長手術

**技術要點：**
- RSI（快速順序誘導）
- 7.0 mm 氣管內管
- 50% N2O/O2
- 維持 ETCO2 4.0-4.5 kPa
- 清醒拔管

## 常用劑量摘要（Summary of Dosing Regimes）

### 產程止痛

| 方式 | 劑量 |
|------|------|
| 硬膜外負荷 | 20 mL 0.1% Bupivacaine 加 2 mcg/mL Fentanyl |
| 硬膜外輸注 | 10 mL/h 0.1% Bupivacaine 加 2 mcg/mL Fentanyl |
| CSE 脊椎 | 0.5-1 mL 0.25% Bupivacaine 加 5-25 mcg Fentanyl |
| PCEA | 10 mL 追加，15-20 分鐘鎖定 |

### 剖腹產

| 方式 | 劑量 |
|------|------|
| 脊椎 | 2.5 mL 0.5% Bupivacaine（8% 葡萄糖）加 300 mcg Diamorphine |
| 硬膜外 | 15-20 mL 2% Lidocaine 加 1:200,000 腎上腺素 |

## 產科急症（Obstetric Emergencies）

### 胎兒窘迫

- 立即通知產科醫師
- 母親左側臥位
- 100% 氧氣
- 檢查低血壓並治療

### 產後出血（PPH）

| 病因 | 處理 |
|------|------|
| 子宮張力不足 | 催產素、宮縮劑、子宮按摩 |
| 胎盤滯留 | 移除、刮宮 |
| 產道損傷 | 修復 |
| 凝血病 | 輸血、凝血因子 |

### 吸入性肺炎預防

| 情況 | 藥物 |
|------|------|
| 選擇性手術 | Ranitidine 150 mg PO（12 和 2 小時前）+ Metoclopramide 10 mg PO |
| 緊急手術 | Ranitidine 50 mg 靜脈 + Metoclopramide 10 mg 靜脈 + 30 mL 0.3M 檸檬酸鈉 |

## 哺乳和藥物轉移（Breastfeeding and Drug Transfer）

### 藥物安全性

| 藥物類別 | 安全性 |
|----------|--------|
| 鴉片類 | 安全轉移少量；避免 Codeine |
| NSAID | 大多安全 |
| 抗生素 | Penicillins、Cephalosporins 安全；避免 Tetracycline、Chloramphenicol、Ciprofloxacin |
| 精神科藥物 | 建議避免；劑量可能太小無害 |
| 心臟藥物 | Amiodarone 應停止哺乳；大多 β-阻斷劑安全 |

## 產科急症總覽

| 急症 | 關鍵處理 |
|------|-----------|
| 剖腹產全身麻醉失敗插管 | 遵循 ABC，2nd generation 聲門上氣道，考慮繼續手術 |
| 產後出血 | 液體復甦、催產素、血液製品、必要時手術 |
| 子癲前/子癲 | 控制血壓（Labetalol、Hydralazine）、硫酸鎂、終止妊娠 |
| 羊水栓塞 | 支持治療（心肺復甦）|
