## 內視鏡檢查程序（Cystoscopic Procedures）

### 適應症與患者特徵

多數泌尿外科程序透過膀胱鏡執行，包括膀胱鏡檢查、經尿道前列腺切除術（TURP）、膀胱頸切口、經尿道膀胱腫瘤切除術、輸尿管鏡檢查及結石移除或支架置入。

多數患者因良性前列腺肥大或膀胱癌就診。兩種疾病的發生率在 60 歲後顯著增加，且膀胱癌與吸菸密切相關，因此患者常同時患有冠心病（CAD）和慢性阻塞性肺疾病（COPD）。

### 術前檢查

應進行血液常規檢查、肌酐及電解質檢查。膀胱癌可能導致隱匿性出血。膀胱癌和良性前列腺肥大都可能引起阻塞性尿路病變或腎功能損害，選擇藥物和技術時應一併考量。

### 柔性膀胱鏡檢查（Flexible Cystoscopy）

柔性膀胱鏡主要用於診斷目的，不需要膀胱完全擴張，通常可在局部麻醉下執行。

活檢可透過此方式取得，不適感輕微。技術純熟的外科醫師可進行逆行輸尿管導管置入。部分患者堅持要求鎮靜或全身麻醉，Midazolam 或 Propofol 是合適的選擇。

### 硬式膀胱鏡檢查（Rigid Cystoscopy）

硬式膀胱鏡因鏡徑較粗且需使用沖洗液擴張膀胱以利術野可視化，需要全身麻醉。

若大量沖洗液被吸收，可能導致全身性液體過載併發症（詳見 TURP 症候群章節）。

### 脊髓麻醉

脊髓麻醉適合硬式膀胱鏡檢查程序，也常用於 TURP。尿道、前列腺、膀胱頸和膀胱黏膜的感覺神經支配來自 S2 至 S4。然而膀胱擴張的疼痛由 T10 至 L2 傳導，因此需要較高的阻滯平面。

多數患者會要求鎮靜。在老年族群中，1 至 2 mg Midazolam 通常足夠。較高劑量可能導致氣道控制力喪失、意識混亂和躁動。低劑量 Propofol 輸注是替代方案。

脊髓麻醉對嚴重 COPD 患者有優勢，只要患者能平躺而不咳嗽。

### 麻醉藥物選擇

無論使用高低比重 Bupivacaine 皆可。高比重 Bupivacaine 通常產生比等比重溶液更高的阻滯平面，特別是當注射時患者處於側臥位後再翻轉至仰臥位時。

2.5 至 3 mL 的「重比重」Bupivacaine 0.5% 通常可達 T10 平面。除非阻滯平面不夠高，否則不應將患者擺頭低腳高位。

其他短效藥物可增加門診脊椎麻醉的有效選項：40 至 50 mg（4 至 5 mL）Plain Chloroprocaine 1% 可支援最长 40 分鐘的手術，或 40 至 60 mg（2 至 3 mL）高比重 Prilocaine 2% 可支援最长 90 分鐘的手術。這些選項比 Bupivacaine 提供更可預測的脊椎麻醉效果，且患者能更快恢復後出院。

### 脊髓麻醉禁忌與特殊考量

慢性胸廓疾病患者平躺時容易咳嗽。區域阻滯下，手術期間咳嗽會嚴重影響手術操作。鎮靜有助減少咳嗽反射。

脊髓損傷患者（見相關章節）常需重複泌尿外科程序。膀胱鏡檢查時膀胱擴張刺激性極強，容易引起自律神經反射亢進，建議採用全身麻醉或脊椎麻醉，並查看先前的麻醉記錄。

### 體位與設備考量

為老年患者擺置截石位時應特別小心，特別是關節置換後的患者。

心律調節器通常不是問題，即使進行 TURP 所需幾乎持續的電燒，只要將電燒片置於尾側（通常是大腿）即可。

植入式心律去顫器可被電燒觸發，需在術前關閉。

### 陰莖勃起問題

陰莖勃起會使膀胱鏡檢查困難並增加手術風險。通常因麻醉深度不足時的手術刺激引起，可通過加深麻醉來處理。若勃起仍持續，小劑量 Ketamine 可能有效。

### 抗生素預防

通常需要單劑量對革蘭氏陰性菌有覆蓋範圍的抗生素，特別是患者有留置導尿管、輸尿管阻塞或術前中段尿培養陽性時。Gentamicin 3 mg/kg 是常用選擇。

### 靜脈血栓預防

低風險患者接受內視鏡檢查程序時，漸進式加壓襪或氣動小腿加壓裝置通常足夠，因為多數患者術後會快速活動。額外危險因素或近期曾手術且需一段時間臥床的患者，應同時使用低分子量肝素（LMWH）。

## 經尿道前列腺切除術（TURP）

### 手術概覽

經尿道前列腺切除術是一種使用電燒線圈進行的內視鏡前列腺切除術。傳統使用單極電燒，沖洗液為甘氨酸，逐漸被雙極電燒手術取代，後者使用 0.9% 生理食鹽水沖洗。

### 手術基本資料

| 項目 | 內容 |
|------|------|
| 手術時間 | 30 至 90 分鐘，視前列腺大小而定 |
| 疼痛程度 | + |
| 體位 | 截石位，可 +/- 頭低腳高位 |
| 失血量 | 變異極大（200 至 2000 mL），可能大量並持續至術後；需備血型及交叉比對 |
| 實用技術 | 脊髓 +/- 鎮靜（首選）或 全身麻醉（喉罩、聲門上呼吸道或氣管內管及正壓換氣） |

### 術前評估

患者通常年齡較大，常有多種共病症和多重用藥。

應檢查肌酐和血清鈉離子濃度。若鈉離子顯著偏低，建議延後手術，因為吸收沖洗液後鈉離子可能進一步下降。

心衰竭或未控制的房顫因液體吸收風險而特別危險。術前應設法達到最佳的醫療控制。

應評估精神狀態和溝通能力。若患者意識混亂或聽力受損，脊椎麻醉會較困難。

### 術中管理

應置入大口徑針頭，並使用加溫靜脈输液。

脊椎麻醉：理論上更容易察覺精神狀態變化和液體過載徵象（見後述）。部分研究顯示可減少失血，但非全部。劑量為 Bupivacaine（等比重或高比重）2.5 至 3 mL，或考慮替代局部麻醉藥。應頻繁測量血壓——上述劑量血壓下降並不常見，但可能突然發生。應在雙腿放下後（此時血壓下降會顯現）再次測量血壓。

全身麻醉：若患者過度肥胖或有胃食道逆流病史，應考慮氣管內插管。術中給予 Fentanyl 或 Morphine 加多模式鎮痛（Acetaminophen、NSAID 和 Tramadol/Codeine）通常足夠。術後通常不需要鴉片類藥物。

失血量評估困難。理論上可透過測量血紅素和拋棄沖洗液量來計算。臨床上更常以眼觀評估體積和顏色，但可能誤導。床邊血紅素檢測儀（如 HemoCue）有幫助。失血量通常與切除的前列腺組織大小和重量（通常 15 至 60 g）、手術時間和外科醫師技術相關。

沖洗液療法：晶體液可先用。應牢記大量低張沖洗液可能被吸收，因此不要給予過量液體，且千萬不要使用葡萄糖。若血紅素低於輸血閾值，應準備輸血。

### 術後護理

術後使用三腔導尿管以 0.9% 生理食鹽水進行膀胱沖洗，持續 7 至 24 小時，直至出血減少。沖洗不暢可能導致血塊滯留。

通常疼痛輕微，但導尿管不適或膀胱痙攣可能造成困擾。嚴重疼痛提示血塊滯留、膀胱痙攣或膀胱穿孔。

血塊滯留可能導致膀胱極度擴張、疼痛和迷走神經症狀。需要沖洗，有時需在麻醉下進行。出血可能持續並需要再次手術；可能需要復甦處理。

術後第一天應測量血液常規、肌酐和電解質。

### 特殊考量

使用大量沖洗液時可能導致低體溫（沖洗液應加溫至 37°C）。

若前列腺非常大（>100 g），開放性單純前列腺切除術可能併發症較少。

併發症風險隨切除時間超過 1 小時而增加。若切除可能超過一小時，考慮只切除一葉，另一葉留待日後處理。

雙極 TURP 已被證明可降低整體併發症發生率、輸血率和 TURP 症候群。

### 雷射前列腺切除術及汽化術

多種微創技術使用雷射和其他熱能形式來縮小前列腺體積。這些技術通常減少出血和液體吸收，有時被認為是高風險患者的首選。

雷射汽化技術可在局部麻醉下門診執行。對於正在接受抗凝治療的患者，由於其對前列腺組織的止血效果，可能是較好的選擇。

### 前列腺近接治療（Brachytherapy）

適用於局部前列腺癌。透過超聲波引導下的桿狀物將放射性粒子植入前列腺內。通常在放射治療科的手術室外進行。

患者可能需要在同一天接受兩次或多次程序。重複全身麻醉雖可行，但在每次程序前以脊髓導管追加注射效果良好。需考慮遠端手術麻醉的風險。

單次治療時，脊髓麻醉（使用 Bupivacaine 0.5% 加 Fentanyl 15 mcg）有效，且可安全轉送至放射治療科。

## TURP 症候群（TURP Syndrome）

### 病理生理

TURP 症候群是液體過載和低鈉血症的組合。當大量沖洗液透過開放的靜脈竇被吸收時發生。

傳統單極電燒使用非導電（非電解質）沖洗液。最常用的是甘氨酸 1.5%（溶於水），為低張液（滲透壓 220 mmol/L）。

甘氨酸是一種非必需胺基酸，作為抑制性神經傳導物質。甘氨酸毒性是否參與症候群尚不清楚。氨是甘氨酸的代謝物，也可能導致中樞神經系統障礙。

雙極 TURP 已成為更常見的手術方式，允許使用等張沖洗液。低鈉血症少見得多；然而，大量液體吸收仍可能導致過載。比較單極與雙極 TURP 的綜合分析發現，雙極技術消除了「TURP 症候群」且整體併發症發生率較低。

### 預防

預防措施包括：避免使用單極電燒（非等張沖洗液）、使用低灌注壓力（膀胱壓力 <15 mmHg）、監測吸收的沖洗液量（若液體負載 >2 L 考慮終止手術）、限制手術時間。

### 臨床表徵

肺水腫、腦水腫和低鈉血症的表徵是常見的表現。選擇脊椎麻醉可早期發現。

早期症狀包括躁動、頭痛和呼吸急促，可能進展為呼吸窘迫、低氧血症、明顯肺水腫、噁心、嘔吐、視覺障礙、意識混亂、痙攣和昏迷。全身麻醉患者唯一可能的證據是心搏過速和高血壓。大量快速吸收可導致反射性心搏過緩。低血壓也可能發生。

診斷可經由低血清鈉來確認。急性下降至 <120 mmol/L 一定有症狀。快速檢查鈉離子通常可經由檢測血氣分析（使用靜脈血，除非有酸鹼平衡疑慮）。

### 治療

若在術中檢測到，應通知外科醫師。止血點應盡快電燒，手術應盡快終止，並停止靜脈輸液。給予 Furosemide 20 至 40 mg，並檢查血清鈉和血紅素。需要時以氧氣或氣管內插管及換氣支持呼吸。若有痙攣，給予靜脈抗痙攣藥物。

嚴重急性低鈉血症和過快校正慢性低鈉血症都可能導致永久性神經損傷（最常見的是橋腦中央髓鞘溶解）。

若血清鈉維持 >120 mmol/L，限制液體攝取應足夠。若 <120 mmol/L 且伴有神經學徵象，考慮給予高張氯化鈉（1.8% 至 3%），將鈉離子恢復至約 125 mmol/L。以 1.2 至 2.4 mL/kg/h 的 3% 氯化鈉，直到症狀改善。校正速度理想上不應超過 1.5 至 2 mmol/L/h，持續 3 至 4 小時，然後 1 mmol/L/h，直到症狀改善或鈉離子 >125 mmol/L。24 小時內最大上升不應超過 12 mmol/L。

應注意其他同時治療對鈉離子的複合效應（利尿劑、膠體等）。

## 經尿道膀胱腫瘤切除術（TURBT）

### 手術基本資料

| 項目 | 內容 |
|------|------|
| 手術方式 | 內視鏡電燒切除膀胱腫瘤 |
| 手術時間 | 10 至 40 分鐘 |
| 疼痛程度 | +/++ 及膀胱痙攣 |
| 體位 | 截石位 |
| 失血量 | 0 至 >500 mL |
| 實用技術 | 全身麻醉（喉罩）；脊椎 +/- 鎮靜 |

### 術前評估

膀胱上皮生長緩慢的腫瘤在老年人中常見，建議以週期性腫瘤切除而非積極完全切除來管理。

膀胱癌最常見於吸菸者，應檢查冠心病和COPD。慢性失血常見，應檢查血紅素。應檢查腎功能。許多患者需重複手術，應查看先前的麻醉記錄。

### 術中考量

閉孔痙攣（Obturator spasm）發生於閉孔神經（位於膀胱側壁附近）被電燒電流直接刺激時。會導致腿部內收，嚴重損害手術入路並增加膀胱穿孔風險。通常可透過減少電燒電流或間歇性使用神經肌肉阻斷劑來控制。

需要抗生素預防。

### 術後護理

廣泛切除後疼痛可能成問題。NSAID（有助於腎功能）可用，口服鴉片類可能需要。

膀胱痙攣常見。

### 特殊考量

若使用脊椎麻醉，確保阻滯平面高於 T10。

## 開放性單純前列腺切除術及根除性前列腺切除術

### 開放性單純前列腺切除術

| 項目 | 內容 |
|------|------|
| 手術方式 | 逆行恥骨上（Millen 手術）：開放切除明顯良性前列腺肥大組織 |
| 手術時間 | 60 至 120 分鐘 |
| 疼痛程度 | +++/++++ |
| 體位 | 仰臥位 |
| 失血量 | 300 至 1000 mL；備血 2 單位，有條件使用自體血回收 |
| 實用技術 | 氣管內管、正壓換氣、Remifentanil +/- 腹直肌鞘導管 +/- PCIA |

### 根除性前列腺切除術

| 項目 | 內容 |
|------|------|
| 手術方式 | 開放完整切除惡性前列腺及骨盆腔淋巴結，並進行泌尿系統重建 |
| 手術時間 | 120 至 180 分鐘 |
| 疼痛程度 | ++++/++++ |
| 體位 | 仰臥位 |
| 失血量 | 500 至 2000 mL；備血 2 單位，有條件使用自體血回收 |
| 實用技術 | 氣管內管、正壓換氣、Remifentanil +/- 腹直肌鞘導管 +/- PCIA |

### 術前評估

單純前列腺切除術通常在老年男性中進行，管理方式可參考 TURP。

根除性手術：患者需相對年輕且醫療狀況良好。應檢查腎功能。考慮術前鎮痛介入措施（COX-2 選擇性抑制劑、Gabapentinoids 和 Dexamethasone）。根據當地常規和醫療因素，考慮安排重癥監護床位。

### 術中管理

備血、大口徑靜脈針、保暖毯等。單純前列腺切除術使用恥骨上切口；根除性手術使用下腹正中切口。

考慮使用动脉導管和中心靜脈壓或心輸出量監測，特別是心血管疾病患者。備血，必要時術中追加。自體血回收技術在預期大量失血時有用。

開放性前列腺切除術推薦術末給予局部麻醉創口浸潤。根除性前列腺切除術推薦 Lidocaine 輸注。PCIA 是有用的替代方案。

### 術後護理

局部麻醉創口浸潤推薦用於開放性前列腺切除術。 Lidocaine 輸注推薦用於根除性前列腺切除術。PCIA 是有用的替代方案。

## 腎切除術及部分腎切除術（Nephrectomy and Partial Nephrectomy）

### 手術基本資料

| 項目 | 內容 |
|------|------|
| 手術方式 | 腫瘤、其他病理或活體捐贈者的腎臟切除 |
| 手術時間 | 1 至 2.5 小時 |
| 疼痛程度 | +++/++++ |
| 體位 | 仰臥位或側臥位（腎臟體位） |
| 失血量 | 取決於病理，300 至 >3000 mL；備血型及交叉比對 |
| 實用技術 | 氣管內管、正壓換氣 +/- 胸椎硬膜外或創口浸潤導管 |

### 術前評估

術前應確定病理和計劃的手術切口，以決定技術和監測方式。

腎腫瘤可在無失血的情況下引起貧血，應檢查血紅素。應檢查血壓和腎功能。「無功能」腎或腎血管疾病與腎功能損害和高血壓相關。

考慮術中自體血回收。應檢查血清電解質；腎腫瘤可引起不適當的抗利尿激素（ADH）分泌。

若為腫瘤，應檢查胸部 X 光片；可能有轉移、胸腔積液等。射頻消融（RFA）適用於小皮質腫瘤（<3 cm）、高手術風險或腎功能受損的患者。

### 術中管理

腎切除術和部分腎切除術可經開放、腹腔鏡或機器人輔助腹腔鏡進行。

大型腫瘤和多囊腎在英國的標準做法是經腹直肌旁或橫切口開放手術。腎臟切口需「腎臟體位」——側臥位，身體在手術床關節處過伸。此體位因腿部靜脈回流減少和可能的下腔靜脈壓迫，血壓通常會大幅下降。術中進一步壓迫可能導致靜脈回流嚴重減少和心輸出量下降。

應詢問外科醫師預期的手術範圍；大型腫瘤可能需廣泛解剖，可能經胸廓切開術，或打開下腔靜脈切除腫瘤邊緣，此時可能發生突然且大量的失血。有時暫時夾閉下腔靜脈以利解剖和控制出血；這會導致心輸出量突然下降。若血壓突然下降，應告知外科醫師；備好液體和血液，隨時準備加壓輸液，並備好血管收縮劑或如 Metaraminol 或 Ephedrine 的強心劑。

使用大口徑靜脈針、血液加溫器、中心靜脈導管和动脉導管。若使用硬膜外，術中需高平面阻滯，但應謹慎使用直至出血得到控制。

### 術後護理

所有手術方式都會疼痛。硬膜外有用，但腰部切口需覆蓋至 T7/8。腹直肌鞘導管對前方入路有用。PCIA 或鴉片類輸注是替代方案。

肋間神經阻斷可在腰部切口後數小時提供鎮痛。創口浸潤導管也被證明非常有效。

若腎功能術後良好且患者無低血容量，NSAID 有用，應謹慎使用。每小時監測尿量。

### 部分腎切除術（Partial Nephrectomy）

推薦作為最大 7 cm 器官局限性腫瘤的首選方案。失血可能很大，因為血管更難控制。

部分外科醫師建議在夾閉腎动脉前給予甘露醇 12.5 g、Furosemide 10 mg 和/或 Heparin 3000 單位，以維持腎灌注並減少缺血。也可使用冰敷降溫。

若術前腎功能明顯受損，整個術期優化液體平衡極為重要。應考慮術後入住重癥監護。

## 根除性膀胱切除術（Radical Cystectomy）

### 手術基本資料

| 項目 | 內容 |
|------|------|
| 手術方式 | 膀胱切除加尿路改道（如迴腸導管）或新膀胱重建（正位膀胱形成） |
| 手術時間 | 3 至 5 小時（膀胱重建時間更長） |
| 疼痛程度 | ++++ |
| 體位 | 截石位加頭低腳高位 |
| 失血量 | 700 至 >3000 mL；備血 4 單位，有條件使用自體血回收 |
| 實用技術 | 氣管內管、正壓換氣、动脉導管 +/- 食道都卜勒/中心靜脈壓 +/- 腹直肌鞘導管/PCIA/硬膜外 |

### 術前考量

考慮使用「加速康復路徑」。檢查缺血性心臟病/慢性阻塞性肺疾病，以及腎功能和血液常規。根據當地常規和共病症安排重癥監護床位。確保開立靜脈血栓預防。

### 術中管理

術後最常見的問題是延長性腸麻痺，對發病率和死亡率有顯著貢獻，多項建議的措施被認為可減少其發生。

為大量失血做準備；大口徑靜脈針、血液加溫器、中心靜脈壓或食道都卜勒心輸出量和直接动脉監測是常規配備。備血，必要時術中追加。考慮術中自體輸血。若使用血液回收，在腸道打開時停止。

若使用腹直肌鞘導管，在誘導後置入。手術切口是臍下正中，術中不易放置導管。Remifentanil 輸注提供穩定的麻醉和可控血壓；輕度低血壓可通過減少失血來幫助手術。

硬膜外現在使用不多，但若放置，術中應謹慎使用；待主要失血環節過後有足夠時間建立有效阻滯。採取保暖措施（如加壓空氣保暖毯）。

抗生素預防如同腸道切除。現在很少使用鼻胃管；如有需要應詢問外科醫師。骨盆腔靜脈叢失血可能緩慢，考慮秤重紗布。空氣栓塞是任何大型骨盆腔手術的可能併發症。

### 術後護理

腹直肌鞘導管（可達 5 天），有或無 PCIA，被證明非常有效。內臟疼痛通常持續 24 至 36 小時，需要腸胃外鴉片類。此技術可早期活動、恢復腸功能和減少術後腸麻痺/住院時間。有證據顯示術後腸吻合口滲漏與 NSAID 使用相關，因此使用時應謹慎檢查腎功能。

以中心靜脈壓/尿量指導液體置換；需求量通常很大，因為腹膜內損失和腸麻痺。經迴腸導管的新尿路引流難以監測，因為引流取決於體位。經正位重建後，尿液從多根不同的導管流出，需每小時計算以監測輸出量。

作為加速康復的一部分，早期進食可能與減少某些術後併發症相關。輸尿管吻合口滲漏可能表現為腹部引流管中的尿液；透過比較引流液和導管尿液的生化指標來確認。

## 機器人輔助腹腔鏡前列腺切除術（Robot-Assisted Laparoscopic Prostatectomy）

### 手術基本資料

| 項目 | 內容 |
|------|------|
| 手術方式 | 機器人輔助腹腔鏡前列腺切除術 |
| 手術時間 | 120 至 180 分鐘 |
| 疼痛程度 | +/++ |
| 體位 | 截石位加 steep Trendelenburg |
| 失血量 | 極少量 |
| 實用技術 | 氣管內管、正壓換氣、Remifentanil、动脉導管 |

### 術前考量

到目前為止，手術機器人在泌尿外科中最常見，主要用於根除性前列腺切除術。

對患者的優勢可能包括減少失血、疼痛和住院時間，可能減少尿失禁和勃起功能障礙的發生率。對外科醫師相對於傳統腹腔鏡的優勢包括提供 3D 視野、過濾手部顫抖、縮放手部動作、在患者體內更大的活動範圍，以及舒適穩定的姿勢。

### 術中管理

患者體位擺放很重要。長時間頭低腳高位與以下情況相關：

- **神經損傷（特別是臂叢）**：注意體位擺放，使用肩托。不要過度伸展截石位中的腿部。
- **面部/氣道水腫和喘鳴**。
- **眼睛和口腔潰疡的酸蝕**：術前給予質子泵抑制劑（Omeprazole 40 mg）；考慮口胃管（非鼻胃管，因有鼻出血風險）和喉塞；用潤滑眼藥膏/墊保護眼睛。
- **顱內壓增加（因高碳酸血癥加重）和眼壓增加**（青光眼患者應注意）。

氣管內管應盡可能短（支氣管內插管風險），並固定以盡量減少腦靜脈阻塞。機器人設備一旦插入腹腔就會鎖定位置，任何意外的患者移動都可能導致嚴重的手術併發症。Remifentanil 輸注效果好，允許只在需要時間歇性給予神經肌肉阻斷劑。

避免使用氧化亞氮。笑氣可能干擾機器人操作的精確性並增加腸道擴張。

患者通道受限，確保左側有可靠的大口徑靜脈通路（因機器人手臂位置）。與外科醫師溝通可能因設備所需的大量空間而困難。團隊需熟悉音頻設備，並能夠在需要復甦的緊急情況下快速分離並移除機器人。

大手術時間內大量的尿量可能干擾術野；在吻合完成前使用少量靜脈液體（<1000 mL），這也可減少氣道水腫的風險。

### 術後護理

拔管前進行洩漏測試以評估氣道腫脹。大腦水腫可能造成問題。短效吸入麻醉劑加 Remifentanil 可快速術後評估意識狀態。多數患者最初醒來時會有一定程度的大腦刺激或躁動。

術後疼痛比開放手術少得多，但可能仍需短期鴉片類鎮痛加簡單鎮痛藥。在部分機構，鞘內 Diamorphine（達 1 mg）是常用方案。

## 經皮結石移除（Percutaneous Stone Removal）

### 手術基本資料

| 項目 | 內容 |
|------|------|
| 手術方式 | 經腎造口進行內視鏡結石切除 |
| 手術時間 | 60 至 90 分鐘 |
| 疼痛程度 | ++/+++ |
| 體位 | 俯臥斜位 |
| 失血量 | 變異，0 至 1000 mL |
| 實用技術 | 氣管內管和正壓換氣 |

### 術前考量

患者通常是健康年輕人和中年人，但結石可能因潛在的代謝問題或神經功能障礙導致的膀胱功能障礙而形成。應檢查腎功能。

### 術中管理

患者最初處於截石位以置入輸尿管支架，然後翻身至半俯臥位，在第 12 肋骨下方後外側置入腎造口，在放射顯影控制下進行。應注意可能移開管路和壓瘡風險。

考慮使用加強型氣管內管以防止打折，並妥善固定。頭部需轉向手術側，因此最好將氣管內管置於口腔同側。支撐胸部和骨盆以允許換氣時腹部起伏。支撐並墊好頭部、手臂和小腿，並墊好眼睛。在體位改變期間和之後檢查換氣。換氣可能需暫時中斷以拍攝 X 光片。

### 術後護理

腎造口疼痛變異很大。多模式鎮痛：Acetaminophen、NSAID（檢查腎功能）、PCIA Morphine 或口服 Tramadol。

### 特殊考量

使用大量沖洗液時可能發生低體溫。腎造口置入通常靠近橫膈，有穿透胸膜導致氣胸或水胸的風險——如有疑問，術後進行胸部 X 光片。大量沖洗液進入後腹膜腔是腎盂破裂的已知併發症。任何尿路結石手術後的術後革蘭氏陰性菌敗血症是重要風險。

## 體外震波碎石術（ESWL）

### 手術基本資料

| 項目 | 內容 |
|------|------|
| 手術方式 | 使用脈衝超聲波非侵入性粉碎腎結石 |
| 手術時間 | 20 至 40 分鐘 |
| 疼痛程度 | +/++ |
| 失血量 | 無 |
| 實用技術 | 成人鎮靜；兒童全身麻醉/聲門上呼吸道 |

### 技術背景

早期體外震波碎石術患者需浸泡在水槽中的半坐位，產生許多對麻醉醫師來說是問題的情況。1980 年代的發展不再需要水槽，近期超聲波束的改進使其不那么不舒服，大多數患者不需要麻醉或鎮靜。

### 術前考量

患者常接受重複碎石治療，尽可能查看先前治療記錄。Acetaminophen/NSAID（注意腎功能）術前給藥通常有效。

### 術中管理

側臥位，雙臂高舉頭部。腎結石使用超聲波或影像增強器定位，衝擊波聚焦於結石。抗生素預防可能需要。

### 特殊考量

衝擊波可能偶爾引起心律不整，通常是自限性的。若持續，衝擊波可與心電圖同步（在不反應期）傳遞。謹慎使用抗膽鹼能藥物（Glycopyrronium 200 mcg）可增加心率和增加傳遞的衝擊頻率。

心律調節器可被衝擊波解除程式；需諮詢心律調節器技術人員。衝擊波能量在遇到空氣/水介面時釋放。若放置硬膜外時需要「阻力消失」技術，建議使用 0.9% 生理食鹽水而非空氣。

## 睪丸手術（Testicular Surgery）

### 手術基本資料

| 項目 | 內容 |
|------|------|
| 手術方式 | 睪丸切除/活檢、睪丸積液袋狀縫合、輸精管切除術、睪丸扭轉 |
| 手術時間 | 30 分鐘至 1 小時 |
| 疼痛程度 | ++/+++ |
| 體位 | 仰臥位 |
| 失血量 | 不明顯 |
| 實用技術 | 全身麻醉、聲門上呼吸道、精索阻斷；緊急情況（如扭轉）時使用快速順序插管/氣管內管。脊髓或局部浸潤 |

### 術前考量

通常適合門診手術。

### 術中考量

注意迷走神經反應——備好 Atropine。

### 特殊考量

睪丸和陰囊的神經支配：軀體神經支配來自髂腹股溝神經、陰部股神經、陰部神經和後側陰囊神經（大腿後側皮神經分支），神經根來自 L1 至 S3。自律神經支配來自交感神經鏈 T10 至 L4 和副交感神經叢 S1 至 S3。因此局部技術需覆蓋 T10 至 S3。

精索阻斷可作為全身麻醉的輔助或作為局部技術的一部分用於陰囊手術。阻斷覆蓋所有神經，除陰部神經和後側陰囊分支。如作為局部技術的一部分，還需要補充陰囊皮膚浸潤。

精索在直接視野下由外科醫師阻斷最好。然而，若計劃局部技術，觸摸精索從陰囊頂部進入的位置，並在周圍浸潤 5 至 10 mL 局部麻醉藥。

## 腎臟移植（Renal Transplant）

### 手術基本資料

| 項目 | 內容|
|------|------|
| 手術方式 | 屍體或活體捐贈者器官移植 |
| 手術時間 | 90 至 180 分鐘 |
| 疼痛程度 | ++/+++ |
| 體位 | 仰臥位 |
| 失血量 | 不明顯至 500 mL |
| 實用技術 | 氣管內管、正壓換氣、中心靜脈壓監測 |

### 術前考量

常見問題與慢性腎病和尿毒癥相關。慢性貧血常見（血紅素通常約 80 至 100 g/L）。不要輸血至正常水平。缺血性心臟病常見，應在工作評估中進行治療和處理。

患者通常近期有血液透析，因此常有一定程度的低血容量。檢查透析後血液，包括鉀離子。

### 術中管理

放置靜脈針和血壓袖帶時，避免動靜脈瘻管。盡量避免使用肘窩靜脈或手腕橈側靜脈（可能未來需做瘻管）。

誘導前溫和地進行液體負荷；血壓可能大幅波動。

腎功能衰竭安全使用的常用藥物包括 Fentanyl、Remifentanil、Propofol、Atracurium 和 Sevoflurane。

考慮在超聲引導下放置中心靜脈導管並監測中心靜脈壓。避免鎖骨下靜脈（有未來瘻管風險）。考慮动脉導管。

移植腎再灌注前，給予液體以將中心靜脈壓提高至 12 至 15 mmHg、平均動脈壓提高至 >80 mmHg 以優化灌注。若患者液體已充足，可謹慎使用 Ephedrine 和/或 Metaraminol 來提高血壓。

維持正常體溫。多數中心在移植腎再灌注前給予利尿劑（如甘露醇/Furosemide）和免疫抑制劑（如甲基強的松龍）。查看當地方案。

盡可能避免輸血，但若有手術出血或血紅素 <80 g/L 可能需要。

### 術後護理

常規 Acetaminophen 和 PCIA 通常足夠。Fentanyl PCIA 是首選。對嗎啡要謹慎，因為清除率降低有呼吸抑制風險。硬膜外/脊椎通常不需要；有插入時出血的風險（殘餘抗凝來自血液透析、血小板功能差等），且可能導致低血壓。考慮腹橫肌平面（TAP）阻斷。

避免 NSAID。 每小時監測中心靜脈壓和尿量。維持輕度高血容量以促進利尿。多數單位有方案。
