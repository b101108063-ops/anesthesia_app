## 19.1 術前考量

心臟手術常採用跨專科團隊合作模式。評估時需確認介入治療（PCI）與外科手術（CABG）的利弊。

根據指引，對於左主幹疾病、嚴重的三條血管病變（>70% 狹窄）、低射出分率（EF）、糖尿病及 SYNTAX 分數 >22 的患者，CABG 的存活率優於 PCI。

### 病史評估重點

| 項目 | 內容 |
|-----|------|
| 心臟病史 | 近期心肌梗塞（MI）、心衰竭 |
| 出血史 | 評估出血風險 |
| 心律不整 | 任何心律失常病史 |
| 其他器官功能 | 腦部儲備能力、肝腎功能 |
| 既往史 | 曾接受胸部放射治療 |

### 心臟超音波（Echo）發現

| 發現 | 臨床意義 |
|-----|---------|
| 左心室（LV）功能 | 評估心臟收縮能力 |
| 瓣膜疾病 | 確認病變類型和嚴重度 |
| 肺動脈壓（PAP）升高 | 評估肺高壓程度 |
| 右心室（RV）功能障礙 | 影響術前決策和預後 |

### 藥物管理

| 藥物類別 | 建議 |
|---------|------|
| ACE抑制劑/ARBs | 許多醫學中心會在圍手術期停用 |
| 抗血小板藥物 | 確認已適當停用 |
| 抗凝血藥物 | 確認已適當停用 |
| 術前鎮靜藥物 | 對於嚴重主動脈瓣狹窄（AS）、肺動脈高壓或RV功能障礙的患者應格外謹慎 |

### 風險分層

最常使用 **EuroSCORE II** 與 **STS 評分系統** 來預測心臟手術的死亡率。

兩者皆考量：
- 患者人口學特徵
- 心臟因素（如 NYHA 分級、LV 功能、肺動脈高壓）
- 手術因素（緊急程度、手側類型）

## 19.2 術中經食道心臟超音波（TOE）

在許多醫學中心，TOE 已成為常規。

### 併發症

| 項目 | 發生率 |
|-----|--------|
| 總體併發症發生率 | 約 0.2% |
| 食道破裂（致命併發症） | 0–0.3%（老年女性風險較高） |

### 適應症

| 等級 | 適應症 |
|-----|--------|
| Class I | 瓣膜修補或置換、無法解釋的血流動力學不穩定 |

### 絕對禁忌症

| 禁忌症 |
|-------|
| 空腔臟器穿孔 |
| 食道狹窄 |
| 食道腫瘤 |
| 食道撕裂/穿孔 |
| 食道憩室 |
| 活動性上消化道出血 |

## 19.3 心肺繞道（CPB）

CPB 在心臟停搏期間取代心肺功能，提供無血且靜止的手術視野。

### 運作機制

| 參數 | 數值/說明 |
|-----|-----------|
| 血流類型 | 非搏動性血流 |
| 流量 | 約 2.4 L/min/m² |
| MAP 目標 | 50–70 mmHg |
| 氣體交換 | 膜式氧合器，透過濃度梯度進行 |

### 抗凝血

- 需要使用高劑量肝素（Heparin）達到全身完全抗凝血
- 目標 ACT > 480 秒

### 副作用

| 副作用 | 說明 |
|--------|------|
| 血小板功能障礙 | 隨時間延長而惡化 |
| 溶血 | 隨時間延長而惡化 |
| 凝血因子消耗 | 隨時間延長而惡化 |
| 前兩小時 | 影響較小 |

## 19.4 心肌保護

透過注入心肌停搏液（Cardioplegia）達成舒張期電化學停搏。

### 心肌停搏液成分

| 成分 | 作用 |
|-----|------|
| 高濃度 K⁺ | 改變靜息膜電位 |
| 晶體液或血液混合 | 作為載體 |

### 給予途徑

| 途徑 | 說明 |
|-----|------|
| 順行性（Antegrade） | 經主動脈根部或冠狀動脈開口 |
| 逆行性（Retrograde） | 經冠狀靜脈竇 |

### 全身低溫的好處

| 好處 | 說明 |
|-----|------|
| 降低代謝率 | 保護重要器官 |
| 腦部保護 | 深度低溫提供神經保護 |

### 深度低溫的風險

- 增加凝血病變風險
- 溫度範圍：通常降至 18–32°C

## 19.5 常規心臟麻醉流程

以下為常規心臟麻醉（如 CABG 或瓣膜手術）的標準流程：

### 第一階段：誘導與準備（Pre-CPB）

| 項目 | 內容 |
|-----|------|
| 靜脈通路 | 建立大口徑周邊靜脈與动脉導管（A-line） |
| 誘導 | 給予 Fentanyl 並使用揮發性氣體或 TIVA 輕柔誘導 |
| 插管後 | 置入中央靜脈導管（CVC）或肺動脈導管（PAFC） |
| 監測 | 放置 TOE，鼻咽/膀胱溫度計 |
| 胸骨切開前 | 額外劑量 Fentanyl 以抑制高血壓反應 |
| 抗纖維溶解劑 | 給予 Tranexamic acid 以減少失血 |

### 第二階段：建立心肺繞道（Establishing CPB）

| 項目 | 內容 |
|-----|------|
| 肝素劑量 | Heparin 300–400 U/kg |
| 主動脈導管插入 | 收縮壓應控制在 <100 mmHg 以減少主動脈剝離風險 |
| 全流量 | 幫浦達到「全流量（Full flow）」後關閉呼吸器 |
| 期間給藥 | 重新給予抗生素與肌肉鬆弛劑 |
| 目標 ACT | >480 秒 |

### 第三階段：脫離心肺繞道（Separation from CPB）

| 項目 | 內容 |
|-----|------|
| 核心體溫 | 必須 >36°C |
| ABG | 必須正常 |
| 心律 | 恢復至 70–100 bpm（需要時去顫或節律） |
| 呼吸器 | 重新開啟 |
| 升壓藥 | 確保運作中 |
| TOE | 評估心臟與瓣膜功能 |
| Protamine | 待血流動力學穩定後給予（每 100 單位 Heparin 給予 1 mg） |
| 給藥速度 | 緩慢以避免低血壓與肺動脈高壓 |

### 第四階段：術後轉送（Postoperative）

| 項目 | 內容 |
|-----|------|
| 導管拔除 | 拔除主動脈導管 |
| 止血 | 輸血與止血處理妥當 |
| 轉送 | 帶著氣管內管轉送至心臟加護病房（ICU） |

### 拔管條件

| 條件 | 數值/標準 |
|-----|-----------|
| MAP | >65 mmHg 且血流動力學穩定 |
| FiO₂ | ≤28% 且自主呼吸良好 |
| 酸鹼值 | 正常 |
| 出血量 | <100 mL/h |
| 意識 | 清醒 |

## 19.6 特定手術考量

### 19.6.1 不停跳繞道手術（OPCABG）

| 項目 | 內容 |
|-----|------|
| 定義 | 在跳動的心臟上進行，避免 CPB 帶來的全身發炎反應與主動脈插管的栓塞風險 |
| 血流動力學不穩 | 為了吻合後側血管（如左迴旋支），心臟必須被抬起，常導致嚴重的血壓下降 |

**應對策略：**

| 策略 | 說明 |
|-----|------|
| 升壓藥 | Phenylephrine |
| 增加前負荷 | 給予晶體液 |
| 調整床位 | 頭低腳高 |
| 預防性 Atropine | 防止迷走神經誘發的心搏過緩 |

**抗凝血：**
- Heparin 劑量較低：150 U/kg
- 目標 ACT >300
- 需隨時做好緊急轉為 CPB 的準備

### 19.6.2 緊急繞道手術（Emergency CABG）

| 項目 | 內容 |
|-----|------|
| 患者狀態 | 通常已處於心因性休克，或在心導管室發生併發症 |
| 出血風險 | 極高（因近期服用雙重抗血小板藥物 DAPT） |
| 誘導 | 應極度謹慎 |
| 特點 | 患者高度依賴交感神經驅動 |

**準備：**
- 需隨時準備好血小板與血液製品

### 19.6.3 瓣膜手術要點

#### 主動脈瓣狹窄（Aortic Stenosis, AS）

| 目標 | 內容 |
|-----|------|
| 維持 MAP | 足夠的MAP/舒張壓以確保冠狀動脈灌注 |
| 心率 | **嚴禁心跳過快（Tachycardia）** |
| 心房收縮 | 維持心房收縮（Atrial kick）對肥厚的左心室充盈極為重要 |
| 藥物 | 避免使用血管擴張劑 |

#### 主動脈瓣逆流（Aortic Regurgitation, AR）

| 目標 | 內容 |
|-----|------|
| 心率 | 較快的心率有助於減少逆流時間 |
| 後負荷 | 血管擴張劑有助於減少後負荷 |
| 血流 | 增加前向血流 |
| 急性逆流 | 耐受度極差，常引發低心輸出量與呼吸衰竭 |

#### 二尖瓣狹窄（Mitral Stenosis, MS）

| 目標 | 內容 |
|-----|------|
| 前負荷 | 極度小心避免前負荷下降 |
| 碳酸血症/缺氧 | 避免（會惡化肺動脈高壓） |
| 輸液 | 嚴格控制（，左心房壓升高極易引發急性肺水腫） |

#### 二尖瓣逆流（Mitral Regurgitation, MR）

| 目標 | 內容 |
|-----|------|
| 前向血流 | 血管擴張劑有助於增加前向血流 |
| 術中評估 | TOE 對於評估修補效果至關重要 |
| 術後 | 置換術後常需強心劑支持功能不全的左心室 |

**瓣膜手術麻醉目標摘要：**

| 瓣膜病變 | 麻醉目標與處置重點 |
|---------|-------------------|
| 主動脈瓣狹窄 | 維持足夠MAP/舒張壓；嚴禁Tachycardia；維持心房收縮；避免血管擴張劑 |
| 主動脈瓣逆流 | 較快心率；血管擴張劑減少後負荷；增加前向血流 |
| 二尖瓣狹窄 | 避免前負荷下降；避免高碳酸血症/缺氧；嚴格控制輸液 |
| 二尖瓣逆流 | 血管擴張劑增加前向血流；TOE評估修補效果 |

## 19.7 深低溫停循環（DHCA）

用於主動脈弓手術或肺動脈內膜切除術，需完全停止血液循環以提供無血視野。

### 降溫保護

| 項目 | 內容 |
|-----|------|
| 傳統降溫溫度 | 降至 18°C |
| 建議 | 目前建議降溫至腦波圖（EEG）呈現等電位（Isoelectric EEG）為止 |

### 安全時間

| 項目 | 內容 |
|-----|------|
| 安全的停循環時間 | 通常認為不超過 40 分鐘（在等電位 EEG 下） |
| 超過 40 分鐘 | 應使用順行性選擇性腦灌注（SACP） |

### 併發症

| 併發症 | 說明 |
|--------|------|
| 凝血系統影響 | 嚴重影響 |
| 回溫策略 | 緩慢回溫有助於改善術後 6 週的認知功能表現 |

## 19.8 主動脈內氣球幫浦（IABP）

放置於降主動脈內（左鎖骨下動脈遠端 1-2 公分處）。

### 運作機制

| 時相 | 作用 |
|-----|------|
| 舒張早期 | 充氣，增加主動脈舒張壓與冠狀動脈血流 |
| 收縮期開始 | 迅速放氣，降低左心室射血的後負荷 |

### 適應症

| 適應症 |
|-------|
| 心肌梗塞併發症（心因性休克、急性二尖瓣逆流、心室中膈破裂） |
| 無法脫離 CPB |
| 術後低心輸出症候群 |

### 禁忌症

| 禁忌症 |
|-------|
| 中度以上主動脈瓣逆流 |
| 主動脈剝離/動脈瘤 |
| 嚴重周邊動脈疾病 |

## 19.9 肺動脈高壓與右心室

右心室（RV）對於後負荷（PAP）急劇升高的代償能力極差。

當 RV 衰竭時：
1. RV 會擴張
2. 導致心室中膈推向左心室
3. 進而減少 LV 前負荷
4. 導致心輸出量（CO）崩跌

### 處置目標

| 目標 | 具體措施 |
|-----|---------|
| 維持 RV 冠狀動脈灌注 | 維持 SVR 與 MAP |
| 避免 RV 後負荷增加 | 保持 PaCO₂ 偏低至正常值 |
| | **絕對避免缺氧** |
| | 避免過高的 PEEP 與氣道壓 |
| 肺血管擴張劑 | 吸入性一氧化氮（iNO） |
| | 霧化 Iloprost |
| | Milrinone |
| | GTN |
| 強心劑支持 | 低劑量 Adrenaline |
| | Dobutamine/Milrinone 改善 RV 收縮力 |
| 心律 | 維持竇性心律 |

## 19.10 心臟手術的爭議

### Aprotinin（抑肽酶）

| 項目 | 內容 |
|-----|------|
| 歷史 | 曾因增加腎功能障礙與死亡率疑慮被下架，後重新核准用於極高出血風險患者 |
| 現況 | 一線抗纖維溶解劑多使用 Tranexamic acid（TXA） |
| 風險 | TXA 會增加術後癲癇風險 |

### 輸血閾值（Transfusion Threshold）

大型試驗（TITRE2、TRICS 3）顯示：

| 策略 | 內容 |
|-----|------|
| 限制性輸血策略 | Hb <70 g/L 才輸血 |
| 結論 | 預後不劣於寬鬆性輸血策略 |

### 類固醇（Steroids）

| 項目 | 內容 |
|-----|------|
| 死亡率 | 大型試驗未能證明術中使用高劑量類固醇能顯著降低死亡率 |
| 可能益處 | 減少感染性併發症或改善恢復品質 |
