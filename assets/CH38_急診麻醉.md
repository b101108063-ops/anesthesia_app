## 重症成人氣管插管（Tracheal Intubation of Critically Ill Adults）

### 適應症

- 氣道阻塞且基本手法無效
- 即將發生的氣道阻塞
- 呼吸衰竭
- 意識減退（GCS ≤8）
- 損傷控制或復甦手術

### 誘導方案

| 患者類型 | Fentanyl | Ketamine | Rocuronium |
|----------|----------|----------|-------------|
| 無生理損害 | 3 mcg/kg | 2 mg/kg | 1 mg/kg |
| 生理不穩定 | 1 mcg/kg | 1 mg/kg | 1 mg/kg |

## 敗血症和敗血性休克（Sepsis and Septic Shock）

### 定義

| 術語 | 定義 |
|------|------|
| 敗血症 | 危及生命的器官功能障礙，由於失調的宿主反應 |
| qSOFA | 快速臨床篩檢：血壓 <100 mmHg、意識改變（GCS <15）、呼吸頻率 >22/min |
| 敗血性休克 | 需要血管收縮劑維持 MAP >65 mmHg 且乳酸 >2 mmol/L |

### 復甦目標（首小時）

| 項目 | 內容 |
|------|------|
| 乳酸測量 | 測量並複測（若 >2 mmol/L）|
| 血液培養 | 給予抗生素前 |
| 抗生素 | 廣覆蓋，根據當地微生物政策 |
| 液體 | 30 mL/kg 晶體液 |
| 血管收縮劑 | 若低血壓持續，維持 MAP >65 mmHg |

### 麻醉誘導

- 減少劑量
- Ketamine 1-2 mg/kg IV 常用
- Etomidate 與敗血症死亡率增加相關
- 準備好血管收縮劑（腎上腺素、去甲腎上腺素）

## 急診剖腹術（Emergency Laparotomy）

### 適應症

腸道阻塞、穿孔、感染、缺血和出血。

### 術前考量

| 項目 | 內容 |
|------|------|
| 風險評估 | 使用 NELA 風險計算機 |
| 抗生素 | 1 小時內覆蓋革蘭氏陽性和陰性菌 |
| 液體 | 平衡晶體液 |
| 血糖控制 | 目標 <10 mmol/L |

### 術中管理

| 項目 | 內容 |
|------|------|
| 監測 | 有創血壓監測 |
| 誘導 | 準備好血管收縮劑 |
| 通氣 | 肺保護策略 |
| 液體 | 目標導向治療 |

## 加護病房患者到手術室（ICU Patient Going to Theatre）

### 術前評估重點

| 系統 | 評估項目 |
|------|----------|
| 呼吸 | 目前呼吸支持程度 |
| 心血管 | 液體量、血管活性藥物劑量 |
| 確認 | 靜脈通路、實驗室數值 |

## 轉送重症患者（Transferring the Critically Ill）

### 轉送前檢查清單

- 氣道安全
- 呼吸機設定
- 監測儀器
- 藥物和液體
- 文件記錄
