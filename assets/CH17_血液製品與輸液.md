## 17.1 捐血與檢測

### 捐血類型

#### 全血捐贈

- 每位捐血者提供500 mL血液
- 其中470 mL用於加工，30 mL用於常規檢測
- 捐血者通常每年可捐血4次

#### 單採（成分捐血）

- 使用細胞分離器收集血漿、血小板、白細胞或造血祖細胞
- 捐血者的紅細胞會返回給捐血者
- 捐血者每2週可捐贈這些成分

### 檢測項目

捐贈的血液需進行以下檢測：

| 檢測項目 | 說明 |
|---------|------|
| ABO/RhD血型 | 確認血型 |
| 傳染病篩檢 | HIV、肝炎B、肝炎C、人類T細胞白血病病毒（HTLV）-1/2、梅毒 |
| 特殊地區風險 | 瘧疾或查格斯病的捐贈需檢測瘧原蟲和克氏錐蟲 |
| CMV | 為免疫抑制患者和新生兒提供CMV抗體陰性成分 |

### 血液加工

#### 加工方式

| 加工方式 | 目的 |
|---------|------|
| 預先儲存去白細胞 | 減少HTLV-1/2和CMV等血源性感染風險；防止同種異體免疫和發熱反應 |
| 離心和冷沉澱 | 分離紅細胞、血小板、血漿和蛋白質成分 |
| 照射 | 為有輸血相關移植物抗宿主病（TA-GvHD）風險的患者消除淋巴細胞 |
| 病毒去活化 | 血漿來源的產品（如FFP和冷沉澱）經過溶劑去污或亞甲藍處理 |

## 17.2 血液成分比較

### 血液成分比較表

| 特性 | 濃縮紅細胞（PRBCs） | 血小板 | 新鮮冰凍血漿（FFP） | 冷沉澱 |
|-----|---------------------|--------|---------------------|--------|
| **來源** | 單一捐血者全血離心 | 4-6個全血捐贈匯集或單採 | 男性捐血者單一捐贈快速冷凍 | 4-6個捐贈者匯集 |
| **組成** | 血細胞比容50-70%，懸於SaGM*中 | >240×10⁹血小板 | 0.5g纖維蛋白原、纖維蛋白原、vWF、因子VIII、因子XIII、纖連蛋白、因子VII>1.5 IU/mL | 2g纖維蛋白原、vWF、因子VIII、因子XIII、纖連蛋白 |
| **體積** | 300 mL | 100-200 mL | 300 mL | 匯集200-300 mL |
| **儲存** | 2-6°C | 20-24°C搖床上 | -30°C | -30°C |
| **保存期** | 35天 | 7天 | 2年（冷凍）；4小時（2-6°C解凍後）；4小時（室溫解凍後） | 2年（冷凍）；4小時（解凍後） |
| **劑量** | 4 mL/kg增加Hb 10g/L | 1單位增加血小板計數20-40×10⁹/L | 10-15 mL/kg | 每30kg體重1單位增加血漿纖維蛋白原1.0g/L |

*SaGM = 0.9%氯化鈉、腺嘌呤、葡萄糖和甘露醇

### 其他衍生產品

| 產品 | 說明 |
|-----|------|
| **纖維蛋白原濃縮物** | 凍乾粉末，含1g純化人類纖維蛋白原。室溫保存最多30個月。重新配製後可使患者纖維蛋白原增加0.25g/L。无需解凍或血型配對。適用於低纖維蛋白原血症、心臟手術、產科出血和重大創傷。 |
| **重組因子VIIa** | 凍乾粉末。用於先天性因子VII缺乏、血友病A或B伴VIII或IX因子抑制劑，以及嚴重無法控制的出血作為大量輸血的一部分。劑量應與血液學家討論。 |
| **凝血酶原複合物濃縮物（PCC）** | 無菌凍乾粉末，含500 IU純化人類因子II、IX和X。也含少量因子V和VII、25 IU抗凝血酶III和200 IU肝素。用於維生素K拮抗劑（如warfarin）的逆轉。无需交叉配對或解凍。 |
| **免疫球蛋白產品** | 含160 mg/mL人類血漿蛋白，98%是免疫球蛋白（主要為IgG）。特定免疫球蛋白包括破傷風、抗D和B型肝炎免疫球蛋白。主要通過慢速IM注射給藥，靜脈注射免疫球蛋白除外。 |
| **白蛋白** | 懸浮濃度通常為4-5%（40-50 g/L）和20%（200 g/L）。經過巴氏消毒以減少病毒疾病傳播。 |

## 17.3 安全輸血

### 輸血前檢查

| 步驟 | 內容 |
|-----|------|
| 知情同意 | 為合格患者提供有關風險、益處和替代療法的信息 |
| 身份確認 | 尽可能口头确认患者身份，並通過識別手環確認 |
| 血袋核對 | 將待輸血單位與處方核對；檢查有效期；確保實驗室標籤與血袋本身上的單位編號匹配 |
| 血袋檢查 | 檢查完整性、變色或凝集跡象 |
| 輸血器 | 使用170-200微米 integral screen filter的輸血套裝；正常輸血時每套可用於4單位濃縮紅細胞；大量輸血時可用8-10單位；血小板不應通過已用於紅細胞的套裝輸注 |
| 監測 | 監測不良事件 |
| 文件 | 100%追蹤是法律要求 |

### 單一單位輸血

在穩定、非出血、正常血容量患者中，輸血應每次1單位進行。

### 成分输血時間限制

| 成分 | 開始時間 | 完成時間 |
|-----|---------|---------|
| 濃縮紅細胞 | <30分鐘 | <4小時 |
| 血小板 | 立即 | <1小時 |
| FFP | <30分鐘 | <4小時 |
| 冷沉澱 | 立即 | <4小時 |

## 17.4 相容性檢測

### 血型與篩檢

- 檢測患者的ABO和RhD抗原類型
- 檢測紅細胞抗體
- 樣本有效期取決於患者的輸血和產科病史，範圍從72小時到21天
- 輸血開始後，樣本有效期為72小時

### ABO相容性

ABO相容性可防止由接受者IgM抗體結合捐贈者紅細胞上的A或B抗原引起的急性溶血性輸血反應。

**ABO成分和接受者相容性表：**

| 接受者 | 相容捐贈者 | | |
|-------|-----------|---|---|
| | 紅細胞 | 血小板 | FFP/冷沉澱 |
| O | O | O或A | O、A、B或AB |
| A | A或O | A或O | A或AB |
| B | B或O | B或O | B或AB |
| AB | AB、A、B或O | AB、A、B或O | AB |
| 未知 | O | O | AB |

### RhD相容性

- 抗RhD（也稱為抗D）是對RhD陰性接受者預防未來新生兒溶血性疾病的重要因素
- 所有具有生育潛力的女性都應給予RhD相容血液
- 血小板不表達RhD抗原，但應給予RhD相容的單位
- 冷沉澱和FFP可能含有紅細胞碎片，但任何RhD類型的冷沉澱和FFP都可以安全給予

### 抗體篩檢陽性

有陽性抗體篩檢的患者可能需要特定抗原陰性血液成分。並非所有檢測到的抗體都有臨床意義，需要與血庫和輸血專家討論。

### 未交叉配對血液

- 在沒有血型和篩檢的情況下，立即發放緊急O RhD陰性血液僅保留用於出血緊急情況
- 獲取輸血交叉配對的血液樣本是出血管理早期的優先事項
- 特血型血液（ABO/RhD）比完全分型和篩檢血液發放快得多（15分鐘 vs 45分鐘）

## 17.5 輸血適應症和觸發點

### 紅細胞輸血觸發點

| Hb水平 | 紅細胞輸血適應症 |
|--------|----------------|
| <70 g/L | 臨床穩定ICU患者的輸血閾值 |
| <80 g/L | 急性冠狀動脈綜合徵、髖部骨折或心血管疾病患者 |
| 70-100 g/L | 大手術中大失血或缺氧證據 |
| >90 g/L | 輸血通常不合適。例外包括組織氧輸送受損，如腦缺血或敗血症 |

### 冷沉澱輸血觸發點

| 情況 | 冷沉澱輸血適應症 |
|-----|----------------|
| 大量輸血 | 經驗性給予；目標纖維蛋白原>1.5 g/L |
| 產科出血 | 經驗性給予；目標纖維蛋白原>1.5 g/L |
| DIC | 如果有臨床出血和消耗性凝血病。目標纖維蛋白原水平>1.0 g/L |

功能纖維蛋白原檢測可能比絕對水平更準確地指導紅血球減少的冷沉澱輸血。

### 血小板輸血觸發點

| 血小板計數 | 血小板輸血適應症 |
|-----------|----------------|
| <10×10⁹/L | 輸血指示，因為有自發性出血風險 |
| <20×10⁹/L | 在有出血風險因素（輕微出血、敗血症）或重症患者中指示輸血 |
| <50×10⁹/L | 大多數侵入性手術需要輸血。AOA建議區域阻滯時>50×10⁹/L |
| <75×10⁹/L | 大量出血時需要輸血以持續維持血小板計數>50×10⁹/L |
| <100×10⁹/L | 高風險封閉腔室手術（眼科、神經外科）需要輸血 |
| 其他 | 無論血小板計數如何，藥理學血小板抑制或遺傳性血小板功能障礙的情況下輸血可能合適 |

**禁忌症/注意事項：**TTP、HUS、HIT、免疫性血小板減少性紫癜

### FFP輸血觸發點

| 情況 | FFP輸血適應症 |
|-----|--------------|
| 凝血因子/蛋白質缺乏 | 僅在特定因子不可用時 |
| 大量輸血 | 經驗性以1:1比例與RBC給予，或15 mL/kg每血容量。維持INR<1.5 |
| Warfarin逆轉 | 當PCC不合適或不可用時使用FFP |
| TTP | 用於血漿交換治療 |
| DIC | 如果DIC伴隨出血指示。初始劑量為15 mL/kg以維持INR<1.5 |

## 17.6 輸血風險

### 常見風險

| 風險 | 發生率 | 說明 |
|-----|--------|------|
| **發熱性非溶血性輸血反應** | 1-3:100 | 輸血30分鐘內發生，由細胞因子或對污染白細胞的同種免疫反應介導。輕度反應給予簡單降溫和撲熱息痛。減慢輸注可緩解。 |
| **輕度過敏** | 1-5:500 | 常見表現為黏膜皮膚表現、潮紅、血管性水腫或蕁麻疹。 |
| **低血壓** | 1-2:1000 | 特異反應，可能與接受ACE抑制劑患者輸血中的緩激肽有關。定義為收縮壓下降30 mmHg。 |
| **輸血相關循環超負荷（TACO）** | 1-10:1000 | 發生率由輸血量和患者合併症共同決定。 |
| **低體溫** | 常見 | 大容量血液製品快速輸注時普遍存在。可加重所有生理過程，包括心血管功能和凝血。 |
| **免疫抑制** | 可能 | 輸血可能通過免疫調節影響惡性腫瘤復發或擴散以及術後細菌感染的發生率。 |

### 罕見風險

| 風險 | 發生率 | 說明 |
|-----|--------|------|
| **急性溶血性輸血反應** | 1-8:100,000 | 嚴重、危及生命的反應，由ABO不相容引起。症狀可能與細菌敗血症或過敏反應無法區分。 |
| **延遲溶血性輸血反應** | 1:5000 | 通常在7天內發生，但可在輸血後最多28天發生。表現為黃疸、貧血，罕見脾腫大和腎損傷。 |
| **細菌敗血症** | <1:10,000 | 罕見，但這是輸血相關死亡的主要原因。血小板輸血風險最高（室溫儲存）。污染紅細胞輸血導致的敗血症死亡率為60%。 |
| **過敏反應** | <1:20,000 | 接受者免疫反應可針對任何成分。 |
| **病毒感染** | HIV 1:920萬；HCV 1:690萬；HBV 1:80萬 | 捐贈者篩檢可減少大多數風險。窗口期捐贈仍有殘餘風險。 |
| **輸血後紫癜** | <1:100,000 | 最常見於先前懷孕過的女性。 |
| **TRALI** | <1:5000 | 輸血後6小時內發生的非心源性肺水腫。與ARDS臨床無法區分。使用男性血漿成分可降低風險（因為多次妊娠女性有更高滴度的抗人中性粒細胞抗原和抗人白細胞抗原）。 |
| **TA-GvHD** | 通常致命 | 輸血後1-6週發生。免疫抑制宿主中移植物抗宿主淋巴細胞增殖並通過自身免疫引起多器官衰竭和死亡。 |

### 血液加溫器

**不常規需要。適應症包括：**

- 大量輸血
- 新生兒交換輸血
- CPB期間復溫
- 有臨床顯著冷反應抗體（如冷凝集素病）患者的輸血

**注意事項：** 血液成分不得加熱>41°C；加溫器應配備可見溫度計和聲音警報以避免溶血。

## 17.7 患者血液管理

圍手術期貧血是發病率和死亡率的獨立預測因子，輸血是不良臨床結局的進一步獨立預測因子。

患者血液管理是一種基於證據的多學科方法，旨在減少同種異體血液輸血需求並改善患者結局。

### 三個支柱

| 支柱 | 內容 |
|-----|------|
| 改善紅細胞質量 | 通過調查貧血、鐵替代、B12和葉酸替代、造血生長因子 |
| 減少失血 | 識別出血病史、抗凝管理、外科技術、麻醉技術、氨甲環酸、去氨加壓素、紅細胞回收、自體輸血、靜脈切開放血 |
| 優化對貧血的生理耐受性 | 通過優化心肺功能、減少組織代謝需求 |

### 改善紅細胞質量

**貧血定義（WHO）：**
- 女性：Hb <120 g/L
- 男性：Hb <130 g/L

**干預措施：**
- 調查貧血原因
- 鐵替代
- B12和葉酸替代
- 造血生長因子（如人類重組EPO，對腎衰竭患者最有效）

### 減少失血

| 方法 | 說明 |
|-----|------|
| **病史** | 識別先前出血、月經過多、牙科出血、易瘀傷和遺傳性出血傾向 |
| **抗凝管理** | 如適當停用抗凝/抗血小板藥物 |
| **外科技術** | 微創/經皮手術、局部血管收縮、局部止血劑、止血帶、超聲或激光手術刀 |
| **麻醉技術** | 通過避免高胸內壓、仔細患者定位和避免高碳酸血症減少靜脈壓；通過維持正常體溫和避免酸中毒和低鈣血症優化凝血 |
| **氨甲環酸** | 廣泛使用的抗纖溶藥物，用於創傷、產科、心臟手術和骨科 |
| **去氨加壓素** | 合成ADH類似物，用於von Willebrand病和改善尿毒症、肝硬化和其他血小板功能障礙的血小板功能 |
| **紅細胞回收** | 適用於預測失血>500 mL或>10%總血容量的患者。絕對禁忌症包括不應IV給予的物質（如氯己定沖洗或纖維蛋白膠）。相對禁忌症需要風險-益處評估和患者同意，包括收集含有膿毒性材料、惡性細胞和羊水的血液 |
| **自體輸血** | 術前自體捐血：手術前每週捐血一次，持續4週。急性正常血容量血液稀釋：術前立即收集全血；用晶體液擴容維持正常血容量 |
| **靜脈切開放血** | 避免不必要的血液採樣，所需檢驗最少化血液採樣量 |

### 優化對貧血的生理耐受性

**方法：**
- **氧合：** 增加FiO₂、高通氣
- **組織灌注：** 用液體治療、血管加壓素和正性肌力藥物增加CO和組織灌注
- **O₂消耗：** 通過治療疼痛和感染減少。嚴重危及生命的貧血中，低溫、高壓氧、鎮靜、肌肉麻痹和機械通氣也可以減少代謝工作

## 17.8 大量輸血

### 定義

- 4小時內輸注>5單位PRBCs
- 24小時內輸注>10單位PRBCs

### 治療目標

- 快速、經濟、無不良反應地擴充血容量
- 目前嚴重出血復甦的護理標準是以1:1:1比例給予血液成分（血小板、血漿和PRBCs）

### 大量出血和輸血的病理生理學

**死亡三聯徵：**

| 組成 | 說明 |
|-----|------|
| 酸中毒 | 快速大容量出血導致全身低灌注和組織氧輸送減少 |
| 低體溫 | 冷IV液體和患者暴露於血管通路和復甦 |
| 凝血病 | 低灌注激活蛋白C途徑，滅活因子Va和VIIIa，啟動纖溶和凝血病 |

**機制：**
- 酸中毒、體溫過低和凝血病各自加重其他兩種成分
- 低灌注也消耗凝血因子，可能導致DIC
- 低溫和酸中毒降低心肌收縮力，導致心動過緩和心律失常
- 它們還降低凝血因子和血小板活性

### 大量出血和輸血期間的復甦優先事項

#### 人員和協助

| 需求 | 內容 |
|-----|------|
| 額外協助 | 呼叫額外的高級外科和麻醉幫助 |
| 危機領導 | 指定一名危機領導者來管理和分配角色 |
| 血庫通知 | 通知血庫並宣布需要大量輸血；這將導致更多血庫工作人員專門負責交叉配對和解凍血液製品 |
| 專門傳遞者 | 專門從血庫獲取血液製品 |
| 額外工作人員 | 檢查/管理到達的血液製品 |
| 額外手術室工作人員 | 設置關鍵設備 |

#### 維持循環容量

| 步驟 | 內容 |
|-----|------|
| 快速出血控制 | 獲得快速出血控制（如外科、內鏡、介入放射學） |
| 立即請求 | 請求3單位PRBCs；如果交叉配對血液不能立即獲得，使用O陰性緊急血液 |
| 獲得通路 | 獲得大孔徑IV通路；血液到達前給予晶體液 |
| 增加氧合 | 增加FiO₂至100%以改善組織氧輸送。考慮減少給予的麻醉劑劑量 |

#### 請求額外設備

| 設備 | 說明 |
|-----|------|
| 快速輸注裝置 | 如Belmont™或Level-1™ |
| 細胞回收 | 部署細胞回收 |

#### 預防低體溫

- 強制空氣加溫裝置（如Bair Hugger™）
- 增加環境室溫
- 通過液體加溫器輸注

#### 預防和治療凝血病

| 方法 | 內容 |
|-----|------|
| 平衡輸血 | 以1:1:1比例平衡輸血 |
| 即時檢測 | 識別因子缺乏和檢測DIC |
| 實驗室檢測 | 每30分鐘進行ABG、正式FBC、凝血研究和電解質 |
| 抗纖溶藥物 | 如果在創傷事件後3小時內，考慮抗纖溶藥物（氨甲環酸1g IV超過10分鐘） |
| 重組因子VIIa | 如果手術出血控制且pH>7.2，考慮使用（NovoSeven®；90 mcg/kg IV）。血栓形成風險患者謹慎使用，並與血液學家討論 |

#### 治療大量輸血的後遺症

- 用慢速IV注射10 mL 10%氯化鈣治療低Ca²⁺和 高K⁺

### 治療目標

| 參數 | 目標 |
|-----|------|
| 平均動脈壓 | 不要瞄準正常化，直到有外科控制（>50 mmHg；如有頭部損傷則>70 mmHg） |
| 血紅蛋白 | >70 g/L |
| 纖維蛋白原 | >1.5 g/L（產科患者>2 g/L） |
| 血小板 | >75×10⁹/L |
| 離子鈣 | >1 mmol/L（血氣樣本） |
| pH | 7.35-7.45 |
| PT/APTT | <正常/基線上限的1.5倍 |

## 17.9 耶和華見證人患者

### 血液製品接受態度

**拒絕的「主要血液成分」：**
- 全血（包括術前自體捐血）
- PRBCs、血小板、白細胞和FFP

**可變接受度的「血液衍生物」：**
- 冷沉澱、凝血因子濃縮物、白蛋白、免疫球蛋白

**通常可接受的：**
- 術中細胞回收（如管路用非血液液體預充）
- 血液透析、CPB、正常血容量血液稀釋
- IV鐵和氨甲環酸

### 倫理和法律考量

#### 合格患者

- 合格患者有權利拒絕輸血
- 麻醉師有義務告知患者其決定的風險、益處和替代方案
- 臨床醫生必須確認患者的決定是自願的
- 向拒绝输血的合格患者给予血液製品是違法的

#### 預先指令

- 說明患者對特定醫療干預的接受或拒絕
- 必須包含清晰且醫學上可解釋的說明
- 必須明確這些指令即使在患者生命受到威脅時也適用

#### 急診手術

| 情況 | 處理 |
|-----|------|
| 有意識且合格 | 必須為患者提供知情同意的機會，並尊重其意願 |
| 無意識或不合格 | 在患者意願未知的情況下，醫生必須以患者最大利益行事，這可能包括給予輸血 |

#### 兒童

- 如果父母拒絕輸血同意，可能需要申請法律監護權
- 在緊急情況下，當耶和華見證人的孩子沒有輸血可能會死亡時，應給予輸血
- 法院可能會支持這一醫療決定

#### 擇期手術

| 階段 | 建議 |
|-----|------|
| 早期評估 | 盡早見患者以最大化溝通、優化和計劃的可用時間 |
| 患者血液管理 | 術前優化紅細胞質量；最小化出血；優化對貧血的生理耐受性。患者應在大手術前>6週評估以確保Hb>130 g/L |
| 多學科投入 | 製定全面管理策略 |
| 溝通 | 在每個計劃階段與患者清晰溝通 |
| 文件 | 記錄所有會議、討論、通信、同意書和預先指令 |
| 手術計劃 | 考慮微創手術和分期手術；考虑介入放射學醫生的角色 |
| 自體輸血 | 強烈考慮紅細胞回收；急性正常血容量血液稀釋也可能可接受 |
| 術中 | 在團隊簡報和手術安全檢查清單中突出問題 |
| 術後 | 預期對重症監護的需求；確保徹底交接，包括患者意願 |

## 17.10 液體治療

### 正常禁食影響

- 常規禁食對輕微手術通常耐受良好
- 對液體狀態或電解質水平的干擾程度取決於禁食時間、手术范围、年齡、患者合併症和急性病理的性質

### 正常液體和電解質平衡

#### 每日攝入和輸出

| 類別 | 項目 | 體積 |
|-----|------|------|
| **攝入** | 液體 | 1500 mL |
| | 食物 | 750 mL |
| | 代謝水 | 250 mL |
| **總計** | | 2500 mL |
| **輸出** | 尿液 | 1500 mL |
| | 糞便 | 100 mL |
| | 不感蒸發丟失 | 900 mL |

#### 每日電解質需求

| 電解質 | 血漿濃度（mmol/L） | 每日需求（mmol/kg/d） |
|--------|--------------------|----------------------|
| Na⁺ | 135-145 | 1-1.5 |
| K⁺ | 3.5-5.0 | 1-1.5 |
| Mg²⁺ | 0.75-1.05 | 0.1-0.2 |
| Ca²⁺ | 2.12-2.65（總量）、1.0（離子） | 0.1-0.2 |
| Cl⁻ | 95-105 | 0.07-0.22 |
| 磷酸鹽 | 0.8-1.45 | 20-40 |

### 液體隔室

| 隔室 | 百分比 | 體積 |
|-----|--------|------|
| 總體水 | 60%體重（600 mL/kg） | - |
| 細胞內液 | 66%體重（400 mL/kg） | - |
| 細胞外液（ECF） | 33%體重（200 mL/kg） | - |
| 間質液 | 75% ECF（150 mL/kg） | - |
| 血漿 | 25% ECF（50 mL/kg） | - |

### 糖萼層

#### 結構

- 血管內皮腔側的膜結合、碳水化合物豐富的蛋白聚糖和糖蛋白層

#### 功能

- 血管通透性、凝血、炎症、抗氧化和調節經腔腫滲透壓
- 700-1000 mL血漿量的非循環容積固定在內皮糖萼層內
- 理論上，等渗晶體液會通過其中擴散，而膠體液不會

#### 病理

- 創傷、代謝/滲透損傷、炎症、低血容量、高血容量和人工膠體都會輕易擾亂糖萼層
- 這被統稱為休克誘導的內皮病，導致液體從血管內向間質室移動

### 液體狀態評估

#### 主訴和病史

| 項目 | 內容 |
|-----|------|
| 禁食時間 | 禁食時間長短 |
| 口服攝入/胃腸道損失 | 自我報告 |
| 出血 | 估計失血量 |
| 透析 | 距離上次透析的時間 |
| 藥物 | 利尿劑、抗高血壓藥物 |

#### 臨床評估

**脫水體徵和症狀：**

| 體重減輕（%） | 體徵和症狀 |
|--------------|-----------|
| 5 | 口渴、口乾 |
| 5-10 | 周圍灌注減少、皮膚彈性減少、體位性頭暈、少尿、CVP降低、倦怠、心動過速 |
| 10-15 | 呼吸頻率增加、低血壓、無尿、譫妄、昏迷 |
| >15 | 危及生命 |

#### 液體反應性測試

| 測試 | 說明 |
|-----|------|
| 被動抬腿 | 增加容量不足患者的前負荷以增加每搏量和血壓。這是評估低血壓患者液體治療的有用性的簡單、高度預測性的方法。如果為陰性，患者不太可能是液體應答者。 |
| SVV/PPV | 正壓通氣的胸內壓周期性變化誘導左室前負荷變化。SVV和PPV中的這種變化可預測液體反應性。 |
| CVP | CVP<8 mmHg與液體反應性相關。血流動力學穩定患者的CVP監測不太可能有幫助。 |
| 血流動力學監測 | PPV、食管超聲多普勒、PICCO™、LiDCO™ rapid、PA導管和Cheetah NICOM®，特別是配合動態液體挑戰，可能是測量和補充丟失液體的最可靠方法。 |
| 胸部X光 | 提供心臟擴大、肺充血和液體超負荷導致肺水腫的證據 |
| 實驗室檢查 | 組織低灌注將表現為酸中毒、酸血症、乳酸血症和鹼剩餘減少。肌酐/尿素/血細胞比容升高、CC減少和低尿 acidity/Na⁺濃度表示脫水 |
| 超聲心動圖 | 量化心室功能，以及通過腔靜脈和心室充盈的前負荷。腔靜脈直徑呼吸變異>15%與液體反應性相關 |

### 動態液體挑戰

| 步驟 | 內容 |
|-----|------|
| 晶體bolus | 使用100-200 mL晶體bolus |
| 評估 | 評估對bolus的臨床反應 |
| 血流動力學終點 | 尿量、心率、每搏輸出量、血壓、CVP、PPV、SVV、乳酸、pH等改善 |
| 警惕 | 對血流動力學損害的其他原因保持警惕 |
| 累計評估 | 評估累計失血和輸血需求 |
| 重複 | 頻繁重新評估，重複bolus |

## 17.11 輸液類型

### 晶體液

#### 平衡鹽溶液

**示例：** Hartmann's solution、Plasma-Lyte 148®

| 特性 | 說明 |
|-----|------|
| 地位 | 圍手術期一線替代治療 |
| 用途 | 所有大手術的主要維持液，1-3 mL/kg/h |
| 優點 | 生理成分，用於替代ECF；有機陰離子抵消氯離子濃度；可能減少與高氯溶液輸注相關的醫源性高氯性代謝性酸中毒和相關腎功能障礙 |
| 限制 | K⁺和Ca²⁺的添加可能限制其在高K⁺或含檸檬酸血液輸血中的用處 |

#### 0.9%氯化鈉

| 特性 | 說明 |
|-----|------|
| 用途 | 電解質替代和用於替換富含電解質的胃腸道損失 |
| 風險 | 與高氯性酸中毒相關，可能誘導腎低灌注和炎症；可能導致重症和非重症患者的AKI |

#### 葡萄糖溶液

**示例：** 5%葡萄糖、4%-0.18%葡萄糖氯化鈉

| 特性 | 說明 |
|-----|------|
| 5%葡萄糖 | 方便的給予游離水的方式，用於恢復與水分流失相關的脫水 |
| 風險 | 過度使用可能導致低Na⁺；在成人日常液體管理中作用不大 |
| 局限性 | 不是有效的血漿擴張劑（因為有害的滲透效應）；不是有效的代謝能量來源 |

### 膠體液

#### 人血清白蛋白

| 特性 | 說明 |
|-----|------|
| 分子量 | 69,000 |
| 可用濃度 | 4-5%溶液用於治療低血容量；20%溶液用於治療低白蛋白血症 |
| 安全性 | 未顯示可改善ICU患者的存活率；未顯示可改善嚴重敗血症的治療 |
| 適應症 | 低血容量性休克的治療（但TBI患者復甦時與死亡率增加相關）；CPB中的預充液；血漿交換中的置換液 |

## 17.12 液體治療策略

### 自由 vs 限制性液體

**歷史背景：**
- 自由策略：旨在替補「第三間隙」液體轉移和手術的「不感蒸發」丟失
- 限制策略：旨在實現「零平衡」或術後液體瀦留導致的零體重增加
- 許多ERAS途徑包含限制性策略

**RELIEF試驗（2018）發現：**
- 3000名接受大手術的患者，限制性和自由性手臂
- 自由性手臂的AKI和腎臟替代治療率較低
- 兩組的無殘疾存活率相同

**當前建議：**
- 支持大手術的「適度自由」液體方案
- 目標：術中10-12 mL/kg/h IV液體，術後24小時內1.5 mL/kg/h
- 不導致大幅液體轉移的其他大手術可能不需要這麼多液體

### 目標導向液體治療（GDFT）

| 特性 | 說明 |
|-----|------|
| 定義 | 使用液體和正性肌力藥物的組合，以預定算法給予，以優化器官血流 |
| 目標 | 依次給予IV液體和正性肌力藥物以達到特定血流動力學終點（CVP、MAP、每搏輸出量、CO、心臟指數等） |
| 證據 | 臨床上有吸引力的想法，但小型試驗中看到的臨床益處在後續文獻中未得到證實 |
| 現狀 | 仍然是全世界腹部手術ERAS方案的一部分 |
