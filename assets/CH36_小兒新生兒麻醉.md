## 新生兒/嬰兒生理學（Neonatal/Infant Physiology）

### 定義

| 術語 | 定義 |
|------|------|
| 新生兒 | 足月後首 44 週 |
| 嬰兒 | 1 個月至 1 歲 |
| 兒童 | 1-16 歲 |
| 早產兒 | 妊娠 <37 週 |

### 呼吸系統考量

| 參數 | 新生兒 | 成人 |
|------|--------|------|
| 潮氣量（mL/kg） | 7 | 7-10 |
| 呼吸頻率（/min） | 30-40 | 15 |
| 順應性（mL/cmH2O） | 5 | 100 |
| 阻力（cmH2O/L/s） | 25 | 5 |

- 胸壁順應性高，肋軟骨水平排列
- 換氣基本為橫膈依賴性
- 關閉容量在潮氣容積內
- 麻醉誘導和甦醒比成人快

### 心血管系統考量

| 參數 | 新生兒 |
|------|--------|
| 心輸出量 | 高（200 mL/kg/min）|
| 正常心率 | 可達 200 bpm |
| 正常收縮壓 | 70-90 mmHg |

### 血液系統考量

| 估計血量 | |
|----------|
| 新生兒 | 90 mL/kg |
| 嬰兒 | 85 mL/kg |
| 兒童 | 80 mL/kg |

## 液體管理（Fluid Management）

### 新生兒液體需求（前 5 天）

| 生命天數 | 需求量（mL/kg/day）|
|----------|---------------------|
| 第 1 天 | 50-60 |
| 第 2 天 | 70-80 |
| 第 3 天 | 80-100 |
| 第 4 天 | 100-120 |
| 第 5 天 | 120-150 |

### 兒童維持液體（4-2-1 法則）

- 每第一個 10 kg：4 mL/kg/h
- 每第二個 10 kg：2 mL/kg/h
- 每額外 kg：1 mL/kg/h

### 脫水評估

| 徵象 | 5% 脫水 | 10% 脫水 |
|------|-----------|-----------|
| 皮膚 | 張力喪失 | 斑駁、毛細血管回覆差 |
| 鹵門 |凹陷 | 深凹陷 |
| 眼睛 | 凹陷 | 深度凹陷 |
| 心智狀態 | 嗜睡 | 無反應 |

### 休克處理

| 休克類型 | 處理 |
|----------|------|
| 低血容量（最常見）| 晶體液 20 mL/kg |
| 心因性 | 正性肌力藥物 |
| 分布性（敗血症）| 液體 + 抗生素 |
| 阻塞性 | 解除病因 |

## 麻醉執行（Conduct of Anaesthesia）

### 術前評估

- 建立與父母和兒童的 rapport
- 詢問兒童本身（適當時）
- 術前訪視減少父母焦慮

### 上呼吸道感染兒童

| 情況 | 建議 |
|------|------|
| 輕度 URI | 可進行手術 |
| 中重度胸部感染 | 延期 |
| 明顯 URI | 延期至少 2 週 |

### 焦慮或不合作兒童

- 父母陪伴誘導
- 分散注意力技巧
- 必要時術前用藥

### 術前禁食

| 攝入物 | 最短禁食時間（小時）|
|--------|---------------------|
| 清液 | 1 |
| 母乳 | 4 |
| 配方奶 | 6 |

## 新生兒/嬰兒特殊考量

### 體溫調節

**保暖措施：**
- 手術室預熱（21-26°C）
- 覆蓋頭部
- 保溫毯
- 加溫靜脈液體

### 藥物考量

- 氯胺酮：分泌過多，術後譫妄
- Midazolam：常用，靜脈製劑極苦
- Clonidine：無味，降低術後鎮痛需求

## 小兒急症（Paediatric Emergencies）

### 小兒心肺復甦

| 年齡 | 胸外按壓深度 | 按壓/通氣比 |
|------|-------------|-------------|
| 嬰兒（<1歲）| 約 4 cm | 15:2 |
| 兒童（1-8歲）| 約 5 cm | 15:2 |

### 新生兒復甦

| 步驟 | 說明 |
|------|------|
| 初始評估 | 呼吸、心率、顏色 |
| 、保暖 | 輻射加溫台 |
| 清理氣道 | 必要時 |
| 刺激 | 摩擦背部 |
| 正壓換氣 | 21% 氧氣 |
| 胸外按壓 | 若心率 <60 |

| 項目 | 內容 |

| 疼痛程度 | + |

| 項目 | 內容 |
|------|------|

## 區域麻醉（Regional Anaesthesia）

### 骶管阻斷

| 項目 | 說明 |
|------|------|
| 適應症 | 下腹部/下肢手術 |
| 劑量 | 0.5-1 mL/kg Bupivacaine 0.25% |
| 最大劑量 | 20 mL |

### 尾側阻斷

- 最常用的小兒區域麻醉技術
- 單次注射操作簡單
- 輔助全身麻醉提供術後鎮痛
