## 23.1 一般原則

英格蘭和威爾斯每年約進行220,000例主要關節置換手術。

### 患者特點

| 特點 | 說明 |
|-----|------|
| 人口老齡化 | 患者現在更可能肥胖並有多種合併症 |
| 手術類型 | 許多手術適合區域麻醉和超聲引導的LA阻滯 |
| 患者教育 | 關節學校促進患者選擇和突出麻醉技術等領域 |

### 術前考量

| 項目 | 內容 |
|-----|------|
| 外科聯繫 | 特別是進行區域技術時 |
| 關節炎 | 常使心肺 fitness評估困難 |
| 區域技術 | 如果計劃CNB，考慮影響凝血的因素（最後一次抗凝劑的時間） |
| VTE風險 | 某些手術VTE風險高，需要抗血栓措施（如LMWh、彈力襪、足泵） |

### 圍手術期考量

| 項目 | 內容 |
|-----|------|
| 抗生素 | 根據當地方案給予IV抗生素預防（注意過敏反應風險） |
| 體位 | 避免軟組織或神經損傷（這是麻醉師和外科醫生的共同責任） |
| 溫度 | 通過血液加溫器和強制空氣加溫毯維持正常體溫可減少發病率和死亡率 |
| 低血壓 | 應避免 |
| 侵入性監測 | 考慮用於有心血管疾病和需要翻修下肢手術的患者 |
| 失血 | 可能顯著；準確監測失血；考慮細胞回收（包括引流回收） |
| 導尿 | 長時間手術或使用硬膜外/脊髓阿片類藥物時應插入 |
| LIA | 局部浸潤鎮痛（使用低劑量、高容量LA）是有效的鎮痛技術 |

### 術後考量

| 項目 | 內容 |
|-----|------|
| 鎮痛 | 良好的鎮痛對康復、活動和出院有積極影響 |
| NSAID | 與外科醫生聯繫後開處方；75歲以上患者謹慎使用 |

## 23.2 區域麻醉

### 優勢

| 優勢 | 說明 |
|-----|------|
| 術後鎮痛 | 可能減少PONV |
| 結果改善 | 有證據表明區域麻醉單獨或與GA聯合可能改善髖關節和膝關節置換術的結局 |
| 手術野 | 區域麻醉（特別是脊髓/硬膜外）減少的手術部位出血（不需要其他藥理學低血壓麻醉技術） |
| 外科醫生偏好 | 外科醫生通常偏好區域技術產生的手術條件 |

## 23.3 脂肪栓塞綜合徵

### 定義

脂肪栓塞綜合徵（FES）與創傷或手術相關，發生率約1%。

### 特徵（根據Gurd定義）

**主要：**

| 特徵 | 說明 |
|-----|------|
| 呼吸症狀 | 呼吸急促、呼吸困難、雙側爆裂音、咳血、CXR彌散陰影 |
| 神經學體徵 | 混亂、嗜睡 |
| 瘀點皮疹 | - |

**次要：**

| 特徵 | 說明 |
|-----|------|
| 心動過速 | - |
| 視網膜改變 | 脂肪或瘀點 |
| 黃疸 | - |
| 腎臟 | 少尿或無尿 |

**實驗室：**

| 發現 | 說明 |
|-----|------|
| 血小板減少 | - |
| 血紅蛋白突然下降20% | - |
| ESR升高 | - |
| 脂肪巨球蛋白血症 | - |

### 治療

| 治療 | 內容 |
|-----|------|
| 早期復甦和穩定 | 至關重要 |
| 早期O₂治療 | 可能預防綜合徵發作 |
| 機械通氣 | 10-40%患者可能需要 |
| 類固醇 | 存在爭議 |
| 病程 | FES通常在7天內消退 |

## 23.4 骨水泥植入綜合徵

### 定義

甲基丙烯酸甲酯骨水泥是一種丙烯酸聚合物，在骨科手術中已廣泛使用40年。

### 特點

| 特點 | 說明 |
|-----|------|
| 潛在風險 | 缺氧、低血壓和心血管衰竭 |
| 併發症 | 報告的致命心臟驟停 |
| 分類 | BCIS分級從1級（輕度低血壓和缺氧）到3級（心血管衰竭） |

### 預防和治療

**高風險患者：** 有顯著心肺疾病和服用利尿劑的患者。

**共識措施：**
- 「水泥宵禁」應作為WHO檢查清單的一部分
- 劇院團隊每個成員在假體插入周圍應有特定角色且不應離開劇院

**手術和麻醉執行建議：**

| 角色 | 建議 |
|-----|------|
| 外科醫生 | 確認股骨髓腔準備開始；推薦使用加壓灌洗系統清潔骨內膜脂肪和骨髓；遠端吸引導管在髓內塞上方； retrograde方式插入水泥；高風險患者避免過度加壓 |
| 麻醉師 | 術前和術中保證充足水分；股骨頭移除後保持警覺；當外科醫生溝通股骨髓腔準備時封閉迴路溝通；整個過程維持收縮壓在誘導前值的20%以內；使用血管加壓素和/或液體；高風險患者使用侵入性血壓監測；為心血管衰竭做好準備（間羥胺/腎上腺素） |

## 23.5 止血帶

### 使用原則

| 原則 | 說明 |
|-----|------|
| 類型 | 僅使用氣動止血帶；機械止血帶可造成下方組織不可預測的高壓區域 |
| 手指/腳趾小止血帶 | 危險（容易被遺忘）；使用橡膠條配合止血鉗 |
| 驅血 | 腫瘤或嚴重感染時禁用（傳播風險）；疑似DVT時禁用；可能導致LV衰竭（雙腿同時壓迫增加循環量的15%） |

### 應用部位

| 部位 | 建議 |
|-----|------|
| 上臂和大腿 | 有足夠肌肉體積均勻分布氣囊壓力；推薦 |
| 短手術（<1小時）fit患者 | 一些外科醫生偏好小腿止血帶 |

### 氣囊寬度

| 原則 | 說明 |
|-----|------|
| 上臂 | 寬度為直徑的40%或更大（不超過20 cm） |
| 大腿 | 寬度為周長的40% |
| 現代矽膠氣囊 | 通常測量上臂90 mm（氣囊70 mm）和大腿105 mm（氣囊75 mm） |

### 壓力設置

| 部位 | 壓力建議 |
|-----|---------|
| 上肢 | 收縮壓 + 50 mmHg |
| 下肢 | 收縮壓的兩倍 |

### 止血帶時間

| 建議 | 內容 |
|-----|------|
| 目標 | 盡可能最短的時間 |
| 提醒 | 1小時時通知外科醫生；盡快移除 |
| 最大 | 1.5小時（如果手術困難）；2小時應作為最大值 |
| 風險 | 止血帶釋放後可發生PE |

### 止血帶疼痛

| 內容 | 說明 |
|-----|------|
| 機制 | 氣囊充盈30-60分鐘後，患者可能出現心率和舒張壓增加 |
| 處理 | β-阻滯劑（特別是拉貝他洛）可能有用；止血帶充盈前給予小劑量氯胺酮IV（0.25 mg/kg）可預防這些血壓升高 |

## 23.6 全髖關節置換術

| 項目 | 內容 |
|-----|------|
| 程序 | 股骨頭和髖臼的假體置換 |
| 時間 | 60-90分鐘 |
| 疼痛 | +++ |
| 體位 | 側臥或仰臥 |
| 失血 | 300-500 mL |
| 實用技術 | 脊髓±鎮靜或GA ± LIA或神經阻滯 |

### 統計數據

- 每年約100,000例原發髖關節手術（2016-2019）
- 平均年齡近69歲
- 26%為骨水泥型，35%為非骨水泥型

### 術前考量

| 考量 | 內容 |
|-----|------|
| 術前評估 | 仔細評估患者 |
| 貧血 | 無患者應擇期手術（瞄準Hb >130 g/L） |
| 抗血栓 | 入院時開始預防措施 |

### 圍手術期考量

| 考量 | 內容 |
|-----|------|
| 脊髓 | 單次脊髓：2-3 mL 0.5%高比重布比卡因；可添加二醋嗎啡（0.25-0.5 mg） |
| 鎮靜 | 咪達唑侖在老年人避免（可能導致急性譫妄） |
| 血壓 | 維持術前讀數的適當水平；低血壓不合適 |
| 抗生素 | 術中抗生素預防 |
| 加溫 | 主動加溫患者減少術中失血 |
| 氨甲環酸 | 考慮IV |

### 術後考量

| 考量 | 內容 |
|-----|------|
| 體位 | 仰臥，腿部外展用枕頭防止假體脫位 |
| 抗血栓 | 預防重要（即使有措施，>1%患者發生DVT） |
| 氧疗 | 建議最多24小時 |
| 活動 | 盡快活動（理想情況下，手術當天） |

## 23.7 全膝關節置換術

| 項目 | 內容 |
|-----|------|
| 程序 | 膝關節假體置換 |
| 時間 | 45-90分鐘 |
| 疼痛 | +++/++++ |
| 體位 | 仰臥 |
| 失血 | 帶止血帶最少；無止血帶250-500 mL |
| 實用技術 | 脊髓鎮痛加LIA；GA加LIA；超聲引導內收肌管阻滯 |

### 特點

| 特點 | 內容 |
|-----|------|
| 相似人群 | 與髖關節手術相似 |
| 手術時間 | 通常較短，失血較少 |
| 止血帶 | 常用（注意止血帶疼痛） |
| 術後疼痛 | 可能非常嚴重，必須預期 |

### 內收肌管阻滯

- 運動成分 spared的神經阻滯越來越受歡迎
- 不損害活動能力

### 止血帶釋放反應

| 反應 | 內容 |
|-----|------|
| 常見 | 血壓和SaO₂下降，ETCO₂升高 |
| 預防 | 止血帶釋放前保證充足負荷 |

### 雙側全膝關節置換

| 考量 | 內容 |
|-----|------|
| 適用患者 | 僅年輕fit積極患者 |
| 風險 | 老年和有心血管疾病的患者高風險 |
| 優點 | 避免兩次入院/手術 |
| 缺點 | 雙側是主要心血管應激，與不可預測的失血和液體需求相關 |

## 23.8 關節鏡下下肢程序

| 項目 | 內容 |
|-----|------|
| 程序 | 關節鏡、麻醉下檢查（EUA）和沖洗±切除撕裂軟骨、半月板手術/游離體取出、粘連切除、韌帶重建 |
| 時間 | 20-60分鐘 |
| 疼痛 | ++ |
| 體位 | 仰臥，腿部在手術台側面 |
| 失血 | 無 |
| 實用技術 | GA/LMA或門診脊髓麻醉 |

### 技術

| 技術 | 內容 |
|-----|------|
| 術前用藥 | 對乙醯氨基酚和NSAID |
| GA/LMA | 標准門診麻醉，IV阿片類藥物如芬太尼1 mcg/kg |
| 門診脊髓 | 可用丙胺卡因（快速起效、可預測消退、低不良事件發生率） |
| 止血帶 | 常用 |
| 關節內LA | 外科醫生可在手術結束時注入10-20 mL 0.5%布比卡因±嗎啡（10 mg） |

## 23.9 前交叉韌帶修復

| 項目 | 內容 |
|-----|------|
| 程序 | 使用髕腱±腘繩肌腱進行的關節鏡下前交叉韌帶重建 |
| 時間 | 1.5-2小時 |
| 疼痛 | +++/++++ |
| 體位 | 仰臥 |
| 失血 | 無 |
| 實用技術 | LMA + GA ±內收肌管阻滯；脊髓是替代方案 |

### 技術

| 考量 | 內容 |
|-----|------|
| 類型 | 僅用髕腱修復；或髕腱和腘繩肌腱 |
| 鎮痛需求 | 活動前需要約12小時鎮痛 |
| 神經阻滯 | 阻滯會阻礙術後活動 |
| 主要鎮痛 | 口服阿片類藥物配合對乙醯氨基酚和NSAIDs |

## 23.10 踝關節手術

### 程序類型

| 程序 | 時間 | 疼痛 | 技術 |
|-----|------|------|------|
| 肌腱轉移 | 較短 | ++ | LIA |

| 急診ORIF | 如果血管供應受損可能需要RSI |

## 23.11 足部手術

### 程序類型

| 程序 | 說明 |
|-----|------|
| 前足和腳趾 | 第一蹠骨截骨術、Keller's、嵌甲切除、腳趾終端化 |
| 中足 | 肌腱轉移和一些截骨術 |

### 技術

| 技術 | 內容 |
|-----|------|
| 網/趾間隙或踝阻滯 | 腳趾手術 |
| 踝阻滯 | 前足手術 |
| 坐骨（或膕）神經阻滯 | 中足手術 |
| 脊髓 | 所有病例的替代方案 |

### 特殊考量

| 考量 | 內容 |
|-----|------|
| 腎上腺素 | 禁用於「環」或「網」阻滯 |
| 止血帶疼痛 | 45分鐘以上可能為問題 |

## 23.12 脊柱手術

### 程序分類

| 分類 | 說明 |
|-----|------|
| 脊髓和神經減壓 | - |
| 脊柱畸形穩定和矫正 | - |
| 脊柱腫瘤切除 | - |
| 創傷 | - |

### 一般原則

| 原則 | 內容 |
|-----|------|
| 體位 | 大多數為俯臥位（前和側入路也有） |
| 氣道 | 氣道訪問在手術期間受限（必須安全） |
| 腹部壓力 | 防止過度腹部或胸部壓力（可能損害通氣和循環） |
| 失血 | 可能相當可觀；考慮氨甲環酸和細胞回收 |
| 體溫 | 長時間程序需要積極防止熱量損失 |
| 神經監測 | 可能需要脊髓功能評估 |

### 麻醉技術

| 項目 | 內容 |
|-----|------|
| 誘導 | IV誘導配合肌肉鬆弛和阿片類藥物補充 |
| 維持 | 吸入麻醉和TIVA都常用；術中神經監測需要可能排除吸入麻醉氣體 |
| 監測 | 侵入性动脉壓監測用於：計劃術後ICU入院、預測術中不穩定、預測大量失血、需要術中血液調查 |

### 脊髓完整性監測

| 技術 | 內容 |
|-----|------|
| 軀體感覺誘發電位 | - |
| 運動誘發電位 | - |
| EMG | - |
| EEG | - |

### 俯臥位

| 考量 | 內容 |
|-----|------|
| 翻轉 | 需要訓練有素的團隊（至少六名成員）避免在軸向平面施加扭力 |
| 隔室預防 | 每小時移動患者肢體和頭部避免外周血液停滯和壓力壞死發展 |
| 眼部並發症 | 包括角膜磨損到視力喪失 |
| 腹部壓力 | 可能傳遞到無瓣膜硬膜外靜脈並導致出血增加 |

## 23.13 肩關節手術

### 一般考量

| 考量 | 內容 |
|-----|------|
| 疼痛程度 | 常常極度疼痛（比其他許多門診程序更疼痛） |
| 疼痛持續 | 可持續數天（48小時內無疑是最嚴重的） |
| 鎮痛方法 | 多模式方法至關重要 |
| 患者人群 | 異質（從創傷的年輕運動員到有許多合併症的老年患者） |

### 關節鏡 vs 開放程序

| 程序類型 | 疼痛程度 | 說明 |
|---------|---------|------|
| 關節鏡 | 通常較少疼痛 | 外科醫生在手術結束時注射LA入關節腔可獲得有效術後鎮痛 |
| 開放程序 | 通常更疼痛 | 取決於程序（更大、更肌肉的患者） |

### 超聲引導區域麻醉技術

| 技術 | 說明 |
|-----|------|
| 常用阻滯 | 肌間溝和鎖骨上阻滯 |
| 優勢 | 減少術後疼痛（減少伴隨的圍手術期阿片類藥物使用）、減少PONV、改善患者滿意度 |
| 持續導管 | 可能提供優於單次注射的鎮痛 |

### 沙灘椅位併發症

| 併發症 | 內容 |
|--------|------|
| 心血管不穩定或腦缺血 | 可能受益於血管加壓素輸注或腦氧化測定監測 |
| 空氣栓塞 | - |

### 術後考量

| 考量 | 內容 |
|-----|------|
| 門診 | 大多數程序可作為門診進行 |
| 患肢保護 | 當神經阻滯仍在起作用時必須保護患肢 |
| 多模式鎮痛 | 包括對乙醯氨基酚、NSAIDs和阿片類藥物 |

## 23.14 全肩關節置換術

| 項目 | 內容 |
|-----|------|
| 程序 | 假體肩關節置換 |

| 體位 | 仰臥，頭高腳低或 deck-chair |
| 失血 | 250-500 mL |
| 實用技術 | GA with ETT（south-facing RAE）+ IPPV；肌間溝阻滯±肌間溝導管 |

| 適應症 | 說明 |
|--------|------|
| 急性創傷 | - |

| 旋轉袖撕裂 | - |
| 創傷後遺症 | - |
| 其他炎症性關節病 | - |

### 程序特定考量

| 考量 | 內容 |
|-----|------|
| 疼痛 | 術後24小時內最嚴重；PCA/間歇性嗎啡通常滿意 |
| 氨甲環酸 | 如果有顯著失血考慮 |

## 23.16 手部手術麻醉

| 時間 | 可變 |
| 疼痛 | +/+++ |

| 失血 | 最少 |
| 實用技術 | 區域鎮痛± GA；多模式鎮痛；止血帶 |

### 技術選擇

| 技術 | 說明 |
|-----|------|

### 特殊考量

| 考量 | 內容 |
|-----|------|
| 止血帶疼痛 | 可通過在止血帶水平上方上臂內側阻斷肋間臂神經來減少 |
| 腎上腺素 | 應避免用於近指節 |

## 23.17 股骨頸骨折麻醉

| 項目 | 內容 |
|-----|------|
| 程序 | 空心螺絲.Dynamic髖螺絲；骨水泥/非骨水泥半髖置換；全髖關節置換 |
| 時間 | 10-90分鐘 |
| 疼痛 | +/+++ |
| 體位 | 仰臥（在髖關節台上）；偶爾側臥 |
| 失血 | 250-750 mL |
| 實用技術 | SV LMA和區域阻滯；脊髓±鎮靜；ETT + IPPV |

### 統計數據

| 項目 | 數值 |
|-----|------|
| 每年骨折數 | 約767,000（2020年報告；80%女性） |
| 平均年齡 | 84歲；80%發生在>75歲 |
| 終身風險 | 女性18%；男性6% |
| 3個月死亡率 | 7-12% |
| 1年死亡率 | 21% |

### 術前考量

| 考量 | 內容 |
|-----|------|
| 生理儲備 | 減少；合併症常見 |
| 術前評估 | 必須進行；手術應安排在最早的日間會議 |
| 術前評分 | 諾丁漢髖骨折評分、衰弱評分、4AT譫妄評分 |
| 脫水 | 常見（口服攝入通常大大減少） |
| 鎮痛 | 應開始（患者常在相當疼痛中） |
| 手術時間 | 36小時內應進行 |
| 抗凝/抗血小板 | 許多患者可能正在服用；遵循指南 |

### 程序類型

| 程序 | 說明 |
|-----|------|
| 空心髖螺絲 | 快速；微創；失血少 |
| 半髖置換 | 與原發髖關節置換的股骨部分相似 |
| DHS/Richards螺絲和板 | 中間程序 |
| 全髖關節置換 | - |

### 術後併發症

| 併發症 | 內容 |
|--------|------|
| 認知功能障礙 | 發生率增加 |
| 心臟併發症 | - |
| 肺部併發症 | - |
| 感染 | - |
