## 21.1 一般原則

### 顱內壓（ICP）

正常的ICP為5–15 mmHg。顱內壓的變化反映了頭骨限制範圍內內容物體積的改變。

**顱內內容物組成：**

| 組成 | 體積 |
|-----|------|
| 腦實質 | 1200–1600 mL |
| 血液 | 100–150 mL |
| 腦脊髓液（CSF） | 100–150 mL |
| 細胞外液（ECF） | <75 mL |

### 代償機制

當顱內出現佔位性病變時，代償機制會：
1. 先將CSF排入脊髓蜘蛛膜下腔
2. 增加CSF吸收
3. 減少顱內血容積

當這些機制耗盡後，體積的微小增加就會導致ICP急劇上升。

### 顱內壓升高的原因

| 類別 | 原因 |
|-----|------|
| 腦體積增加 | 全身性腫脹、腦水腫 |
| 質量效應（Mass effect） | 佔位性病變、血腫、腫瘤、膿疡 |
| CSF體積增加 | 產生增加或吸收減少（如水腦症、腦膜發炎） |
| 血容積增加 | 腦血流（CBF）增加：缺氧、高碳酸血症、吸入性麻醉藥；腦靜脈容積增加：胸腔壓力增加、靜脈鬱滯/血栓、頸部靜脈阻塞、頭低腳高姿勢、咳嗽 |
| 其他 | 良性或特發性顱內高壓 |

### 腦灌注壓（CPP）

腦灌注壓是驅動血液流向大腦的有效壓力。

**公式：**
```
CPP = MAP - (ICP + VP)
```

其中VP（頸靜脈球的靜脈壓）通常為零或更低，因此CPP主要與ICP和MAP有關。

### 腦血流（CBF）

**自我調節機制：**
- 可在MAP 60–150 mmHg之間維持CBF（約50 mL/100g/min）
- 在此範圍外，CBF會隨灌注壓被動變化

**影響CBF的因素：**

| 因素 | 效應 |
|-----|------|
| 大腦代謝 | 癲癇發作、疼痛/焦慮時增加；昏迷、低溫、使用麻醉藥時減少 |
| CO₂分壓 | 低碳酸血症導致腦血管收縮及CBF減少 |
| O₂分壓 | 只有在PaO₂ < 7 kPa（53 mmHg）時才會引起腦血管擴張 |
| 溫度 | 體溫每下降1°C，腦代謝率下降7%，從而使CBF減少 |
| 麻醉藥物 | 各有不同的影響 |

### 顱內壓測量

| 方法 | 說明 |
|-----|------|
| 腦室內 | 透過顱骨鑽孔將導管置入側腦室；是**黃金標準**，並允許引流CSF |
| 腦實質內 | 放置微型應變計監測器；是目前最常見的技術 |

## 21.2 開顱手術（Craniotomy）

| 項目 | 內容 |
|-----|------|
| 手術內容 | 腫瘤切除或減積、腦部切片、腦膿疡引流 |
| 時間 | 1–12小時 |
| 疼痛程度 | + / +++ |
| 姿勢 | 仰臥，頭高腳低（head-up）或側臥 |
| 失血量 | 0–2000 mL，需備血 |
| 實用技術 | 氣管內管（ETT）、IPPV、动脉導管（A-line） |

### 術前與術中處置

| 項目 | 內容 |
|-----|------|
| 神經學評估 | 評估神經學狀態及ICP升高的症狀 |
| 腦水腫 | 若有腦水腫應限制靜脈輸液，**避免使用含葡萄糖的溶液** |
| 誘導 | Propofol靜脈推注或TCI，配合Remifentanil；給藥應緩慢以避免血壓與CPP下降 |
| 麻醉維持 | 避免使用N₂O；使用揮發性麻醉劑（<1 MAC）或Propofol TCI維持麻醉 |
| 通氣目標 | 維持正常血壓，正常碳酸血症（PaCO₂ 33–38 mmHg） |
| 體溫 | 保持正常體溫 |

## 21.3 腦室腹腔分流術（Ventriculoperitoneal Shunt）

| 項目 | 內容 |
|-----|------|
| 手術內容 | 水腦症的CSF引流 |
| 時間 | 45–120分鐘 |
| 疼痛程度 | ++ |
| 姿勢 | 仰臥，頭高腳低 |
| 失血量 | 極少 |
| 實用技術 | ETT、IPPV |

### 處置重點

| 項目 | 內容 |
|-----|------|
| 假設 | 假定所有患者的ICP皆升高 |
| 刺激 | 推進套管以建立皮下隧道的過程刺激性極大 |
| 管理 | 通常需要額外的止痛或肌肉鬆弛劑 |

## 21.4 創傷性顱內血腫清除術（Evacuation of Traumatic Intracranial Haematoma）

| 項目 | 內容 |
|-----|------|
| 手術內容 | 硬腦膜外、硬腦膜下或腦實質內血腫清除 |
| 時間 | 1–6小時 |
| 疼痛程度 | + / +++ |
| 姿勢 | 仰臥，頭高腳低 |
| 失血量 | 200 mL以上，需備血 |
| 實用技術 | ETT、IPPV、A-line、CVC |

### 處置重點

| 項目 | 內容 |
|-----|------|
| 監測 | 需要標準監測及侵入性动脉血壓監測 |
| CPP目標 | 維持CPP在60–70 mmHg |
| 禁用 | 避免使用N₂O |
| 藥物 | 視需要給予Mannitol（0.25–1 g/kg）或Furosemide |

## 21.5 腦下垂體手術（Pituitary Surgery）

| 項目 | 內容 |
|-----|------|
| 手術內容 | 經蝶竇腦下垂體切除術 |
| 時間 | 90–300分鐘 |
| 疼痛程度 | ++ |
| 姿勢 | 仰臥，頭高腳低 |
| 失血量 | 通常無，若傷及靜脈竇則失血量大 |
| 實用技術 | ETT、IPPV、A-line |

### 特殊患者考量

#### 肢端肥大症（Acromegaly）

| 考量 | 內容 |
|-----|------|
| 氣道 | 巨舌，下顎突出可能導致呼吸道困難 |
| 合併症 | 常伴隨高血壓、缺血性心臟病與OSA |

#### 庫欣氏症（Cushing's）

| 考量 | 內容 |
|-----|------|
| 表現 | 高血壓、中心型肥胖、低血鉀、葡萄糖耐受不良 |
| 管理 | 圍手術期需要類固醇覆蓋（Steroid cover） |

### 術中處置

| 項目 | 內容 |
|-----|------|
| 喉部紗布塞 | 插管後應置入喉部紗布塞以減少血液流入喉部 |

## 21.6 後顱窩手術（Posterior Fossa Surgery）

| 項目 | 內容 |
|-----|------|
| 手術內容 | 腫瘤切除、血管手術、枕骨大孔減壓術 |
| 時間 | 3–14小時 |
| 疼痛程度 | + / +++ |
| 姿勢 | 坐姿、趴姿或側臥 |
| 實用技術 | ETT、IPPV、A-line、考慮CVC / VAE監測 |

### 患者姿勢

#### 坐姿（Sitting）

| 項目 | 內容 |
|-----|------|
| 優點 | 提供中線病灶最佳視野並降低ICP |
| 併發症 | 血流動力學不穩、靜脈空氣栓塞（VAE）及矛盾性空氣栓塞 |

#### 趴姿（Prone）

| 項目 | 內容 |
|-----|------|
| 優點 | 提供良好的手術視野且無坐姿的風險 |
| 注意 | 需避免腹部受壓 |

#### 側臥（Lateral）

| 項目 | 內容 |
|-----|------|
| 適用 | 適合聽神經瘤等外側病灶 |
| 注意 | 確保頸靜脈不被阻塞 |

## 21.7 清醒開顱手術（Awake Craniotomy）

用於切除語言、運動等重要皮質區的腫瘤，藉由術中喚醒來評估神經功能。

### 技術

| 技術 | 說明 |
|-----|------|
| Asleep–Awake–Asleep | 在開顱初期進行全身麻醉，隨後喚醒患者進行功能測試，關閉顱骨時再次麻醉 |
| Conscious Sedation（清醒鎮靜） | 初期不使用進階呼吸道，僅給予鎮靜，測試時喚醒患者 |

### 常用藥物

| 藥物 | 說明 |
|-----|------|
| Propofol | 誘導和維持 |
| Remifentanil | 鎮痛 |
| Dexmedetomidine | 鎮靜 |

## 21.8 顱內動脈瘤與蛛網膜下腔出血（Intracranial Aneurysms & SAH）

### 臨床表現

動脈瘤破裂會表現為嚴重的突發性頭痛與意識喪失。預後與WFNS臨床分級密切相關。

### 併發症

| 併發症 | 說明 |
|--------|------|
| 再出血（Rebleeding） | 前72小時風險為5–10%，前24小時風險最高 |
| 延遲性腦缺血/血管痙攣（Vasospasm） | 高峰期在出血後第4–10天；治療包含Nimodipine（鈣離子通道阻斷劑）及維持等血容與誘導性高血壓 |
| 水腦症（Hydrocephalus） | 血液阻塞CSF引流所致 |

### 圍手術期麻醉管理（夾閉術/栓塞術）

| 目標 | 內容 |
|-----|------|
| 核心目標 | 維持腦灌注，同時避免血壓突升導致動脈瘤破裂 |

**術中破裂處理：**
- 可推注Propofol誘導低血壓（MAP 50–60 mmHg）以協助外科醫師夾閉
- 或應外科要求使用Adenosine造成暫時性血流停搏

### 血管內血栓移除術（Thrombectomy）

| 項目 | 內容 |
|-----|------|
| 性質 | 為分秒必爭的急症 |
| 重點 | 維持血流動力學穩定，避免過度低血壓損害缺血半暗帶 |
| 目標 | 維持SpO₂ 94–98%與正常碳酸血症 |

## 21.9 靜脈空氣栓塞（Venous Air Embolism, VAE）

### 風險

只要手術部位高於右心房（特別是坐姿開顱手術），就有VAE風險。

### 偵測

| 方法 | 敏感性 |
|-----|--------|
| EtCO₂突然下降 | 最實用的指標 |
| 經食道心臟超音波（TOE） | 最敏感且特異的監測儀器 |

### 治療

| 步驟 | 內容 |
|-----|------|
| 通知外科 | 立即通知外科醫師向手術區注水 |
| 調整麻醉 | 停用N₂O並將FiO₂調至100% |
| 血壓支持 | 使用輸液與升壓藥支持血壓 |
| 抽吸 | 嘗試從CVC抽吸空氣 |

## 21.10 神經外科急救復甦（Resuscitation in Neurosurgery）

### 一般原則

- 心搏驟停的診斷與處理應遵循高級生命支持（ALS）規範

### 特殊考量

| 考量 | 內容 |
|-----|------|
| 患者姿勢 | 開始胸外按壓前，必須先鬆開Mayfield頭部固定架，以免損傷頸椎與顱骨 |
| 腎上腺素劑量 | 初始可給予較小劑量的腎上腺素（50–100 mcg IV），以防止復甦成功後出現反彈性高血壓導致顱內出血 |
