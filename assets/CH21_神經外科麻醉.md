# Chapter 21: Neurosurgery 神經外科麻醉

## General Principles 基本原則

### Intracranial Pressure (ICP) 顱內壓
正常的 ICP 為 5–15 mmHg。顱內壓的變化反映了頭骨限制範圍內內容物體積的改變（腦實質 1200–1600 mL，血液 100–150 mL，腦脊髓液 CSF 100–150 mL，細胞外液 ECF <75 mL）。

當顱內出現佔位性病變時，代償機制會先將 CSF 排入脊髓蜘蛛膜下腔、增加 CSF 吸收並減少顱內血容積，藉此緩解對 ICP 的影響。當這些機制耗盡後，體積的微小增加就會導致 ICP 急劇上升。

**顱內壓升高的原因：**
* **腦體積 ↑：** 全身性腫脹、腦水腫。
* **質量效應 (Mass effect)：** 佔位性病變、血腫、腫瘤、膿瘍。
* **CSF 體積 ↑：** 產生增加或吸收減少（如：水腦症、腦膜發炎）。
* **血容積 ↑：** 
  * 腦血流 (CBF) ↑：缺氧、高碳酸血症、吸入性麻醉藥。
  * 腦靜脈容積 ↑：胸腔壓力增加、靜脈鬱滯/血栓、頸部靜脈阻塞、頭低腳高姿勢、咳嗽。
* **其他：** 良性或特發性顱內高壓。

### Cerebral Perfusion Pressure (CPP) 腦灌注壓
腦灌注壓是驅動血液流向大腦的有效壓力。由於頸靜脈球的靜脈壓 (VP) 通常為零或更低，因此 CPP 主要與 ICP 和平均動脈壓 (MAP) 有關。

$CPP = MAP - (ICP + VP)$

大腦會透過自我調節 (Autoregulation) 讓 CBF 保持恆定。

### Cerebral Blood Flow (CBF) 腦血流
自我調節機制可在 MAP 60–150 mmHg 之間維持 CBF (約 50 mL/100g/min)。在此範圍外，CBF 會隨灌注壓被動變化。

**影響 CBF 的因素：**
* **大腦代謝：** 癲癇發作與疼痛/焦慮時 ↑；昏迷、低溫與使用麻醉藥時 ↓。
* **CO₂ 分壓：** 低碳酸血症會導致腦血管收縮及 CBF ↓。
* **O₂ 分壓：** 只有在 PaO₂ < 7 kPa (53 mmHg) 時才會引起腦血管擴張。
* **溫度：** 體溫每下降 1°C，腦代謝率下降 7%，從而使 CBF ↓。
* **麻醉藥物。**

### Measuring Intracranial Pressure 顱內壓測量
* **腦室內 (Ventricular)：** 透過顱骨鑽孔將導管置入側腦室，是**黃金標準**，並允許引流 CSF。
* **腦實質內 (Intraparenchymal)：** 放置微型應變計監測器，是目前最常見的技術。

## Craniotomy 開顱手術

| | |
|---|---|
| **手術內容** | 腫瘤切除或減積、腦部切片、腦膿瘍引流 |
| **時間** | 1–12 小時 |
| **疼痛程度** | + / +++ |
| **姿勢** | 仰臥、頭高腳低 (head-up) 或側臥 |
| **失血量** | 0–2000 mL，需備血 |
| **實用技術** | 氣管內管 (ETT)、IPPV、動脈導管 (A-line) |

### 術前與術中處置
* 評估神經學狀態及 ICP 升高的症狀。若有腦水腫應限制靜脈輸液，並**避免使用含葡萄糖的溶液**。
* 誘導時可使用 Propofol 靜脈推注或 TCI，配合 Remifentanil。給藥應緩慢以避免血壓與 CPP 下降。
* 避免使用 N₂O。使用揮發性麻醉劑 (<1 MAC) 或 Propofol TCI 維持麻醉。
* 大多數手術中目標為維持正常血壓，並維持正常碳酸血症 (PaCO₂ 33–38 mmHg)。保持正常體溫。

## Ventriculoperitoneal Shunt 腦室腹腔分流術

| | |
|---|---|
| **手術內容** | 水腦症的 CSF 引流 |
| **時間** | 45–120 分鐘 |
| **疼痛程度** | ++ |
| **姿勢** | 仰臥、頭高腳低 |
| **失血量** | 極少 |
| **實用技術** | ETT、IPPV |

* **處置重點：** 假定所有患者的 ICP 皆升高。推進套管以建立皮下隧道的過程刺激性極大，通常需要額外的止痛或肌肉鬆弛劑。

## Evacuation of Traumatic Intracranial Haematoma 創傷性顱內血腫清除術

| | |
|---|---|
| **手術內容** | 硬腦膜外、硬腦膜下或腦實質內血腫清除 |
| **時間** | 1–6 小時 |
| **疼痛程度** | + / +++ |
| **姿勢** | 仰臥、頭高腳低 |
| **失血量** | 200 mL 以上，需備血 |
| **實用技術** | ETT、IPPV、A-line、CVC |

* **處置重點：** 需要標準監測及侵入性動脈血壓監測。維持 CPP 在 60–70 mmHg，避免使用 N₂O。視需要給予 Mannitol (0.25–1 g/kg) 或 Furosemide。

## Pituitary Surgery 腦下垂體手術

| | |
|---|---|
| **手術內容** | 經蝶竇腦下垂體切除術 |
| **時間** | 90–300 分鐘 |
| **疼痛程度** | ++ |
| **姿勢** | 仰臥、頭高腳低 |
| **失血量** | 通常無，若傷及靜脈竇則失血量大 |
| **實用技術** | ETT、IPPV、A-line |

* **肢端肥大症 (Acromegaly) 注意事項：** 巨舌、下顎突出可能導致呼吸道困難；常伴隨高血壓、缺血性心臟病與 OSA。
* **庫欣氏症 (Cushing’s) 注意事項：** 高血壓、中心型肥胖、低血鉀、葡萄糖耐受不良，圍手術期需要類固醇覆蓋 (Steroid cover)。
* **術中：** 插管後應置入喉部紗布塞 (Throat pack) 以減少血液流入喉部。

## Posterior Fossa Surgery 後顱窩手術

| | |
|---|---|
| **手術內容** | 腫瘤切除、血管手術、枕骨大孔減壓術 |
| **時間** | 3–14 小時 |
| **疼痛程度** | + / +++ |
| **姿勢** | 坐姿、趴姿或側臥 |
| **實用技術** | ETT、IPPV、A-line、考慮 CVC / VAE監測 |

### 患者姿勢 (Patient Positioning)
* **坐姿 (Sitting)：** 提供中線病灶最佳視野並降低 ICP。但併發症包括血流動力學不穩、靜脈空氣栓塞 (VAE) 及矛盾性空氣栓塞。
* **趴姿 (Prone)：** 提供良好的手術視野且無坐姿的風險，但需避免腹部受壓。
* **側臥 (Lateral)：** 適合聽神經瘤等外側病灶，需確保頸靜脈不被阻塞。

## Awake Craniotomy 清醒開顱手術

用於切除語言、運動等重要皮質區的腫瘤，藉由術中喚醒來評估神經功能。

* **Asleep–Awake–Asleep 技術：** 在開顱初期進行全身麻醉，隨後喚醒患者進行功能測試，關閉顱骨時再次麻醉。
* **Conscious Sedation (清醒鎮靜) 技術：** 初期不使用進階呼吸道，僅給予鎮靜，測試時喚醒患者。常用藥物包括 Propofol、Remifentanil 與 Dexmedetomidine。

## Intracranial Aneurysms & SAH 顱內動脈瘤與蛛網膜下腔出血

動脈瘤破裂會表現為嚴重的突發性頭痛與意識喪失。預後與 WFNS 臨床分級密切相關。

### 動脈瘤破裂的併發症
* **再出血 (Rebleeding)：** 前 72 小時風險為 5–10%，前 24 小時風險最高。
* **延遲性腦缺血 / 血管痙攣 (Vasospasm)：** 高峰期在出血後第 4–10 天。治療包含 Nimodipine（鈣離子通道阻斷劑）及維持等血容與誘導性高血壓。
* **水腦症 (Hydrocephalus)：** 血液阻塞 CSF 引流所致。

### 圍手術期麻醉管理 (夾閉術 / 栓塞術)
* **核心目標：** 維持腦灌注，同時避免血壓突升導致動脈瘤破裂。
* 若術中動脈瘤破裂，可推注 Propofol 誘導低血壓 (MAP 50–60 mmHg) 以協助外科醫師夾閉，或應外科要求使用 Adenosine 造成暫時性血流停搏。
* **血管內血栓移除術 (Thrombectomy)：** 為分秒必爭的急症。重點在維持血流動力學穩定，避免過度低血壓損害缺血半暗帶。維持 SpO₂ 94–98% 與正常碳酸血症。

## Venous Air Embolism (VAE) 靜脈空氣栓塞

只要手術部位高於右心房（特別是坐姿開顱手術），就有 VAE 風險。

* **偵測：** EtCO₂ 突然下降是最實用的指標。經食道心臟超音波 (TOE) 是最敏感且特異的監測儀器。
* **治療：** 立即通知外科醫師向手術區注水；停用 N₂O 並將 FiO₂ 調至 100%；使用輸液與升壓藥支持血壓；嘗試從 CVC 抽吸空氣。

## Resuscitation in Neurosurgery 神經外科急救復甦

* 心搏驟停的診斷與處理應遵循高級生命支持 (ALS) 規範。
* **患者姿勢：** 開始胸外按壓前，必須先鬆開 Mayfield 頭部固定架，以免損傷頸椎與顱骨。
* **與一般 ALS 的差異：** 初始可給予較小劑量的腎上腺素 (50–100 mcg IV)，以防止復甦成功後出現反彈性高血壓導致顱內出血。
