## 18.1 麻醉對腫瘤學的影響

### 圍手術期的重要性

圍手術期是癌症外科治療的關鍵階段，此期間發生的事件可能影響患者的腫瘤學結局。

外科組織創傷和操作已被證明會從原發腫瘤產生局部和血源性腫瘤細胞傳播。這和切除不完全的情況導致惡性細胞在整個圍手術期持續存在，稱為「微小殘留疾病」。

### 手術應激反應的促腫瘤機制

| 機制 | 說明 |
|-----|------|
| 生長和血管生成因子釋放 | 前列腺素E2、血小板衍生生長因子（PDGF）、胰島素生長因子（IGF）、缺氧誘導因子1α、血管內皮生長因子（VEGF） |
| 促进肿瘤细胞侵袭性的因子释放 | 基质金属蛋白酶等 |
| 細胞內絲裂原信號通路誘導 | 促進細胞增殖和減少凋亡 |
| 免疫抑制 | 特別是細胞免疫抑制，NK、CD4和CD8細胞的數量和活性減少 |

因此，圍手術期可能構成有利於癌症早期復發/轉移的「完美風暴」。

### 麻醉技術影響腫瘤學結局的理論依據

有大量證據證明各種麻醉技術/圍手術期干預措施可能改變手術應激反應的不同要素。

通過改變應激反應的性質和幅度，不同的麻醉技術可能產生不同的術後腫瘤學結局。

### 當前爭議和相應證據

#### 吸入麻醉 vs TIVA

| 麻醉類型 | 效應 |
|---------|------|
| 吸入麻醉藥物 | 已被證明可誘導腫瘤生成生長因子表達，並主要通過抑制NK細胞和細胞毒性T淋巴細胞產生免疫抑制 |
| Propofol | 已被證明可減少缺氧誘導因子1α的表達，並具有小得多的免疫抑制效應 |

**臨床建議：** 基於回顧性隊列研究的支持，propofol-based TIVA可能比吸入藥物更適合腫瘤手術。然而，目前尚無前瞻性研究證明其益處。

#### 阿片類藥物鎮痛

- 阿片類藥物主要通過μ-阿片受體發揮作用
- 該受體在許多癌細胞類型上過度表達
- 高水平μ-阿片受體表達與較差的腫瘤學結局相關
- 臨床證據目前不支持這一點，這仍然是理論風險

#### 區域麻醉

- 理論上可通過最小化手術應激和減少阿片類藥物使用有益
- 目前證據混合，單獨為此目的推薦使用區域麻醉尚不合理

#### NSAIDs

- 通過抑制COX減少許多生長因子的表達，包括前列腺素E2
- 理論上可減少腫瘤復發/轉移的風險
- 缺乏臨床顯著效應的證據

#### 地塞米松

- 類固醇已知具有免疫抑制作用
- 缺乏臨床證據支持在腫瘤手術中不合適使用

### 實踐建議

| 原則 | 內容 |
|-----|------|
| 核心原則 | 採取最小化手術生理損傷和應激並產生快速、高質量康復和恢復到發病前功能的策略 |
| 推薦方法 | ERAS或其他快速康復方案 |
| 目標 | 允許患者在最佳時間框架內進入下一階段癌症治療 |

## 18.2 腫瘤學考量

### 一般考量

#### 心臟損傷

| 原因 | 說明 |
|-----|------|
| 藥物引起 | 蔥環類藥物、5-氟尿嘧啶、曲妥珠單抗 |
| 化療對受損心臟的應激 | 蔥環類誘導的心臟衰竭可能是不可逆的，死亡率>30% |

#### 博來霉素暴露

- 作為化療的一部分存在快速進行性肺毒性的潛在風險
- 10%暴露於博來霉素的患者發生肺毒性
- 表現為急性，然後是慢性纖維化肺泡炎
- 建議避免高吸入O₂濃度和仔細的液體管理
- 主要用於生殖細胞腫瘤和霍奇金淋巴瘤

#### 細胞因子釋放綜合徵

- 對免疫療法（包括CAR-T細胞療法）的急性全身炎症反應，以發熱為先兆
- 相關特徵可能從輕度流感樣症狀到嚴重危及生命的循環衰竭

#### 肝靜脈閉塞性疾病

- 肝臟靜脈通道的進行性閉塞

#### 腫瘤溶解綜合徵

- 可發生在初始化療後（通常為淋巴瘤和高計數白血病）
- 大量細胞死亡導致急性腎損傷、高鉀血症、高尿酸血症、高磷酸鹽血症和低鈣血症

#### 縱隔腫塊

- 特別是白血病或淋巴瘤患者
- 可在麻醉下引起完全氣道塌陷，即使在無症狀患者中
- 警告體徵包括喘鳴、哮鳴、端坐呼吸和上腔靜脈阻塞

#### 上腔靜脈（SVC）阻塞

- 可能由腫瘤或淋巴結（通常為支氣管源性癌）壓迫或直接血管侵犯引起
- 卵巢癌、轉移性疾病和間皮瘤常見胸腔積液和腹水

### 副腫瘤綜合徵

約10%癌症患者發生（特別是肺癌、淋巴瘤、乳腺癌、前列腺癌、卵巢癌和胰腺癌）。

| 綜合徵 | 相關腫瘤 |
|--------|---------|
| Lambert-Eaton肌無力綜合徵 | 小細胞肺癌、乳腺癌、胸腺和胃腸道腫瘤 |
|庫欣綜合徵 | 肺癌、胰腺癌、胸腺癌、卵巢癌 |
| 高鈣血症 | 骨轉移或甲狀旁腺素樣化合物 |
| 低鈉血症和SIADH樣綜合徵 | 小細胞肺癌、淋巴瘤、白血病、胰腺/類癌腫瘤 |
| 惡病質 | 由嘔吐、食慾不振或其他胃腸道紊亂引起；低白蛋白血症（<35 g/L）是預後不佳的風險因素 |

### 放射治療考量

- 可能引起發熱和噁心/嘔吐
- 患者可能脫水
- 先前放射治療導致持續局部纖維化，可能妨礙喉鏡和氣道管理

### 化療考量

- 常引起免疫抑制和骨髓抑制

### 靜脈血栓栓塞

- 至少15%癌症患者受影響

### 不嘗試復甦（DNAR）命令

- 可能在癌症患者中存在
- 如果與安全麻醉原則衝突，在圍手術期修改或暫停命令是合理的

### 腫瘤手術的麻醉

#### 靜脈通路港植入（如Portacath）

| 項目 | 內容 |
|-----|------|
| 首選通路 | 左鎖骨下靜脈 |
| 設備 | 完全可植入靜脈通路裝置 |
| 麻醉方式 | 全身麻醉或局部麻醉±鎮靜，取決於當地實踐和患者選擇 |
| 術後 | 需要胸部X光確認放置並排除併發症，包括氣胸和血胸 |

#### 孤立肢體灌注

- 用於某些中心的不可切除黑色素瘤和肉瘤病例
- 涉及獲得肢體動脈和靜脈通路、體外迴路和氣動止血帶
- 實現顯著更高濃度的細胞毒性藥物（如melphalan和TNF-α），，同時限制全身副作用
- 需要仔細監測全身洩漏的證據
- 建議適當的侵入性監測和中心靜脈通路

#### 近距離放射治療

- 通過施用器將放射性源靠近腫瘤放置
- 通常用於不適合手術的患者
- 手術通常持續1.5-3小時，可能更長
- 失血通常最少，但術後疼痛可能是問題
- 麻醉選項：
  - 淺全身麻醉
  - 鎮靜（雖然可能不能提供可靠的固定）
  - 硬膜外或脊髓麻醉（某些手術可能超過脊髓麻醉時間，需要導管插入）

## 18.3 激光手術的一般原則

### 激光基礎

**激光是「受激輻射光放大」的縮寫。**

激光是高能光束，能夠將大量能量傳遞到非常小的區域。

**激光的特性：**

| 特性 | 說明 |
|-----|------|
| 相同波長 | 發射原子都相同，釋放的光子波長也相同 |
| 同相位 | 光子同步（波的峰和谷同步，放大而非抵消） |
| 相同偏振 | 波方向相同 |

### 激光與組織的相互作用

當激光照射組織表面時：

| 效應 | 說明 |
|-----|------|
| 反射 | 閃亮表面反射可能損壞附近工作人員的眼睛 |
| 傳輸 | 激光通過組織到可變深度，部分由波長決定 |
| 散射 | 較短波長引起更大散射 |
| 吸收 | 產生臨床效果，光能轉化為熱。有機組織含有各種能夠吸收光的物質，稱為發色團，包括血紅蛋白、膠原蛋白和黑色素 |

### 常用外科激光類型

| 激光類型 | 波長（nm） | 顏色 |
|---------|------------|------|
| 染料激光 | 360-670 | 藍到紅 |
| 氬離子 | 488-515 | 藍/綠 |
| 氦-氖 | 633 | 紅 |
| 紅寶石 | 694 | 紅 |
| Nd-YAG | 1064 | 近紅外 |
| CO₂ | 10,600 | 遠紅外 |

### 醫療激光示例

#### 脈衝染料激光

- 以針對血管內紅細胞的波長使用
- 能量消散在真皮中，僅引起最小表皮疤痕
- 主要用於治療葡萄酒色皮膚病變
- 兒童通常需要全身麻醉下進行多次治療
- 術後疼痛可能是問題，特別是大面積治療時

#### 二氧化碳激光

- 波長長（10,600 nm，在可見光譜之外）
- 優先被水吸收
- 目標細胞被光束加熱到汽化點
- 僅滲透到非常淺的深度
- 組織損傷可直接觀察
- 用於美容面部手術減少衰老相關皺紋
- ENT實踐中用於汽化聲帶和氣道病變

#### Nd-YAG激光

- 也在可見光譜之外
- 與CO₂激光不同，可通過透明液體傳輸並被深色物質吸收
- 可滲透到1 cm深度
- 有多種應用，包括氣道腫瘤、血管畸形和眼科手術

## 18.4 激光手術安全

### 主要危害

| 危害 | 說明 |
|-----|------|
| 意外組織損傷 | 對患者或工作人員 |
| 火災風險 | - |
| 激光煙霧顆粒吸入 | - |

### 安全措施

| 措施 | 內容 |
|-----|------|
| 專門人員 | 指定激光安全員在激光使用時始終在場 |
| 培訓 | 英國任何在激光使用中的手術室人員都應接受適當培訓 |
| 門鎖 | 手術室門應從內部鎖定 |
| 標誌 | 在外部顯示標誌 |
| 檢查清單 | 打開激光前應完成檢查清單 |
| 儀器 | 應使用啞光（而非閃亮）表面以減少激光束在手術室周圍的反射 |

### 眼睛保護

- 眼睛非常容易受傷
- 視網膜和角膜損傷可能發生
- 所有手術室人員必須佩戴適合所用激光的安全眼鏡
- 這些應有側屏蔽以保護眼睛外側
- 如果麻醉患者在眼睛附近接受激光輻射，應使用保護性啞光金屬眼罩

### 皮膚保護

- 某些激光可能損壞暴露的皮膚
- 麻醉患者必須覆蓋所有暴露的皮膚
- 眼睛應貼合並用濕紗布覆蓋
- 許多類型的手術單和麻醉膠帶是可燃的

### 皮膚準備液

- 某些皮膚準備液是易燃的
- 不應在激光手術期間使用

### 氣道維護

| 考量 | 說明 |
|-----|------|
| 激光安全ETT | 如果不使用氣道內激光，最簡單的方法是使用激光安全ETT |
| HFNO或Jet通氣 | 也可使用 |
| 低FiO₂ | 如30%，減少火災風險但也減少呼吸暫停時間 |
| 禁用材料 | NO₂和O₂都易燃 |

### ETT使用注意

- 如果需要使用ETT，未經修改的常規塑料管不能使用
- 因為它們支持燃燒並可能導致氣道火災
- 有各種激光安全ETT可用
- 這些管的氣囊容易受損，可用濕棉墊保護
- 氣囊應充滿0.9%氯化鈉，可與亞甲藍混合，使氣囊穿刺明顯

### 電路選擇

- 如果使用電路，30% O₂配合空氣是明智選擇

### 氣道火災處理

| 步驟 | 內容 |
|-----|------|
| 立即行動 | 立即關閉激光 |
| 移除 | 移除管道 |
| 沖洗 | 用0.9%氯化鈉淹沒氣道 |
| 滅火後 | 用100% O₂開始面罩通氣 |
| 評估 | 可能需要重新插管和支氣管鏡檢查加灌洗 |
| 嚴重損傷 | 可能需要氣管切開術 |

### 激光煙霧

- 激光煙霧在人體組織接觸部位產生
- 含有可能對健康工作者有害的細顆粒
- 必須使用煙霧疏散系統去除激光煙霧污染物
- 乳頭狀瘤激光手術期間可能發生病毒霧化
- 佩戴特殊口罩以減少風險

## 18.5 機器人手術

### 概述

世界各地進行的機器人手術數量持續增長，越來越多的手術現在可以通過機器人系統完成。

#### 優勢

**對外科醫生：**
- 操作時舒適度增加
- 3D視頻
- 過濾任何顫抖
- 運動縮放以允許精確工作

**對患者：**
- 微創方法更好的美容效果
- 端口部位疼痛減少
- 住院時間減少

#### 劣勢

- 對結果的影響存在爭議且取決於具體手術
- 成本增加
- 完成手術時間更長

### 機器人手術程序示例

| 專科 | 手術 | 考量 |
|-----|------|------|
| 乳房 | 乳房切除術 | 仰臥，腋窩入路 |
| 心臟 | 二尖瓣置換術 | 遠程端口部位出血 |
| 結直腸 | 大腸切除術 | 陡頭低位，腸道準備 |
| ENT、甲狀腺 | 甲狀腺切除術 | 腋窩或經口入路 |
| 舌根腫瘤 | - | - |
| 泌尿科 | 前列腺切除術 | 陡頭低位 |
| 骨科 | 關節置換術 | - |

### 術前準備

| 項目 | 內容 |
|-----|------|
| 定位 | 與外科醫生澄清患者定位需求，可能比非機器人手術更極端或不同 |
| 團隊簡報 | 高質量團隊簡報對機器人手術室的順利運行至關重要 |
| 患者評估 | 應正常進行 |
| 特殊考量 | 如果預期長時間陡頭低位，應注意顯著青光眼（可能導致眼科併發症包括視力喪失的發生率增加） |
| 腹腔鏡手術 | 考慮患者是否能耐受長時間氣腹 |
| 腸道準備 | 詢問腸道準備情況以計劃液體治療 |

### 圍手術期

| 項目 | 內容 |
|-----|------|
| 麻醉選擇 | 絕大多數機器人手術在全身麻醉下進行 |
| 氣道選擇 | 默認是ETT（因為手術期間對患者的訪問受限）應粘貼（非綁定）以促進頭部靜脈引流 |
| 眼睛保護 | 特別是在機器人腹腔鏡手術中（氣腹和陡定位的組合增加反流和化學灼傷的風險）眼睛應安全粘貼並放置墊子 |
| 機器人手臂 | 機器人操縱器可顯著延伸到人類操作員手持儀器之外的位置；在激活衝突檢測之前可能對患者造成重大傷害 |
| 安全桿 | 應使用安全的水平金屬棒保護面部或患者其他脆弱區域 |
| IV通路 | 應牢固固定，帶有允許遠程注射藥物的延長管；輸注應在对接前準備和連接 |
| 患者定位 | 這是機器人手術最重要的方面；應仔細注意脆弱區域、電線、IV線和任何患者支撐物的適當襯墊 |
| 肩部支撐 | 如果使用肩部支撐以防止頭低位時患者滑動，應注意頭部不要自由懸掛（可能導致臂叢神經拉伸損傷） |
| 休息 | 在可行時，每90-120分鐘應進行一次極端定位休息以減少隔室綜合徵的風險 |
| 表-機器人同步 | 某些機器人系統允許台和機器人運動之間的同步；在機器人对接時調整台位置之前檢查是否這種情況 |
| 患者運動 | 術中患者運動可能導致重大問題；輸注肌肉鬆弛劑或remifentanil有用 |

### 術後

| 項目 | 內容 |
|-----|------|
| 拔管前 | 進行洩漏測試（可能有顯著氣道水腫） |
| 腦水腫 | 長期頭低位可能導致即時恢復期的激動和譫妄 |
| 鎮痛計劃 | 通常與非機器人病例相同，但隨著對手術的熟悉程度增加可以向下滴定 |
| 脊髓麻醉 | 這是機器人腹腔鏡手術的極好選擇，但考慮到某些手術的時間，可能需要在最後用腹壁阻滯補充 |

## 18.6 日間手術

### 組織

**定義：**
- 手術「日間病例」是當天入院、手術和出院的患者
- 在英國（與美國不同），24小時停留不算日間手術

**組織關鍵要素：**

| 要素 | 內容 |
|-----|------|
| 設施 | 需要「專用」的手術室和病房空間 |
| 最佳配置 | 有自己的設施但在急診醫院內的獨立單位提供最佳選擇 |
| 升級 | 日間手術設施在醫院升級期間不得用於容納過夜患者 |
| 工作人員 | 應由經過充分培訓和經驗的工作人員進行日間病例麻醉和手術 |

### 日間手術程序

**英國日間手術協會**已製作目錄，列出200多個現在被認為適合日間進行的程序。

**日間手術標準：**

| 標準 | 內容 |
|-----|------|
| 疼痛控制 | 應可能用簡單口服鎮痛配合區域麻醉技術控制疼痛 |
| 飲食 | 應能夠在術後進食和飲水 |
| 活動 | 應能夠在術後活動（必要時使用輔助工具） |

**日間手術程序示例：**

| 專科 | 程序 |
|-----|------|
| 婦科 | 腹腔鏡，子宮切除術、陰道修復、陰道，子宮切除術 |
| 普通外科 | 腹腔鏡胃底折疊術 |
| 泌尿科 | 腹腔鏡腎切除術、腹腔鏡前列腺切除術、激光前列腺切除術 |
| 乳房手術 | 乳房切除術、所有非重建乳腺癌手術 |
| 血管外科 | 血管內主動脈修復 |
| 骨科 | 全髖/膝或肩置換術 |
| 急診 | 闌尾切除術、睪丸扭轉、「熱」腹腔鏡膽囊切除術、輸尿管結石激光 |

### 急診日間手術

**调度机会：**

| 機會 | 內容 |
|-----|------|
| 急診名單首選槽 | 患者在早上7:30到日間單位，作為急診名單上的首選患者 |
| 擇期日間手術名單上的計劃槽 | 高容量急診程序可以通過設置專用槽容納在擇期日間手術名單上 |
| 專門急診日間手術名單 | 允許患者出院並在接下來48小時內帶回急診日間手術名單 |
| 填補空缺 | 通過與急診團隊早期溝通利用可用手術時間 |

### 患者準備和優化

| 項目 | 內容 |
|-----|------|
| 理想做法 | 患者在外科門診預約當天在醫院接受評估 |
| 電話評估 | 某些患者群體可以進行電話評估（年輕、健康、不用藥、程序不需要特定測試） |
| 術前調查 | 應保持在絕對必要的手術範圍內 |
| 說明 | 必須給予患者關於術中情況的清晰說明 |

### 日間病例患者選擇標準

#### 健康狀況（ASA）

- 日間手術不再僅限於健康人
- 具有穩定疾病的患者大多適合日間手術
- 許多複雜患者如果管理得當，在日間病例環境中會更好

**關鍵問題：**
1. 我們能做什麼來使該患者能夠作為日間病例進行管理？
2. 如果患者作為住院患者治療，會有任何不同的做法嗎？

如果第二個問題的答案是「否」，那麼日間手術可能是最佳途徑。

#### 年齡

- 沒有年齡上限
- 應考慮生理 fitness而非實際年齡
- 嬰兒可以小到6週大（不包括早產兒<60週後胎齡因為SIDS風險）

#### 肥胖

- 即使是病態肥胖患者也可以成功作為日間病例管理
- 早期活動可以減少併發症風險
- 沒有BMI限制用於日間手術
- 評估應包括關於OSA的問題

#### 交通和社會支持

| 要求 | 內容 |
|-----|------|
| 陪伴 | 所有患者必須由負責任的知情成人護送回家 |
| 監督 | 在大多數情況下，在家恢復期間應有充分監督至少24小時 |
| 特殊情況 | 某些單位現在為沒有家庭支持的患者雇用護理人員 |
| 回家獨自 | 越來越接受更輕微手術的患者在術前評估中被判斷合適並希望獨自回家是可以的 |

#### 地理

- 通常，患者應住在距離能夠處理並發症的醫院1小時車程內

## 18.7 日間病例麻醉的執行

### 原則

| 原則 | 內容 |
|-----|------|
| 術前用藥 | 口服鎮痛 |
| 麻醉劑 | 短效 |
| 避免 | 致嘔吐藥物 |
| 多模式鎮痛 | 是 |
| 搶救鎮痛 | 短效阿片類藥物 |
| 術後鎮痛 | 良好 |

### 術前

| 項目 | 內容 |
|-----|------|
| 避免鎮靜術前用藥 | 除非必要（通常僅用於有行為問題的患者） |
| 抗酸藥物 | 常規使用不必要；但有反流病史者，雷尼替丁300 mg或奧美拉唑40 mg口服是合適的 |
| 口服鎮痛藥 | 對乙醯氨基酚1 g和NSAID在1-2小時達到峰值效果 |
| 保持溫暖 | 鼓勵患者在術前保持溫暖和充足水分 |
| 自由飲水 | 越來越多的證據支持術前自由攝入水而非任意「截止」時間 |

### 圍手術期

| 項目 | 內容 |
|-----|------|
| 短效藥物 | TIVA理想適合日間手術；七氟醚也可以成功 |
| 避免 | NO₂或長效阿片類藥物（如嗎啡） |
| 較大手術 | 考慮增量芬太尼高達2-4 mcg/kg分次劑量；避免IV嗎啡 |
| NSAID | 考慮（如果還沒給予） |
| 局部麻醉 | 每個合適的患者/手術都應考慮 |
| 保溫 | 用晶體液保持溫暖和充足水分（兒科患者15 mL/kg） |
| LMA | 盡可能使用LMA，避免插管、肌肉鬆弛劑和逆轉劑 |
| 止吐藥 | 不常規指示；應保留用於治療PONV或高風險患者預防 |

### 術後

| 項目 | 內容 |
|-----|------|
| 平衡鎮痛 | 對乙醯氨基酚、NSAID、LA和短效阿片類藥物通常足夠 |
| 需要更多鎮痛 | 早期使用是強制性的；考慮IV芬太尼50-100 mcg |
| 強效鎮痛 | 如需要，給予口服嗎啡或其他藥物如曲馬多 |
| 注意 | 嗎啡劑量>0.1 mg/kg會增加入院率 |
| 抗痙攣藥 | 如東莨菪鹼和物理治療（如熱水袋）可能有幫助，特別是用於婦科手術後的下腹絞痛 |

### 術後噁心和嘔吐（PONV）

**高風險患者的方法：**
- LA技術或使用TIVA的GA
- 避免阿片類藥物
- 多模式鎮痛治療
- 良好水分（IV液體）
- 最小化液體禁食
- 地塞米松8 mg配合環嗪50 mg或5-HT₃拮抗劑是有效的預防劑

### 區域麻醉

超聲的廣泛使用使許多程序能夠在純區域麻醉下進行，效率大幅提升。

**特殊阻滯：**

| 阻滯類型 | 說明 |
|---------|------|
| 外科醫生進行的局部阻滯 | 非常適合LA疝修補術 |
| 髂腹股溝神經阻滯 | 麻醉師嘗試有很高的股神經阻滯發生率，導致計劃外入院 |
| 脊髓 | 短效脊髓藥物如1%氯普魯卡因（40-60分鐘）和2%高比重丙胺卡因（90-120分鐘）革新了日間手術脊髓麻醉服務 |
| 股阻滯 | 通常不合適（因為術後活動困難）；但縫匠肌下阻滯阻滯感覺而非運動分支是前交叉韌帶修復或單髁或全膝關節置換的理想選擇 |

### 出院藥物

**方案示例：**

| 預期疼痛 | 示例程序 | 鎮痛方案 |
|---------|---------|---------|
| 無 | 膀胱鏡檢查 | 無 |
| 輕度 | 前列腺活檢 | 對乙醯氨基酚1 g qds |
| 中度 | 牙齒拔除、乳房切除術、關節鏡 | 對乙醯氨基酚1 g qds；布洛芬400 mg qds |
| 重度 | 腹腔鏡膽囊切除術、腹腔鏡，子宮切除術、疝修補術 | 聯合用藥30/500 2片qds；布洛芬400 mg qds；Macrogol一包bd |

### 出院標準

| 標準 | 內容 |
|-----|------|
| 生命體徵 | 穩定 |
| 意識 | 完全清醒和定向 |
| 口服 | 耐受流質 |
| 排尿 | 僅在特定程序後必要（如泌尿科、婦科或脊髓/骶骨麻醉後） |
| 活動 | 能行走 |
| 疼痛和噁心 | 控制良好 |
| 出血 | 最小或傷口引流最少 |

### 區域技術的特殊出院標準

**脊髓麻醉：**
- 運動力量和本體感覺完全恢復
- 患者已排尿

**區域阻滯：**
- 理解對部分阻滯肢體的保護
- 關於阻滯應該消退的時間以及如果沒有消退應聯繫誰的說明
- 必要時使用拐杖的足夠活動能力

### 非預期入院

**最常見原因：**

| 原因 | 說明 |
|-----|------|
| 未滿足出院標準 | 在單位關閉前 |
| 觀察 | 手術或麻醉並發症後 |
| 意外更大手術 | - |
| 未控制的疼痛或PONV | - |

## 18.8 腦死亡的神經學判定

### 死亡的定義

死亡是當這個過程發展到不可逆轉地失去意識能力並伴有不可逆轉地失去呼吸能力時診斷的。

### 腦死亡標準

| 標準 | 說明 |
|-----|------|
| 腦死亡 | 許多國家使用的神經學判定死亡標準 |
| 腦幹死亡 | 英國使用（不等同於腦死亡） |

**腦幹死亡要點：**
- 它提供了撤除侵入性通氣支持和合法和倫理基礎
- 確認腦幹死亡使器官捐贈成為可能（如符合患者和家屬的意願）

### 腦幹死亡的常見原因

| 原因 | 說明 |
|-----|------|
| 創傷性腦損傷 | - |
| 顱內出血 | - |
| 缺氧性腦損傷 | - |

### 腦幹死亡判定的先決條件

| 先決條件 | 要求 |
|---------|------|
| 昏迷 | 患者昏迷、無呼吸、依賴機械通氣 |
| 病因 | 由已知病因引起的不可逆性腦損傷引起 |
| 血壓 | 成人MAP≥60 mmHg；兒童年齡適當的血壓 |
| 體溫 | >34°C |
| 可逆原因排除 | 鎮靜劑、肌肉鬆弛劑、酒精、代謝或內分泌紊亂 |
| 代謝水平 | Na⁺ 115-160 mmol/L；K⁺ >2；Mg²⁺ 0.5-3；磷酸鹽 0.5-3；葡萄糖 3-20；尿素 <40 |
| 神經肌肉阻斷 | 除非確定未給予NMBA，否則應使用PNS確認神經肌肉傳導正常 |
| 腦幹反射檢查 | 至少一耳和一眼可用於檢查 |
| 呼吸暫停測試 | 必須能夠進行 |

### 觀察期要求

- 當患者滿足先決條件且GCS為3時觀察
- 瞳孔固定且無咳嗽
- 無呼吸
- 缺氧-缺血性腦病或低體溫後延遲24小時臨床測試

### 腦幹反射測試

| 反射 | 測試方法 |
|-----|---------|
| 瞳孔 | 固定，對銳利光強變化無反應 |
| 角膜反射 | 不存在 |
| 運動反應 | 在三叉神經分佈區域對中央或外周疼痛刺激無運動反應 |
| 眼前庭反射 | 注射50 mL冰水到外耳道，頭部抬高30°，無眼球運動 |
| 咽反射 | 後咽刺激無作嘔反應 |
| 咳嗽反射 | 通過ETT的吸痰管無咳嗽 |

### 呼吸暫停測試確認

| 步驟 | 內容 |
|-----|------|
| 預充氧 | 用100% O₂通氣至少5分鐘 |
| 減少通氣 | 達到呼氣末PaCO₂ 6 kPa（45 mmHg） |
| 檢查ABG | 確認PaCO₂至少6 kPa且pH <7.4 |
| 斷開 | 從呼吸機斷開，ETT連接到自動充氣囊，帶至少5 cmH₂O PEEP和15 L/min O₂ |
| 觀察 | 持續觀察至少5分鐘是否有任何呼吸運動 |
| 測量PaCO₂ | 應足夠高以確保對通氣的充分刺激 |
| 陽性測試 | 如果無自發呼吸且PaCO₂升高>0.5 kPa |

### 兒童

| 年齡 | 考量 |
|-----|------|
| >2個月 | 標準與成人相同 |
| <37週胎齡新生兒 | 由於腦幹反射不成熟，神經學臨床檢查不可靠 |
| 37週至2個月 | 可能確認死亡，但需要謹慎和專家建議 |

### ECMO上的患者

- 可以在ECMO患者上進行腦幹死亡測試
- 英國重症監護醫學學院和英國重症監護學會提供了測試指南

### 輔助和確認檢查

- 英國不需要輔助檢查
- 如果無法進行完整神經學檢查（如廣泛面部創傷、無法進行呼吸暫停測試），則必須進行進一步檢查
- 可能包括腦電圖和腦血流放射學檢查

### 其他考量

| 項目 | 內容 |
|-----|------|
| 測試次數 | 必須由不同醫生進行兩次 |
| 死亡確認時間 | 通常在無呼吸昏迷發作後6小時；如果是心臟驟停後，24小時 |
| 低體溫後 | 如果發生長時間低體溫（<34°C >6小時），在恢復正常體溫後至少24小時進行測試 |
| 死亡時間 | 第二次測試後確認，但英國記錄為第一次測試完成的時間 |
| 驗屍官通知 | 應以通常方式通知 |

### 與腦幹死亡診斷不相容的觀察

- 癲癇發作
- 去皮質和去腦強直
- 在顱神經分佈刺激時引起的肢體運動

### 與腦幹死亡相容的觀察

- 脊髓反射
- 臉紅
- 尿崩症缺失

### 循環死亡後的器官捐贈

| 項目 | 內容 |
|-----|------|
| 確認死亡觀察 | 負責確認死亡的人必須觀察無呼吸、無意識和动脉線踪跡或心電圖上5分鐘無脈動性或電活動 |
| 5分鐘後 | 如果對光、角膜或眶上壓力無瞳孔反應、無運動反應，確認死亡 |

## 18.9 腦死亡後的器官捐贈手術

### 程序概述

| 項目 | 內容 |
|-----|------|
| 程序 | 通過正中開腹和正中胸骨切開術獲取捐贈器官 |
| 時間 | 最長6小時，取決於獲取哪些器官 |
| 疼痛 | 不適用 |
| 體位 | 仰臥 |
| 失血 | 可能大量，X-match 4單位 |
| 實際技術 | IPPV、中心靜脈通路和动脉線 |

### 腦幹死亡的病理生理學

| 階段 | 變化 |
|-----|------|
| 早期 | 腦幹疝出期間發生短暫大量交感神經輸出，引起高血壓、心動過速、心肌功能障礙、器官灌注受損和組織缺血 |
| 後期 | 自主神經崩潰導致CO減少、血管運動張力喪失、低血壓和阿托品耐藥性心動過緩；如果不治療，所有器官灌注不足 |
| 肺部 | 神經源性肺水腫和急性肺損傷可能發生 |
| 內分泌 | 後葉垂體功能通常喪失；抗利尿激素減少導致神經源性尿崩症伴低血容量和電解質紊亂 |
| 血糖 | 由於循環胰島素減少和胰島素抵抗而升高 |
| 凝血 | 來自壞死腦組織的組織纖溶劑和纖溶酶原激活劑釋放導致凝血病 |
| 體溫 | 由於全身代謝減少、下丘腦功能障礙、未治療尿崩症的大量稀釋尿液和無法顫抖和血管收縮而降低 |

### 捐贈前管理

| 項目 | 內容 |
|-----|------|
| 通路 | 多腔中心線（頸內或鎖骨下首選）、一大孔徑外周IV通路（14G或16G）和动脉線（上肢首選） |
| 確認 | 檢查腦幹死亡已確認且家屬已同意捐贈 |
| 管理重點 | 從腦復甦轉變為最佳器官灌注和氧合 |
| 交感神經過度活躍 | 使用短效β-阻滯劑（如艾司洛爾）治療嚴重高血壓（MAP >130 mmHg）和心動過速 |
| 腦幹死亡時 | 交感神經張力喪失，低血壓發生；給予復甦液（500 mL至1 L）的容量挑戰並準備開始血管加壓素（0.01-0.04 units/min） |
| 糖尿病尿崩症 | 可能需要去氨加壓素（dDAVP）控制 |
| 肺功能 | 通常異常；如果PaO₂在100% O₂和5 cmH₂O PEEP下 >40 kPa，肺捐贈通常可能 |

### 器官獲取的目標參數

| 參數 | 目標 |
|-----|------|
| MAP | 60-80 mmHg |
| 心率 | 60-120/分鐘 |
| 心臟指數 | >2.1 L/min/m² |
| 核心體溫 | 36-37.5°C |
| 血紅蛋白 | >70 g/L（CV穩定捐贈者） |
| | >90 g/L（不穩定捐贈者） |
| SpO₂ | >95%，用最低FiO₂維持PaO₂ >10 kPa |
| 潮氣量 | 6-8 mL/kg |
| PaCO₂ | NHS指南：只要pH>7.25為5-6.5 kPa |
| 平臺壓 | <30 cmH₂O |
| 尿量 | 0.5-2 mL/kg/h |

### 圍手術期

| 項目 | 內容 |
|-----|------|
| WHO安全檢查清單 | 應用 |
| 監測 | 標準監測加动脉線、核心體溫和尿量 |
| 麻醉需求 | 腦死亡患者不能體驗疼痛和缺乏意識，因此不需要真正的「麻醉」；但需要NMBA、揮發性藥物，有時需要阿片類藥物以防止脊髓反射運動反應 |
| 血流動力學波動 | 由於腔靜脈壓迫、腎上腺操作和血/液體流失而發生；低血壓用晶體液、血液製品、血管加壓素和正性肌力藥物治療 |
| 肝素化 | 在主動脈鉗夾前外科醫生要求時給予IV肝素（300 units/kg） |
| 器官獲取順序 | 心臟、肺，然後肝臟；腹部外科團隊在心臟驟停下繼續操作，獲取腎臟、胰腺和血管 |

### 循環死亡後捐贈（DCD）

| 項目 | 內容 |
|-----|------|
| 定義 | 當死亡使用心肺標準確認時，在特定情況下可以進行移植目的的器官獲取 |
| 術語變化 | 以前稱為心臟死亡後捐贈或非心跳器官捐贈 |
| 英國現狀 | 過去十年大幅增加，現在佔死亡器官捐贈者的近40% |
| 可捐贈器官 | 肝臟、腎臟、心臟、胰腺和肺 |
| 考量 | 只能在確定撤回生命維持治療符合患者最大利益後考慮 |
| 撤回生命維持 | 標準臨終護理繼續，包括給予預防痛苦的藥物 |
| 結果 | 如果死亡不在允許器官捐贈的時間範圍內，應有既定計劃進行持續臨終護理 |

## 18.10 主要麻醉事故：事後處理

### 事件類型

大多數麻醉師在職業生涯中會經歷至少一次災難性麻醉事故，導致患者死亡或重大傷害。

### 手術室中的立即行動

#### 聯繫高級同事

- 應盡快讓顧問麻醉師參與
- 這些是罕見事件，帶來沉重情感負載
- 需要幫助

#### 臨床記錄

| 項目 | 內容 |
|-----|------|
| 記錄要求 | 詳細、真實且理想的同期事件記錄 |
| 即時記錄 | 在復甦期間，此任務可能分配給團隊另一成員，但麻醉師必須盡快記錄完整情況 |
| 內容 | 術前討論詳情、患者的發病前狀況、如何識別和治療緊急情況 |
| 保存 | 如果可能，保存加密副本 |

#### 患者護理

| 情況 | 行動 |
|-----|------|
| 患者死亡 | 需要宣布；所有術中死亡必須報告驗屍官；線、管和其他設備通常應保持原位；患者可以清洗、縫合和包紮傷口 |
| 患者存活 | 麻醉師應跟進患者進展；患者現在將由另一團隊（一ICU）主要護理；麻醉師的角色是表示適當關注 |

#### 隔離設備和藥物

- 如果懷疑設備或藥物是災難的原因或原因不明
- 所有程序設備、藥物和注射器必須隔離並儲存以供後續調查
- 如果懷疑手術室或麻醉機促成事件，手術室應關閉等待調查

### 家屬護理

| 項目 | 內容 |
|-----|------|
| 通知 | 壞消息不應通過電話透露；應告知發生了並發症，需要立即來醫院 |
| 支持人 | 建議他們帶支持人 |
| 翻譯 | 如果英語不是家屬的母語，必須使用翻譯 |

**壞消息會議：**

| 項目 | 內容 |
|-----|------|
| 會前會 | 識別主要發言人；總結要討論的事件和事實 |
| 地點 | 安靜的房間，不會被打擾 |
| 開始 | 簡潔地報告壞消息，使用易懂的非醫學語言 |
| 道歉 | 不是承認罪責，而是同理心和關懷的表示 |
| 記錄 | 必須記錄在臨床記錄中 |

### 團隊護理

| 項目 | 內容 |
|-----|------|
| 手術室關閉 | 之後的手術應取消或由另一團隊進行 |
| 即時彙報 | 盡快安排「熱」彙報會議 |
| 目的 | 允許員工談論發生的事情，釋放情緒 |

### 麻醉師護理

**情緒影響：**
- >70%經歷災難性術中事件的麻醉師將有高度情緒影響
- 經歷內疚、焦慮、失眠、對同事評判和訴訟的恐懼、憤怒和重現事件
- 也可能遭受身體影響如疲倦、肌肉緊張和噁心

**支持：**
| 項目 | 內容 |
|-----|------|
| 導師支持 | 應盡快尋求或安排 |
| 心理支持 | 可能也需要 |
| 職業健康 | 可能需要通知 |
| 返回工作 | 可能需要額外支持 |

### 正式調查和醫療法律考量

| 調查類型 | 內容 |
|---------|------|
| 死亡和發病率會議 | 由高級麻醉師主持；應以吸取教訓而非追究責任的方式進行 |
| 內部調查/根本原因分析 | 很可能進行；訪談和陳述 |
| 驗屍官調查 | 所有麻醉下或與手術相關的死亡都需要轉介 |
| 醫療委員會調查 | 如果對醫生的表現或能力有疑慮 |
| 刑事起訴 | 罕見；如果懷疑情況或臨床醫生嚴重過失 |
| 民事訴訟 | 可能對醫生或醫院提起疏忽訴訟 |
