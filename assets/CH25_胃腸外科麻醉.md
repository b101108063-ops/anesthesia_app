## 25.1 主要胃腸手術

### 一般考量

主要腹部手術產生神經內分泌、代謝和炎症反應，可能導致不良生理變化，包括肺部功能障礙、心臟需求增加、疼痛、噁心和嘔吐。

### 術前準備

| 項目 | 內容 |
|-----|------|
| 病史/檢查 | ECG（如指示）、FBC、U&E；其他血液測試根據患者合併症 |
| 運動功能 | 評估運動功能（如CPET）；優化心臟和呼吸功能 |
| 營養 | 優化營養；炎症性腸病患者可能嚴重營養不良和免疫抑制 |
| 多模式鎮痛 | 關鍵組成部分：腹壁阻滯和導管、傷口導管、IV利多卡因輸注、神經軸索阻滯 |
| 藥物管理 | 術前24小時停用ACE抑制劑/ARBs |
| 術前用藥 | 如果有反流風險考慮H₂拮抗劑或PPI；如果指示討論RSI |

### 監測建議

| 監測 | 適應症 |
|-----|--------|
| 微創CO監測 | 心血管抑制、有可能液體轉移、可能需要圍手術期正性肌力藥物 |
| 动脉線 | 心血管、呼吸抑制、重大失血、需要血氣採樣 |
| CVP線 | 需要血管加壓素和/或正性肌力藥物；需要術後腸外營養 |

### 圍手術期考量

| 考量 | 內容 |
|-----|------|
| IV通路 | 大孔徑IV通路；如果手臂訪問受限使用長延長管 |
| 神經軸索阻滯 | 如果指示用於術後鎮痛（如脊髓配合長效阿片類藥物或低位胸椎硬膜外） |
| 監測 | 如指示進行侵入性監測 |
| 體溫 | 避免低體溫；瞄準正常體溫 |
| 低血壓 | 避免；術中低血壓與不良結局（包括心肌損傷和死亡）強相關 |
| 液體管理 | 適度自由（1-2 L正向）或目標導向方法 |

### 術後考量

| 考量 | 內容 |
|-----|------|
| 早期恢復 | 瞄準在術後立即喝水、進食和活動 |
| 氧疗 | 開處方門診氧疗以維持SpO₂ >95% |
| PONV | 積極治療 |
| 液體平衡 | 密切監測；考慮腹部引流、迴腸造口和NG吸引的持續損失 |
| 簡單鎮痛 | 開處方regular對乙醯氨基酚和（如不禁忌）NSAID |
| 急性疼痛團隊 | 轉介進行術後評估 |

## 25.2 開放胃腸手術

| 程序 | 時間（小時） | 體位 | 失血（L） | 鎮痛建議 |
|-----|------------|------|----------|---------|
| 胃切除術 | 2-3 | 仰臥 | 0.5-1 | 神經軸索鎮痛（硬膜外或脊髓）；考慮CVP和动脉線 |
| 胰切除術 | 3-6 | 仰臥 | 0.5-2 | 常為複雜手術；考慮神經軸索鎮痛；考慮CVP和动脉線 |
| 膽囊切除術 | 1 | 仰臥 | 0.5 | 右、上象限；考慮TAP阻滯或傷口導管± PCA |
| 闌尾切除術 | 1 | 仰臥 | 微 | 右、下象限；考慮TAP阻滯± PCA |
| 右結腸切除術 | 1-3 | 仰臥 | 0.5-1 | 考慮神經軸索鎮痛或腹直肌鞘導管± PCA |
| 左結腸切除術 | 1-3 | Lloyd-Davies | 0.5-1 | 考慮神經軸索鎮痛或腹直肌鞘導管± PCA |
| 前切除術 | 2-3 | Lloyd-Davies | 0.5-1.5 | 考慮神經軸索鎮痛或腹直肌鞘導管± PCA |
| 腹會陰切除術 | 2-4 | Lloyd-Davies然後俯臥 | 0.5-2 | 考慮神經軸索鎮痛 |
| Hartmann程序 | 1-3 | Lloyd-Davies | 0.5-1 | 考慮神經軸索鎮痛或腹直肌鞘導管± PCA |
| Hartmann逆轉 | 1-2 | Lloyd-Davies | 0.5-1.5 | 考慮神經軸索鎮痛或腹直肌鞘導管± PCA |
| 腸造口關閉 | 0.5-1 | 仰臥 | 微 | 肌肉鬆弛劑通常足夠 |

## 25.3 腹腔鏡胃腸手術

### 腹腔鏡手術的優點

| 優點 | 內容 |
|-----|------|
| 組織創傷減少 | 減少疼痛和組織創傷 |
| 術後呼吸功能改善 | - |
| 術後腸梗阻減少 | - |
| 早期活動 | 縮短住院時間 |
| 美容效果 | 改善 |

### 氣腹的生理效應

| 系統 | 效應 |
|-----|------|
| 呼吸 | 氣道壓力增加；FRC減少；肺順應性降低；V/Q不匹配增加 |
| 心血管 | 靜脈回流減少；SVR增加；CO減少；心律不整風險增加 |
| 神經 | ICP增加 |
| 胃腸 | ICP增加；反流風險增加 |

### 體位效應

| 體位 | 效應 |
|-----|------|
| 頭低腳高 | 肺部損害風險增加（FRC減少、V/Q不匹配、肺不張）；支氣管內插管風險增加；面部、氣道和腦水腫風險 |
| 頭高腳低 | 血壓和CO減少（靜脈回流減少）；低血容量患者、老年人和有心血管疾病患者風險增加 |

### 圍手術期考量

| 考量 | 內容 |
|-----|------|
| 氣道 | GA配合氣管內插管、肌肉鬆弛劑和控制通氣；保護防止肺誤吸 |
| 避免胃擴張 | 避免面罩通氣期間胃擴張（可能增加套管插入時胃損傷風險） |
| 尿導管 | 下腹部程序可能需要 |
| 通氣 | 瞄準正常碳酸血症；注意高分鐘通氣量需求 |
| 鎮痛 | 根據程序而定；術後疼痛通常比開放程序少很多 |
| PONV | 腹腔鏡手術與高發病率相關；給予預防性止吐藥 |

### 缺氧原因

| 原因 | 說明 |
|-----|------|
| 通氣不足 | 由於氣腹和頭低位導致通氣不足 |
| CO減少 | 下腔靜脈壓迫、心律不整、出血、心肌抑製、靜脈氣體栓塞、膜外氣體 |
| V/Q不匹配 | FRC減少、肺不張、支氣管內插管、靜脈氣體栓塞、肺誤吸 |

## 25.4 腹腔鏡膽囊切除術

| 項目 | 內容 |
|-----|------|
| 程序 | 腹腔鏡膽囊切除 |
| 時間 | 40-80分鐘 |
| 疼痛 | ++ |
| 體位 | 仰臥，15-20°頭高腳低，輕微左傾 |
| 失血 | 不顯著 |
| 實用技術 | GA、ETT、IPPV |

### 特殊考量

| 考量 | 內容 |
|-----|------|
| 患者特點 | 經典型「女性、四十、肥胖、多產」 |
| 轉換 | 約5%病例需要轉換為開放程序 |
| 術後疼痛 | 膽囊床LA可減少術後鎮痛需求 |

## 25.5 腹腔鏡闌尾切除術

| 項目 | 內容 |
|-----|------|
| 程序 | 闌尾切除 |
| 時間 | 20-40分鐘 |
| 疼痛 | ++ |
| 體位 | 仰臥 |
| 失血 | 不顯著 |
| 實用技術 | RSI、ETT、IPPV、TAP阻滯 |

### 特殊考量

| 考量 | 內容 |
|-----|------|
| 患者特點 | 通常年輕fit（除非闌尾破裂） |
| 破裂 | 如果闌尾破裂更可能轉換為開放程序 |

## 25.6 腹股溝疝修復

| 項目 | 內容 |
|-----|------|
| 程序 | 通過腹股溝肌肉管的缺陷修復（腸可能突出） |
| 時間 | 30-60分鐘 |
| 疼痛 | ++ |
| 體位 | 仰臥 |
| 失血 | 不顯著 |
| 實用技術 | GA、SV、SGA、腹股溝區域阻滯；脊髓；局部浸潤和/或鎮靜 |

### 區域阻滯技術

| 神經 | 位置 |
|-----|------|
| 髂腹股溝和髂腹下神經 | 髂前上棘尾部2 cm處 |
| 生殖股神經 | 腹股溝韌帶中點上方1-2 cm |

## 25.7 肛門和肛門周圍程序

| 項目 | 內容 |
|-----|------|
| 程序 | 痔瘡切除、肛裂、括約肌修復、肛門周圍膿腫、藏毛竇 |
| 時間 | 20-30分鐘 |
| 疼痛 | +++ |
| 體位 | 仰臥、截石位、偶爾俯臥 |
| 失血 | 不顯著 |
| 實用技術 | GA、SV、SGA；脊髓（「鞍狀阻滯」） |

### 特殊考量

| 考量 | 內容 |
|-----|------|
| 迷走刺激 | 由於迷走神經張力增加可能導致心動過緩或心搏停止； anticipate並準備迷走神經阻滯藥 |
| 骶骨僅脊髓 | 重 bupivacaine是有用的替代方案，對心血管動力學影響很小 |
| Pilonidal竇 | 可能需要俯臥位；術前確認並準備患者 |
