## 口腔頜面外科手術（Oral/Maxillofacial Surgery）

### 總論

口腔/頜面手術的麻醉需要管理共用的（通常困難的）氣道。鼻插管常用於改善口腔手術入路、方便 X 光片拍攝並允許咬合評估。

術前訪視時，檢查鼻孔通暢度並詢問鼻出血和抗凝血劑使用情況。與外科醫師討論氣道選擇。

### 氣道管理要點

| 項目 | 說明 |
|------|------|
| 聲門上氣道 | 簡單口腔內程序通常可使用加強型聲門上氣道。單側口腔內程序，對側口腔的口腔氣管內管（如 Rae 管）可接受 |
| 鼻腔插管 | 鼻腔途徑選擇時，使用局部麻醉和/或血管收縮劑混合物（如 Lidocaine 5%/Phenylephrine 0.5% 或 Xylometazoline）。鼻腔預成型管（如 Portex Ivory Preformed North-Facing Nasal）理想 |
| 尺寸選擇 | 應備有 6.0、6.5 和 7.0 mm 尺寸。使用前放置於溫水中進一步軟化 |
| 固定 | 管子應以紗布墊好保護患者額部。考慮以透明黏合膜固定氣管內管、鼻胃管和溫度探頭 |
| 眼部保護 | 以膠帶和眼墊或手術放置的塑膠隱形眼鏡（如 Crouch Corneal Protector）保護眼睛 |

### 體位與設備

- 頭部放置於麻醉機的對側
- 通常需要長呼吸回路和氣體分析/肺活量管線
- 以管子固定器固定呼吸回路
- 確使氣囊測試閥門易於觸及且遠離眼睛
- 以馬蹄形墊或頭環穩定頭部
- 上顎手術時，肩膀下墊枕墊進一步伸展頸部
- 頭部稍高可減少出血

### 喉部紗布

不建議常規使用喉部紗布。若使用，應由外科醫師放置並有健全系統確保不被遺留。應包括：

- 「Time Out」時討論
- 視覺提醒（如喉部紗布貼紙和紗布/尖銳物板上的記錄）
- 「Sign Out」紗布計數時確認移除
- 清晰文件記錄

### 拔管

有血液、膿液和碎屑吸入的風險。程序結束時應抽吸口咽部和喉部（最好在直視下）。

患者應坐位（30-45 度）拔管以減少靜脈充血造成的出血（或者左側臥、頭低腳高位如果有高吸入風險）。

部分麻醉醫師以「深」自主呼吸技術為患者拔管。部分將氣管內管交換為聲門上氣道，讓患者在恢復室慢慢醒來。部分偏好清醒拔管。

無論使用何種技術，目標是避免咳嗽和用力引起的出血和水腫。使用對咽反射刺激較少的鼻腔氣管內管或聲門上氣道有助於更順暢的拔管。也考慮在插管時噴霧聲帶。

## 眼眶顴骨複合骨折（Fractures of the Orbitozygomatic Complex）

### 手術基本資料

| 項目 | 內容 |
|------|------|
| 手術方式 | 骨折的顴骨複合體抬高 +/- 固定 |
| 手術時間 | 10 至 180 分鐘 |
| 疼痛程度 | +/++ |
| 體位 | 仰臥，頭高位，頭環 |
| 失血量 | 輕微（內固定時可能較多）|
| 實用技術 | 口腔 Rae 管和間歇性正壓換氣。簡單抬高時用聲門上氣道/自主換氣 |

### 術前考量

骨折可單獨發生或與面部其他部分損傷相關。評估相關損傷，特別是頭頸部損傷。這些骨折的治療不具有高臨床優先級（除非視神經受累引起球後出血）。手術常在 5-7 天後、面部水腫減少後較容易。進行仔細的氣道評估。

### 術中管理

口腔 Rae 管插管。簡單骨折抬高時可用靈活聲門上氣道，但需與外科醫師討論是否計劃開放固定。

潤滑並保護非手術側的眼睛。如放置金屬物品，根據當地方案給予抗生素（如 Co-amoxiclav 1.2 g 或 Clindamycin 600 mg）和類固醇（如 Dexamethasone 6.6 mg 靜脈）。

準備好顴骨骨折被復位時可能發生的心搏過緩（由迷走神經介導）。

### 術後護理

恢復室可能需要靜脈鴉片類。開立口服鎮痛處方。恢復室和病房進行眼部觀察以檢測術後球後出血（需要返回手術室）。

## 下顎骨骨折（Mandibular Fractures）

### 手術基本資料

| 項目 | 內容 |
|------|------|
| 手術方式 | 下顎骨骨折的復位和固定 |
| 手術時間 | 1 至 3 小時 |
| 疼痛程度 | + |
| 體位 | 仰臥，頭高位，頭環 |
| 失血量 | 變異；通常不需要備血 |
| 實用技術 | 鼻腔管和間歇性正壓換氣。可能需要光纖插管 |

### 術前考量

確保對相關損傷進行仔細評估。對氣道進行細緻評估。可能存在嚴重的牙關緊閉和明顯軟組織腫脹。

檢查鼻孔通暢度。檢查是否有顱底骨折和腦脊髓液滲漏的證據，因為這些是鼻腔插管的禁忌，需要與專科神經外科中心討論。

### 術中管理

鼻腔插管、氣管切開術或經舌下插管。外科醫師無法在口腔氣道周圍工作。

急性牙關緊閉使術前插管看起來可能困難，因為張口可能明顯受限，但這傾向在誘導後放鬆。陳舊性骨折和併發感染的患者通常不太會放鬆。

明顯腫脹可能使插管更困難，偶爾需要光纖輔助插管。雙側下顎骨骨折在誘導後可能允許更多的前部顎移位，但由於腫脹和顎結構喪失，面罩維持氣道可能不總是容易。快速順序插管可能合適（使用琥珀膽鹼或羅庫溴銨，備好Sugammadex）。

## 上顎/下顎截骨術（Maxillary/Mandibular Osteotomy）

### 手術基本資料

| 項目 | 內容 |
|------|------|
| 手術方式 | 面部骨骼的手術重新排列 |
| 手術時間 | 3 至 6 小時 |
| 疼痛程度 | ++ |
| 體位 | 仰臥，頭高位，頭環 |
| 失血量 | 變異。偶爾可能嚴重；血型及交叉比對 |
| 實用技術 | 鼻腔管和間歇性正壓換氣。考慮动脉導管 |

### 術前考量

進行仔細的氣道評估。檢查鼻孔通暢度。檢查血紅素和血型及交叉比對（根據當地指南）。

血栓栓塞預防：加壓襪；考慮在手術室使用間歇性氣動加壓靴。

### 術中管理

使用預成型鼻腔管進行鼻腔插管。噴霧聲帶和喉部可能減少拔管時的咳嗽。

良好靜脈通路。考慮由於手術時間長而進行侵入性壓力監測。

潤滑眼藥膏入眼並以墊和膠帶或塑膠隱形眼鏡保護。

輕度誘導性低血壓有助於減少失血。避免高血壓。考慮 Remifentanil 輸注（0.1-0.75 mcg/kg/min，或 TCI 2-6 ng/mL，雖然偶爾可能需要 >12 ng/mL）。其他選擇包括硝酸甘油輸注（2-10 mg/h）和 β-阻斷（ 如 Labetalol 5-10 mg 增量或輸注）。一般指導原則是維持平均動脈壓在 60 mmHg 或以上。

根據當地方案靜脈給予抗生素和類固醇以減少水腫。給予預防性止吐藥。

測量核心體溫；溫暖的靜脈液體和使用加溫墊和/或暖風機。這些可能需要在 2-3 小時後減少，因為患者容易過熱。

### 拔管

術末很少將患者顎部固定在一起。確保喉部紗布移除並清理口咽部血液和碎屑。

患者完全清醒後拔管。退出鼻腔管（鼻孔處 15 cm 標記）；插入安全別針並剪斷，留作鼻咽氣道。

### 術後護理

部分單位將這些患者送往重症監護室。其他在延長恢復期後送往病房。

給予加濕氧氣。確保所有口服鎮痛藥以可溶解形式開立。也應開立靜脈自控鎮痛或肌肉注射鴉片類。

繼續預防性抗生素和類固醇，根據單位方案（通常 24-48 小時）。開立靜脈輸液。鼓勵患者盡快經口攝入液體。

## 頜面部腫瘤手術（Maxillofacial Tumour Surgery）

### 手術基本資料

| 項目 | 內容 |
|------|------|
| 手術方式 | 頭頸部腫瘤切除。可能包括氣管切開術形成、入路手術、頸部廓清、消融手術和移植或皮瓣重建 |
| 手術時間 | 2 至 10+ 小時 |
| 疼痛程度 | ++ |
| 體位 | 仰臥，頭環，頭高位，頸部廓清或上顎手術時肩膀下墊枕 |
| 失血量 | 中等；通常血型及交叉比對即可 |
| 實用技術 | 間歇性正壓換氣配合北向預成型鼻腔氣管內管，或對側 Rae 管。可能需要氣管切開術。長時間重建手術用动脉導管、導尿管 |

### 術前考量

評估氣道，注意腫瘤位置和大小、相關影像、阻塞症狀、張口和頸部活動度、麻醉記錄、放射治療病史和先前頜面/頸部手術。與外科醫師討論。

患者常有吸菸和大量饮酒史。尋找心血管和呼吸共病症。

### 重建皮瓣手術

頜面重建使用局部（如鼻唇）、區域（如胸大肌）或游離皮瓣（特別是橈骨前臂/腓骨/前外側大腿）。

大手術可能漫長，6-10+ 小時。

原則與整形手術游離皮瓣相同（見相關章節），加上氣道可能困難的額外複雜性（術前和術後）。

保持患者溫暖。低體溫增加手術部位感染率，血管收縮和痙攣可能威脅皮瓣存活。避免高血壓。

重症監護或加護病房護理通常在術後指示，用於液體平衡、血壓控制、皮瓣監測和快速氣道救援。

傳統上，由於術後氣道併發症風險，常進行手術氣管切開術。在重症監護以鼻腔氣管內管鎮靜和過夜換氣越來越常見，可能避免氣管切開術。

## 牙科鎮靜（Sedation for Dentistry）

### 總論

無法在局部麻醉下耐受牙科治療的患者常可通過聯合鎮靜技術管理。這些程序通常由牙醫在牙科門診執行。

口服或靜脈鎮靜可由短效苯二氮平類如 Midazolam 給予。吸入鎮靜可由亞麻醉濃度的笑氣（達 50%）經鼻罩給予——稱為「相對鎮痛」。

無論使用何種給予途徑，確保患者全程保持清醒是重要的。患者必須由受訓工作人員監測（而非可能被程序分心的操作者）。

### 適合的方案

| 項目 | 說明 |
|------|------|
| 單一藥物方案更安全 | - |
| 成人 Midazolam | 2 mg 靜脈；等待 90 秒，然後每 30 秒給予 1 mg 直到鎮靜。預期給予 6-10 mg（牙科進展麻醉學會指南）。老年人減少劑量 |
| 低劑量 Propofol 輸注 | 僅在有適當訓練時 |
| 笑氣 | 經鼻罩 100% 氧氣；加入 10% 笑氣 1 分鐘，然後 20% 1 分鐘。持續每次增加 5% 直到鎮靜（達 50%）|

### 特殊考量

- 備好 Flumazenil 用於逆轉 Midazolam
- 靜脈鎮靜後允許至少 1 小時恢復
- 笑氣鎮靜後，必須給予 100% 氧氣以防止擴散性低氧
- 患者能夠站立和獨立行走後方可出院

## 牙科全身麻醉（General Anaesthesia for Dentistry）

### 總論

牙科程序的全身麻醉應在醫院進行，僅用於無法耐受局部麻醉的患者（如幼童、有學習障礙的成人、埋伏/埋藏牙）。

學習障礙患者可能難以理解程序且常焦慮。短效抗焦慮藥如 Midazolam 和局部麻醉乳膏可能有幫助。

患者可能有更複雜的醫療疾病如唐氏症或其他先天性異常。排除任何重大心臟病理是很重要的。

抗生素預防感染性心內膜炎不應常規用於牙科患者。對於高風險患者，預防可能合適。NICE 建議臨床醫師應根據個人情況應用臨床判斷，解釋風險和益處並讓患者參與決策。

### 齒列標記

乳牙以每象限的字母 A-E 標記，成人牙齒以數字 1-8 標記。牙根以「X」表示，多生牙以「$」表示，埋伏或未萌出牙齒以字母/數字周圍的圓圈表示。

## 牙齒拔除（Dental Extractions）

### 手術基本資料

| 項目 | 內容 |
|------|------|
| 手術方式 | 牙齒拔除 |
| 手術時間 | 2 至 30 分鐘（成人附加修復工作可能更長）|
| 疼痛程度 | +/++ |
| 體位 | 仰臥 |
| 失血量 | 無 |
| 實用技術 | 聲門上氣道/鼻罩（長時間修復工作用鼻腔氣管內管）|

### 術前考量

通常為 3-12 歲兒童、牙科恐懼症患者或有學習障礙的患者。

注意未確診的病理，如心臟雜音。必要時取得直腸鎮痛栓劑的同意。給予先發性口服鎮痛，如 Acetaminophen 和 Ibuprofen。

如需要鎮靜術前用藥，考慮口腔或鼻腔 Midazolam（500 mcg/kg，最大 10 mg）。

### 術中管理

靜脈誘導用 Propofol，吸入誘導用七氟烷。以膠帶將眼睛封閉。以吸入劑或靜脈劑維持。

口腔外科醫師給予局部麻醉浸潤；短時間門診手術通常不需要鴉片類。簡單拔牙通常不需要抗生素覆蓋。

程序期間手動穩定頭頸部。

### 特殊考量

操作者在拔牙時可能施加相當大的壓力，常導致氣道阻塞的「反向顎推力」。麻醉醫師應施加反壓力以支撐和穩定頭部和氣道。

## 埋伏/埋藏牙齒拔除（Extraction of Impacted/Buried Teeth）

### 手術基本資料

| 項目 | 內容 |
|------|------|
| 手術方式 | 牙齒移除 |
| 手術時間 | 3 至 45 分鐘 |
| 疼痛程度 | + |
| 體位 | 仰臥，頭環，上牙齒拔除時肩膀下墊枕 |
| 失血量 | 微量 |
| 實用技術 | 聲門上氣道和自主換氣，鼻腔管和間歇性正壓換氣（清醒或深度拔管）|

### 術前考量

仔細評估氣道。檢查鼻孔通暢度。

如患者有牙膿腫，可能有明顯的面部腫脹和嚴重的牙關緊閉。可能需要光纖輔助插管。

### 術後護理

以規律 Acetaminophen 和 NSAID 平衡鎮痛。開立需要時的救援鎮痛，如需要時 Tramadol 或 Codeine Phosphate。（Codeine 不再建議用於兒童。）
