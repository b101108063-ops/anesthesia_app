# 血管外科麻醉

*Source: book2.pdf, pages 610-631*

## 22.1 一般原則

血管外科通常涉及對由動脈粥樣硬化引起的動脈疾病進行手術，導致外周血流不暢（缺血）或栓塞。

### 死亡率

| 手術類型 | 死亡率 |
|---------|--------|
| 擇期腹主動脈瘤（AAA）手術 | 2.4% |
| 破裂AAA | 30-40% |

### 患者特點

血管患者通常為老年人，周圍血管疾病患者，常見：

| 疾病 | 發生率 |
|-----|--------|
| 高血壓 | 66% |
| 缺血性心臟病 | 心絞痛、心肌梗塞 |
| 心衰竭 | - |
| 糖尿病 | - |
| COPD | 許多為現或 former吸煙者 |

**常用藥物：** 阿司匹林、β-阻滯劑、利尿劑、心衰竭藥物、胰島素和/或口服降血糖藥

### 特殊考量

| 考量 | 內容 |
|-----|------|
| 抗凝 | 一些患者抗凝；一些患者將在圍手術期接受抗凝 |
| 區域技術 | 仔細考慮區域技術的優缺點（可減少發病率和死亡率） |
| 系列手術 | 血管患者往往有系列手術（可能有多個既往麻醉記錄） |
| 非工作時間 | 顯著少數血管手術在非工作時間進行（通常為高風險患者的高風險程序） |

### 術前評估

| 項目 | 內容 |
|-----|------|
| 雙臂血壓 | 測量雙臂血壓（由於血管疾病可能不同）；使用較高血壓側；动脉線放在此側 |
| 心肺疾病 | 量化任何心肺疾病的程度 |
| 運動耐受力 | 直接詢問運動耐受力（平地行走距離、爬樓梯能力）和仰臥能力 |
| 心衰竭體徵 | 尋找心衰竭體徵 |
| 調查 | FBC、U&E、凝血、LFTs、ECG、CXR |

### 術前用藥

| 藥物 | 建議 |
|-----|------|
| β-阻滯劑 | 繼續 |
| 他汀類 | 繼續 |
| 抗焦慮術前用藥 | 大手術可能有用 |

## 22.3 腹主動脈瘤修復術

| 項目 | 內容 |
|-----|------|
| 程序 | 切除主動脈瘤囊並用合成移植物替換（管狀/褲狀移植物） |
| 時間 | 2-4小時 |
| 疼痛 | +++ |
| 體位 | 仰臥，手臂外展（十字架） |
| 失血 | 500-2000+ mL；X-match 6單位；適合細胞回收 |
| 實用技術 | ETT + IPPV；arterial + CVP線；CO監測；硬膜外/脊髓嗎啡用於術後鎮痛 |

### 術前考量

| 考量 | 內容 |
|-----|------|
| 患者特點 | 老年人常有多种合併症 |
| 30天死亡率 | 擇期開放手術為2.4%（主要為心肌梗塞和多器官衰竭） |
| 心臟評估 | 所有擇期主動脈手術和有任何症狀/新心臟疾病的患者需要心臟功能動態評估；CPET是所有AAA修復患者的「金標準」 |
| 術前優化 | 有些單位進行術前優化（入院至HDU/ICU插入管路等並「優化」血流動力學狀態） |
| 繼續用藥 | 繼續常規心臟藥物 |

### 圍手術期考量

| 考量 | 內容 |
|-----|------|
| 可用藥物 | 血管收縮劑（麻黃鹼和間羥胺）、血管擴張劑（GTN）和β-阻滯劑（拉貝他洛） |
| IV通路 | 兩個14G或更大IV通路 |
| 溫度控制 | 從一開始就 obsessive about溫度控制；使用液體加溫器（理想為快速流量）和強制空氣加溫器 |
| 細胞回收 | 強制使用；有證據表明可減少主動脈手術中同種異體血的使用 |
| 監測 | 动脉線和胸椎硬膜外（T6-T11）誘導前；鉗夾前獲取基線血氣 |
| 多普勒超聲 | 5導聯ECG（導聯II和V）增加檢測心肌缺血敏感性 |
| 誘導 | 溫和誘導配合动脉內血壓監測；使用適中/大劑量阿片類藥物（如remifentanil 0.1-0.2 mcg/kg/min或大劑量芬太尼5-10 mcg/kg） |
| 尿液輸出 | 插入尿導管每小時測量尿液輸出 |
| 肝素 | 鉗夾前通常給予3000-5000單位肝素；可能需要魚精蛋白0.5-1 mg/100單位肝素IV緩慢反轉 |

### 主動脈鉗夾的反應

**鉗夾後近端高血壓：**
- 原因：突然增加SVR、增加SVC流量和交感-腎上腺反應
- 治療：加深麻醉和/或β-阻滯劑bolus（拉貝他洛5-10 mg）、GTN輸注或硬膜外LA

**鉗夾期間：**
- 代謝性酸中毒會發展（由於下肢缺血）
- 維持分鐘通氣量會導致呼吸性鹼中毒（將最小化此代謝性酸中毒的影響）
- 檢查ABG以評估Hct、代謝性酸中毒、呼吸代償和離子Ca²⁺

**鉗夾時間：** 通常60+分鐘

**脫鉗後低血壓：**
- 原因：SVR降低、相對低血容量、由於冷代謝廢物從腿部返回導致心肌「心頓」
- 治療：IV液體和/或減淺麻醉深度和/或小劑量正性肌力藥物（如腎上腺素10 mcg aliquots [1:100,000的1 mL]和/或葡萄糖酸鈣bolus [高達10 mL的10%]）

### 液體置換

| 建議 | 內容 |
|-----|------|
| 晶體/膠體 | 替代不感蒸發、第三間隙和初始失血 |
| 血液製品 | 當識別到缺乏時（如Hct <25%、血小板 <100×10⁹/L） |
| ACT | 如果懷疑凝血病則檢查（正常 <140秒） |
| TEG® | 將提供整個凝血情況 |

### 術後考量

| 考量 | 內容 |
|-----|------|
| ICU/HDU | 術後必要；可能適合其他方面fit、溫暖且血流動力學穩定且有功能硬膜外的患者；否則插管轉送ICU |
| 鎮痛 | 如果無硬膜外則用阿片類輸注和/或PCA |
| 監測 | 術後繼續arterial和CVP監測、遠端脈搏和尿液輸出 |

### 特殊考量

**硬膜外管理：**
- 誘導時硬膜外二醋嗎啡2-3 mg bolus將持續12-24小時
- 直到主動脈關閉才少量使用硬膜外LA
- 有功能的交感神經系統更容易治療主動脈脫鉗的低血壓

**腎衰竭：**
- 發生率1-2%
- 多因素；與AAA修復後50%死亡率相關
- 更可能發生在腎上鉗夾時
- 沒有證據表明多巴胺預防腎衰竭（僅作為正性肌力藥物）
- 一些常規使用甘露醇（鉗夾期間0.5 g/kg）作為自由基清除劑和滲透性利尿劑
- 避免低血容量；每小時監測尿液輸出

## 22.4 急診開放修復腹主動脈瘤

### 定義

| 類型 | 說明 |
|-----|------|
| 急性 | 以心血管崩潰表現；除非破裂被限制在腹膜後空間，否則可能死亡 |
| 分離 | 沿著動脈內膜分離；以背部/腹痛表現 |

### 統計

| 項目 | 數值 |
|-----|------|
| 院前死亡率 | 接近50% |
| 到達醫院後死亡率 | 50% |
| 總死亡率 | 40% |

### 額外考量

| 考量 | 內容 |
|-----|------|
| 診斷確認 | 如果有懷疑且患者血流動力學穩定，通過超聲或CT掃描確認 |
| 復甦 | 如果有低血容量休克，復甦至收縮壓90 mmHg；避免高血壓、咳嗽和用力（可能促使進一步出血） |
| 術前 | 兩14G外周套管和（如可能）动脉線；brachial artery可能必要；有時需要动脉「cut-down」 |
| 誘導 | 在手術室誘導（外科醫生已擦手、手术準備完成、單覆蓋到位、血液在手術室並已核對）；通常需要RSI |
| 誘導劑 | 咪達唑侖/remifentanil、依托咪酯（也給予氫化可的松50-100 mg）和氯胺酮 |
| 溫度 | 熱空氣加溫和至少一個加溫IV輸液（快速流量血液加溫器不可估量） |
| 液體 | 使用膠體或晶體（取決於偏好）；使用平衡晶體如Hartmann's溶液而非0.9%氯化鈉 |
| 血製品 | 細胞回收強制使用；接近失血時如果沒有X-matched血，使用type-specific |
| 拔管 | 不要在手術結束時嘗試拔管——術後通風期對允許糾正生化/血液學異常必不可少 |

## 22.5 腹主動脈瘤的血管內支架植入

| 項目 | 內容 |
|-----|------|
| 程序 | 通過股動脈經介入放射學家放置和展開分叉支架進入主動脈瘤囊 |
| 時間 | 1-4小時 |
| 疼痛 | + |
| 體位 | 仰臥 |
| 失血 | 0-2000+ mL；X-match 6單位 |
| 實用技術 | 脊髓 + 鎮靜；动脉線 |

### 特點

| 項目 | 內容 |
|-----|------|
| 優點 | 與標準開放AAA修復相比，與較低的手術發病率和死亡率相關 |
| 缺點 | 仍未證明可降低動脈瘤破裂風險；因此術後患者必須保持CT監測一生 |
| 併發症 | 支架遷移和內漏等重大併發症可能發生；以及原始瘤囊破裂 |

### 適應症

| 考量 | 內容 |
|-----|------|
| 場所 | 在血管造影套件由介入放射學家進行 |
| 入路 | 許多醫院血管外科醫生通過股動脈進入主動脈；支架由介入放射學家插入 |
| 簡單EVAR | 大多數可經皮完成；因此LA可能合適 |
| 術後 | 通常不需要ICU；患者可去HDU或血管病房過夜監測；住院時間通常僅24-48小時 |
| 複雜EVAR | 定制的fenestrated移植物允許內臟主動脈分支重新連接的手術可能比標準EVAR更長；因此最好在GA下進行 |

### 破裂AAA的EVAR

| 項目 | 內容 |
|-----|------|
| 技術 | 球囊經皮插入股动脉並在破裂上方充盈（ conferring some血流動力學穩定性） |
| 結果 | IMPROVE試驗顯示開放（37.4%）和血管內（35.4%）技術之間30天死亡率相似 |
| 女性 | 與EVAR技術相比女性比男性更好 |
| 費用 | EVAR組的醫院費用略低 |

## 22.6 胸腹主動脈瘤修復術

| 項目 | 內容 |
|-----|------|
| 程序 | 切除延伸至腎動脈起始處以上的主動脈瘤囊並用合成移植物替換；可能需要胸廓切開術和OLV |
| 時間 | 3-6小時 |
| 疼痛 | +++ |
| 體位 | 仰臥，手臂外展；如果胸廓切開術則可能為右側 |
| 失血 | 1000 mL至+++；X-match 8單位加血小板和FFP |
| 實用技術 | DLT + IPPV；arterial + CVP線；胸椎硬膜外 |

### 主動脈瘤類型

| 類型 | 說明 |
|-----|------|
| 升主動脈 | 需要正中胸骨切開術和CPB |
| 主動脈橫弓 | 修復通常也需要深低溫停循環 |

### 特殊考量

| 考量 | 內容 |
|-----|------|
| 氣管 | 瘤可能壓迫氣管並扭曲上血管解剖 |
| ICU | 對術後通氣和穩定必要；直到酸中毒和低體溫糾正且肺部完全重新擴張 |
| 腎上鉗夾 | 將比簡單AAA高得多；這意味著腎臟、肝臟和內臟循環將缺血持續整個鉗夾時間 |
| 左側DLT | 胸主動脈入路可能需要OLV；可能需要左側DLT（見相關頁面） |
| 腎衰竭 | 發生率達25%；主要與鉗夾時間相關 |

### 脊髓保護技術

（沒有一個是萬無一失的）

| 技術 | 說明 |
|-----|------|
| CSF壓力測量和引流 | 通過脊髓引流 |
| 脊髓冷卻 | 通過硬膜外導管 |
| 鞘內Mg²⁺ | - |
| 遠端灌注技術 | - |
| CPB | - |
| DHCA | - |

## 22.7 頸动脉內膜切除術

| 項目 | 內容 |
|-----|------|
| 程序 | 從頸內动脉切除粥樣硬化斑塊；頸內动脉被鉗夾並打開，斑塊切除，动脉直接閉合或用補片閉合 |
| 時間 | 1-3小時 |
| 疼痛 | ++ |
| 體位 | 仰臥，頭高腳低；對側手臂板 |
| 失血 | 最少；G&S |
| 實用技術 | 頸叢阻滯 + 鎮靜；或ETT + IPPV；动脉線 |

### 適應症

減少有症狀（TIA或腦血管意外）且頸內动脉狹窄>70%的患者的CVE發生率。

### 結果

圍手術期死亡率和腦血管意外發生率：1-5%

### 監測

| 方法 | 說明 |
|-----|------|
| 區域麻醉倡導者 | 優勢包括有清醒患者（可立即檢測神經缺陷） |
| GA下 | 使用其他技術監測腦灌注：EEG處理、軀體感覺誘發電位監測、經顱多普勒超聲、腦中動脉近紅外光譜 |

### 爭論

| 觀點 | 內容 |
|-----|------|
| GA vs 區域 | 迄今最大RCT未能顯示兩種麻醉技術之間結局有任何差異 |
| 團隊合作 | 可能是減少手術風險的最重要因素 |

### 術前考量

| 考量 | 內容 |
|-----|------|
| 患者特點 | 老年血管疾病患者；大多數高血壓 |
| 血壓控制 | CEA期間血壓控制可能困難 |
| 血壓管理 | 從病房圖表確定正常血壓範圍；測量雙臂血壓；使用兩者中較高的；瞄準160/90 mmHg |
| 神經缺陷 | 記錄現有神經缺陷以便更容易檢測新的缺陷 |

### 全身麻醉用於頸动脉內膜切除術

| 項目 | 內容 |
|-----|------|
| 誘導 | 溫和IV誘導；血壓可能在誘導和插管期間不穩定；給予充足短效阿片類藥物；考慮噴霧cord用利多卡因 |
| LMA | 大多數麻醉師使用ETT；LMA cuff被顯示減少頸动脉血流（但意義不明） |
| 維持 | Remifentanil輸注配合淺表頸叢阻滯提供理想條件和快速甦醒；否則isoflurane/阿片類技術；維持正常碳酸血症；避免N₂O |
| 拔管 | 在發生過度咳嗽前拔管；在恢復室進行密切神經監測直至完全清醒 |

### 清醒頸动脉

| 項目 | 內容 |
|-----|------|
| 準備 | 患者準備和溝通至關重要 |
| 頸叢阻滯 | 頸椎皮膚C2-C4可通過超聲或解剖方法被阻滯（深、中或淺表頸叢阻滯） |
| 深度阻滯 | 避免深度阻滯（有呼吸損害的患者可能不能耐受單側膈神經麻痹） |
| 鎮靜 | 阻滯放置和解剖期間可小心使用（如remifentanil 0.05-0.1 mcg/kg/min） |
| 監測 | 鉗夾期間監測患者的言語、認知和對側運動力量 |
| 神經缺陷表現 | 三種形式：鉗夾時深度無意識；鉗夾後微妙但立即的缺陷（混亂、構音障礙、回答問題延遲）；延遲缺陷（通常與相對低血壓相關） |

### 術後考量

| 考量 | 內容 |
|-----|------|
| 恢復觀察 | 強制在恢復室觀察2-4小時 |
| 氣道水腫 | 常見（GA和區域病例都是由於氣道周圍解剖）；宮頸血腫發生率5-10% |
| 血腫處理 | 在恢復室拆除皮膚縫合以允許引流；進行性氣道阻塞需要立即再探查 |
| 血流動力學不穩定 | 常見；高灌注綜合徵典型表現為頭疼（可能最終導致出血性CVE） |
| 血壓控制 | 必須嚴格控制；給予工作人員關於血流動力學管理的仔細書面說明 |

## 22.8 周圍血運重建

| 項目 | 內容 |
|-----|------|
| 程序 | 用於閉塞性动脉疾病的繞道手術；使用靜脈或合成移植物繞過閉塞动脉 |
| 時間 | 1-6小時 |
| 疼痛 | +++ |
| 體位 | 仰臥 |
| 失血 | 通常500-1000 mL；X-match 2單位 |
| 實用技術 | 聯合脊髓/硬膜外配合鎮靜；arterial線；ETT/IPPV；考慮LMA |

### 手術類型

| 類型 | 說明 |
|-----|------|
| 股-膕繞道 | 股至膝上膕动脉 |
| 股-遠端繞道 | 股至前或後脛动脉 |
| 股-股交叉繞道 | 從一股动脉到另一股 |

### 術前考量

| 考量 | 內容 |
|-----|------|
| 手術比例 | 構成擇期血管手術的很大比例 |
| 手術時間 | 不可預測且手術延長並不少見 |
| CVS耐受 | 通常比主動脈手術更能耐受 |

### 區域麻醉優勢

- 提供的手術條件和術後鎮痛良好
- 單次脊髓阻滯（plain bupivacaine）和鞘內可樂定（15-30 mcg）和二醋嗎啡（250-500 mcg）可提供長達6小時的良好手術麻醉

---

## 22.9 腋-雙股繞道

| 項目 | 內容 |
|-----|------|
| 程序 | 從腋动脉到股动脉的腹膜外繞道（褲狀移植物） |
| 時間 | 2-4小時 |
| 疼痛 | +++ |
| 體位 | 仰臥 |
| 失血 | <1000 mL；X-match 2單位 |
| 實用技術 | GA— ETT、IPPV、arterial line；考慮CVC |

### 特點

| 項目 | 內容 |
|-----|------|
| 現況 | 由於支架技術的快速進展現在較少進行 |
| 適應症 | 完全閉塞的主髂血管患者的最後機會手術 |
| 腹膜外 | 是腹膜外手術；因此有嚴重心肺疾病可能不能耐受主動脈手術的患者可能更能耐受 |

## 22.10 截肢

| 項目 | 內容 |
|-----|------|
| 程序 | 由於血管缺血去除壞死或感染組織 |
| 時間 | 30-120分鐘 |
| 疼痛 | +++ |
| 體位 | 仰臥 |
| 失血 | 通常200-500 mL；G&S |
| 實用技術 | 脊髓或硬膜外配合鎮靜；坐骨/股神經阻滯 ± GA |

### 術前考量

| 考量 | 內容 |
|-----|------|
| 患者特點 | 通常為有病、臥床不起的糖尿病患者，有顯著心血管疾病且先前有過多次血運重建嘗試 |
| 不適 | 許多術前有相當大的不適（糖尿病患者較少）；可能在大劑量腸道或腸外阿片類藥物上 |

### 區域麻醉優勢

| 優勢 | 內容 |
|-----|------|
| 單側 | 可定向單側 |
| 術後鎮痛 | 持續時間可通過鞘內二醋嗎啡（250-500 mcg）或鞘內可樂定（15-30 mcg）延長 |

### 幻肢痛

| 項目 | 內容 |
|-----|------|
| 發生率 | 60-70%截肢者在某時有此問題 |
| 處理 | 必須與手術疼痛區分；可能時獲得疼痛團隊投入 |
| 預防 | 術前硬膜外（預防性鎮痛）相信可減少慢性疼痛的發生率和嚴重程度 |

## 22.11 胸腔鏡交感神經切除術

| 項目 | 內容 |
|-----|------|
| 程序 | 用於手掌/腋窩多汗患者；通過小腋窩切口插入胸腔鏡切斷交感神經幹 |
| 時間 | 30-60分鐘 |
| 疼痛 | ++ |
| 體位 | 仰臥，受影響手臂在手臂板上 |
| 失血 | 最少 |
| 實用技術 | IPPV via DLT；SV via LMA |

### 技術

| 技術 | 說明 |
|-----|------|
| 傳統OLV | 患者在反向Trendelenburg位；用單肺麻醉（DLT） |
| 更簡單技術 | 患者通過LMA自發呼吸；外科醫生向胸膜腔注入CO₂時，肺被動推送開；這種技術比OLV產生的分流程度較不戲劇 |

### 特殊考量

| 考量 | 內容 |
|-----|------|
| 手術結束 | 必須重新擴張肺（在外直接視覺下）以防止氣胸 |
| 同步雙側 | 可能導致第二肺塌陷時嚴重缺氧（由於第一肺持續肺不張） |
| 術後 | 需要術後CXR確認肺重新充氣 |

---

## 22.12 第一肋骨切除術

| 項目 | 內容 |
|-----|------|
| 程序 | 在胸廓出口綜合徵患者切除第一/頸肋 |
| 時間 | 1-2小時 |
| 疼痛 | ++ |
| 體位 | 仰臥，受影響手臂在手臂板上 |
| 失血 | 最少 |
| 實用技術 | IPPV via ETT；避免肌肉鬆弛劑 |

### 特殊考量

| 考量 | 內容 |
|-----|------|
| 患者特點 | 通常為年輕fit患者 |
| 肌肉鬆弛劑 | 應避免（外科醫生需要能夠識別臂叢神經） |
| 術後 | 傷口充滿0.9%氯化鈉並手動通氣，充盈壓力>40 cmH₂O（這是為了檢查肺洩漏並排除胸膜損傷） |

## 22.13 動靜脈瘺形成

| 項目 | 內容 |
|-----|------|
| 程序 | 創建A-V瘺通常在上肢以便利血液透析 |
| 時間 | 1-2小時 |
| 疼痛 | + |
| 體位 | 仰臥，手臂在側台 |
| 失血 | 最少 |
| 實用技術 | 臂叢阻滯；IPPV via ETT |

### 特殊考量

| 考量 | 內容 |
|-----|------|
| 腎衰竭 | 定義上這些患者有腎衰竭；可能有高血壓、糖尿病、流體不平衡、先前移植、失敗的瘺、尿毒症、凝血病、高/低鉀血症、貧血或慢性酸中毒 |
| 透析線 | 可能已有靜脈-靜脈透析線在位；避免使用此進行IV通路（除非沒有替代方案） |
| 最後透析 | 知道患者上次透析時間；檢查K⁺和Hb；考慮容量超負荷或低血容量；謹慎給予IV液體（不應含有K⁺） |
| 避免靜脈 | 避免在將來可能需要A-V瘺的前臂或肘窩的大靜脈扎管；盡可能使用手背靜脈 |

## 22.14 靜脈曲張手術

| 項目 | 內容 |
|-----|------|
| 程序 | 去除下肢曲張靜脈：高紮和剝離——長隱靜脈去除（有時雙側）；短隱靜脈手術——在膕窩綁扎 |
| 時間 | 30分鐘至3小時 |
| 疼痛 | ++ |
| 體位 | 仰臥或俯臥（短隱靜脈手術） |
| 失血 | 高達1000 mL |
| 實用技術 | LMA/SV（大多方面）；ETT/IPPV（俯臥） |

### 特點

| 項目 | 內容 |
|-----|------|
| 趨勢 | 由於靜脈內激光消融或硬化劑注射等技術的引入和普及，接受靜脈曲張手術的患者數量急劇下降 |
| 手術 | 主要手術通常結合多次靜脈切除以去除曲張靜脈（微小疤痕但可能出血嚴重） |
| 失血最小化 | 術中抬高腿部 |
| 鎮痛 | NSAIDs和LA注入腹股溝傷口的組合提供良好的術後鎮痛 |
