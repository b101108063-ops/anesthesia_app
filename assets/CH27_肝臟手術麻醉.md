## 27.1 急性食道出血的麻醉管理

急性靜脈曲張出血是一種醫療急症。上消化道出血的總死亡率為2-10%。

### 治療原則

有效的復甦和明確治療應同時進行。

### 復甦

| 步驟 | 內容 |
|-----|------|
| 評估 | 快速評估氣道、呼吸和循環狀態 |
| 氧疗 | 補充O₂根據需要 |
| 氣道控制 | 早期氣道控制（如果指示）；平衡氣道丟失風險與RSI的血流動力學後果 |
| IV通路 | 插入兩個大孔徑IV套管；考慮CVP和动脉線 |
| 液體 | 初始液體復甦使用晶體；平衡避免低血壓和避免加重出血 |
| 氨甲環酸 | 早期使用（降低死亡率） |
| 實驗室 | 緊急FBC、凝血、即時凝血測試、U&E、LFTs、ECG |
| 導尿 | 如果不穩定 |

### 輸血管理

| 策略 | 內容 |
|-----|------|
| 方案 | 大量出血方案 |
| PRBCs | 限制性策略——如果有出血且血流動力學不穩定或Hb <70-80 g/L則輸血 |
| FFP | 血流動力學不穩定和凝血病（INR >正常1.5倍或纖維蛋白原水平<1.5 g/L） |
| 血小板 | 活躍出血且血流動力學不穩定且血小板計數<50×10⁹/L |
| Warfarin逆轉 | 用PCC逆轉 |

### 止血

| 方法 | 內容 |
|-----|------|
| 藥物治療 | 血栓抑制素、生長抑素或生長抑素類似物 |
| 抗生素 | 48小時預防性抗生素治療 |
| 內鏡治療 | 靜脈曲張的帶結紮或硬化劑注射 |

### 搶救干預

| 干預 | 內容 |
|-----|------|
| 介入放射學 | TIPS是治療選擇 |
| 氣囊壓迫 | 僅在內鏡和藥物治療失敗時使用；有高風險致命並發症（誤吸、食道撕裂/破裂和氣道阻塞） |

## 27.2 經頸靜脈肝內門體分流術（TIPS）的麻醉

| 項目 | 內容 |
|-----|------|
| 程序 | 門靜脈和全身循環之間的經皮血管連接建立，降低門靜脈壓力 |
| 適應症 | 門脈高壓及其並發症的治療（難治性靜脈曲張出血和利尿劑抵抗性腹水） |
| 技術 | 放射學家在肝靜脈和門靜脈之間放置支架 |

### 術前考量

| 考量 | 內容 |
|-----|------|
| 評估 | 肝功能損害、血流動力學穩定性、誤吸風險、心衰竭和腦病 |
| 禁忌症 | 嚴重肺高壓、嚴重三尖瓣返流或嚴重心衰竭 |

### 麻醉執行

| 考量 | 內容 |
|-----|------|
| 場所 | 通常在血管造影套件進行 |
| 抗生素 | 術前預防性廣譜抗生素並持續24小時 |
| 技術 | 清醒鎮靜或GA；如果患者高風險，GA配合RSI和ETT更安全 |
| 監測 | 良好的IV通路和侵入性动脉線監測 |

### 術中並發症

| 並發症 | 內容 |
|--------|------|
| 頸內靜脈穿刺 | 相關並發症 |
| 肝动脉穿刺 | 可能大量出血 |
| 急性心衰竭 | 分流導致靜脈回流和前負荷增加 |
| 敗血症 | 術後可能 |
| 黃疸或腦病惡化 | 20%患者 |

## 27.3 肝移植

| 項目 | 內容 |
|-----|------|
| 程序 | 整肝移植 |
| 時間 | 4-10小時 |
| 疼痛 | 可變（比其他可比程序如胃切除術、胸廓切開術少）；PCA；避免NSAIDs |
| 體位 | 仰臥，一或兩臂外展 |
| 失血 | 極可变；0-10,000 mL；X-match 6單位並考慮解凍其他成分 |
| 實用技術 | ETT、IPPV |

### 適應症

| 適應症 | 說明 |
|--------|------|
| 終末期肝病 | 主要適應症 |
| 全球最常見 | 丙型肝炎後肝硬化 |
| 英國最常見 | 酒精相關肝硬化和肝癌 |
| 其他 | 原發性硬化性膽管炎、病毒性肝炎（B、C）、多囊肝病、原發性膽汁性肝硬化、代謝性肝病 |

### 終末期肝病生理學挑戰

| 挑戰 | 內容 |
|-----|------|
| 黃疸 | - |
| 低鈉血症 | - |
| 腹水 | - |
| 胸腔積液 | - |
| 腎衰竭 | 肝病繼發（肝腎綜合徵） |
| 肝肺綜合徵 | - |
| 門脈肺高壓綜合徵 | 相關嚴重門脈和肺高壓，導致右心衰竭和潛在心臟驟停 |
| 凝血病 | PT延長、血小板減少、纖溶 |
| 靜脈曲張 | 食道、胃、直腸、腹壁 |
| 腦病 | - |
| 全身血管擴張 | 低血壓和心衰竭 |

### 移植階段

#### 前期/解剖階段

| 階段 | 內容 |
|-----|------|
| 切口 | 雙側肋下（Mercedes）/反向L肋下 |
| 解剖 | 暴露肝臟、門靜脈和周圍結構 |
| 出血 | 常見（靜脈為主） |

#### 無肝期

| 階段 | 內容 |
|-----|------|
| 門靜脈和肝靜脈 | 分離 |
| 植入 | 新肝插入 |
| 技術1 | 腔靜脈保留（「 piggy back」技術） |
| 技術2 | 經典型（需要腔靜脈鉗夾） |

#### 新肝期

| 階段 | 內容 |
|-----|------|
| 再灌注 | 通過肝臟的血流恢復 |
| 再灌注綜合徵 | 細胞因子釋放、補體激活、核心溫度暫時降低 |
| K⁺升高 | 洗出先前缺血移植物；心律失常常見，心臟驟停偶爾發生 |
| 低鈣血症 | 乳酸性酸中毒、凝血病和低血糖 |

### 術後考量

| 考量 | 內容 |
|-----|------|
| ICU | 應在ICU管理 |
| 早期拔管 | 通常可行並有利於移植物血流 |
| 鎮痛 | PCA/硬膜外/椎旁阻滯；但區域技術因凝血病在許多中心不鼓勵 |
| 液體 | 維持液體/NG餵養1.5 mL/kg/h |
| 移植物功能 | 原發無功能/初始功能障礙可達5-10% |

## 27.4 肝切除術

| 項目 | 內容 |
|-----|------|
| 程序 | 肝組織切除 |
| 時間 | 2-6小時 |
| 疼痛 | 與移植相同；硬膜外更常見 |
| 體位 | 仰臥，雙臂外展，反向Trendelenburg位 |
| 失血 | 最少至900 mL；X-match 5-10單位 |
| 實用技術 | ETT、IPPV |

### 主要適應症

- 轉移性結直腸腺癌（改善5年生存率30%）
- 良性/惡性原發肝膽腫瘤
- 肝移植的供肝切除
- 肝創傷

### 麻醉考量

| 考量 | 內容 |
|-----|------|
| 肝切除範圍 | 通常移除30-75%功能性肝組織 |
| 代謝功能 | 殘留肝細胞功能差數天；應使用短效藥物 |
| 腦病風險 | 避免可能加重腦病的藥物（如苯二氮卓類） |
| 失血 | 微創技術減少失血和輸血需求 |

### 液體和血流動力學管理

| 目標 | 內容 |
|-----|------|
| CVP | 維持低CVP（<5 cmH₂O）大幅減少出血 |
| 技術 | 預切除液體限制、硬膜外bolus、頭高腳低位、硝酸鹽或利尿劑輸注 |
| 止血帶手法 | 可能需要間歇性門靜脈鉗夾 |

### 術後考量

| 考量 | 內容 |
|-----|------|
| 護理單位 | >50%切除應初始在HDU管理 |
| 肝功能不全 | 凝血病或腦病可能在非常大切除或高風險患者中發展 |
| 腎功能障礙 | 廣泛切除或高風險患者更常見 |
| 結果 | 根治性肝切除的結果非常鼓舞人心 |
