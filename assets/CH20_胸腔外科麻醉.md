## 20.1 一般原則

成功的胸腔麻醉需要能夠獨立控制患者兩個肺的通氣、管理共享肺和氣道，並清楚了解計劃的手術。

### 患者特點

長期吸煙、支氣管癌、胸腔積液、心臟疾病、食道阻塞和惡病質都很常見。這些因素會顯著降低心肺生理儲備。

### 一般考量

| 項目 | 內容 |
|-----|------|
| 術前討論 | 與外科醫生討論計劃的手術和潛在問題 |
| 戒菸 | 優化呼吸功能和心血管 fitness 以減少圍手術期發病率 |
| ERAS | 在轉介時將患者納入ERAS途徑 |
| 體位 | 大多數手術使用側臥位，手術台「斷裂」以分開肋骨 |
| 微創手術 | 越來越多的病例通過視頻輔助胸腔鏡手術（VATS）完成 |
| 術後通氣 | 應避免（對肺縫合線造成壓力，增加空氣洩漏和胸部感染風險） |
| 術後氧疗 | 常規補充氧氣以補償V/Q不匹配 |
| 肺部手術後 | 建議通過面罩給予加熱加濕的40% O₂ |

### 術前評估考量

| 項目 | 內容 |
|-----|------|
| 功能狀態 | 注意功能狀態和心肺儲備 |
| 心臟病 | 有顯著心臟病的患者形成高風險組 |
| 影像討論 | 與外科醫生討論CXR和CT掃描，專注於使DLT放置有問題的氣道問題、侵犯胸壁的腫瘤和穿過裂隙或血管的腫瘤 |

### 肺切除術前評估

**風險評估基於：**
- 病史
- 檢查和功能狀態評估
- 肺功能檢查（PFTs）
- 彌散能力測試（如DLCO）
- Thoracoscore計算

**Thoracoscore：**
- 基於多個變量提供死亡風險估計
- 包括人口學特徵、呼吸困難程度、合併症、performance status和手術類型
- 在知情同意過程中特別有用

**患者分類：**

| 分類 | 標準 |
|-----|------|
| 臨床上 fit | 良好的運動耐受力，正常肺功能：可接受手術 |
| 主要醫學問題 | 極小的運動能力，嚴重受損的PFTs：考慮替代治療 |
| 降低的運動能力 | 爬兩層樓梯時呼吸急促，異常肺功能±中等合併症：進一步評估和仔細評估手術風險/益處 |

### 肺功能檢查（PFTs）

| 項目 | 內容 |
|-----|------|
| 用途 | 通過估計術後肺功能確定肺切除手術的適應性 |
| 建議 | 除了彌散能力測試外還應進行肺量測定法 |
| 重點 | 彌漫性肺泡疾病患者可能具有相對正常的肺量測定法但嚴重受損的氣體傳遞 |

**預測術後（PPO）值計算：**
- PPO = 術前值 × (19 - 切除的節段數) / 19

**CPET（心肺運動測試）：**
- 如果術前DLCO或PPOFEV₁ <40%預測正常，患者應在手術前進行CPET
- CPET評估VO₂max，用於 inform圍手術期發病率和死亡率的風險
- VO₂max 10-15 mL/kg/min是較高風險，應入住HDU
- VO₂max <10 mL/kg/min是非常高風險，手術可能不合適

**所有接受肺葉切除術或雙葉切除術的患者應有：**
- CPET
- 超聲心動圖
- 術後HDU入院

## 20.2 鎮痛

### 術後疼痛的重要性

| 項目 | 內容 |
|-----|------|
| 慢性疼痛綜合徵 | 胸腔手術後發生率25-60% |
| 風險因素 | 高強度術後疼痛會增加風險 |
| 鎮痛不足影響 | 增加神經體液應激反應，損害活動和呼吸，增加併發症 |

### 多模式鎮痛建議

**藥物組合：**
- 對乙醯氨基酚
- NSAIDs
- 可樂定
- 鎂
- 氯胺酮
- 糖皮質激素
- 區域阻滯

### 區域麻醉技術

| 技術 | 說明 |
|-----|------|
| 內部肋間神經阻滯 | 外科醫生進行 |
| 椎旁導管 | 用於開放和視頻輔助胸廓切開術（與硬膜外等效的鎮痛效果，但不良事件較少） |

**椎旁導管建議：**
- 濃度：Levobupivacaine 0.375%
- 持續時間：48-72小時
- 速率：0.1 mL/kg/h

**其他技術：**
- 胸椎旁神經阻滯適用於雙側手術
-  serratus前平面或豎脊肌平面的神經周圍導管可用於單端口VATS
- 也可用於肋骨骨折的良好鎮痛

## 20.3 加速康復外科（ERAS）

ERAS途徑在轉介時啟動，隨患者至出院。當組合交付時，可改善患者結局。

### 術前建議

| 建議 | 內容 |
|-----|------|
| 患者參與 | 患者教育、心理準備對提高康復計劃的動機和合規性至關重要 |
| 營養評估 | 營養狀況和體重減輕評估；為營養不良或風險患者開處方口服營養補充 |
| 戒菸戒酒 | 術前4週減少吸菸和酒精消耗可減少肺部併發症和死亡風險 |
| 貧血管理 | 識別貧血患者並糾正鐵缺乏 |
| 預康復計劃 | 邊緣肺功能或差運動能力的患者可改善生理儲備 |
| 禁食 | 避免飢餓，考慮碳水化合物負荷 |
| 術前用藥 | 避免術前鎮靜藥物 |

### 圍手術期建議

| 建議 | 內容 |
|-----|------|
| 抗生素 | 適當的抗生素預防 |
| 溫度監測 | 強制，避免低體溫 |
| 麻醉技術 | 區域和全身麻醉技術組合允許早期拔管 |
| 肺保護性通氣 | 應採用 |
| PONV | 避免；高風險患者應給予多模式止吐預防 |
| 液體管理 | 瞄準正常血容量（乾燥肺）；使用平衡晶體液；避免過度限制（2-3 mL/kg/h未被證明有害） |
| 微創手術 | 首選；可能時避免胸管外部抽吸 |
| 術後氧疗 | 加熱加濕40% O₂補償肺手術後V/Q不匹配；其他患者鼻導管O₂ 3 L/min更能耐受 |
| 早期活動 | 物理治療、激勵肺活量計、機械和藥理學VTE預防 |
| 心房顫動 | 術後新發AF或心房撲動常見，發生率12%；繼續β-阻滯劑；更換鎂；考慮術前地爾硫卓或術後胺碘酮 |

## 20.5 肺隔離技術

獨立通氣兩個肺並不總是簡單的。OLV與併發症相關，僅在益處超過風險時使用。

### OLV適應症

**絕對適應症：**

| 適應症 | 說明 |
|--------|------|
| 避免污染 | 感染、大量肺出血或支氣管肺泡灌洗時隔離一個肺 |
| 控制通氣分佈 | 大量氣胸或嚴重單側肺疾病（如巨大肺大泡和肺囊腫） |
| VATS程序 | 微創手術需要 |

**相對適應症：**
- 改善手術入路
- 如果隔離困難，與外科醫生溝通（肺牽拉可能是替代方案）

### 雙腔氣管內管（DLTs）

**特點：**

| 特點 | 說明 |
|-----|------|
| 描述 | 最常見和最多功能的途徑 |
| 類型 | 根據設計插入的主支氣管分為「右」或「左」 |
| 右側管 | 在支氣管部分壁上有孔（Murphy's eye）以便利右肺上葉通氣 |
| 尺寸 | 以Charrière（Ch） gauge（等同於French gauge）描述，這是管的外部周長（mm） |

**Broncho-Cath® DLT示例：**

| 尺寸 | 支氣管部分ID |
|-----|-------------|
| 39 Ch | 6.0 mm |
| 35 Ch | 4.5 mm |

**放置深度：**
- 平均深度約29 cm（為身高170-180 cm的患者）
- 身高每偏離10 cm，深度變化1 cm

### DLT類型

| 類型 | 說明 |
|-----|------|
| Carlens'（左側） | 有嶺狀「鉤」 aid correct placement |
| White's（右側） | 有嶺狀鉤和管壁狹縫 |
| Robertshaw（左右側） | D形管腔；傳統為紅橡膠，現在有一次性使用的小、中、大尺寸 |
| 一次性PVC（左右側） | 高容量低壓氣囊；可用尺寸28-41 Ch |

### 右側 vs 左側DLT

| 考量 | 左側DLT | 右側DLT |
|-----|---------|---------|
| 放置難度 | 較容易 | 較困難（Murphy's eye需要對準右肺上葉支氣管） |
| 使用頻率 | 更常用 | 較少使用 |
| 需要FOB檢查 | 建議 | 強制 |
| 應用時機 | 左主支氣管手術、左主支氣管阻塞或短左主支氣管時需要右側 | 通常用於左側DLT |

### DLT放置技術

| 步驟 | 內容 |
|-----|------|
| 檢查 | 使用前檢查DLT，包括兩個氣囊和所有連接是否合適 |
| 插管開始 | DLconvexity面向前方通過聲門 |
| 旋轉 | 超過聲門後，退出導芯並將管旋轉90-180°帶來口咽曲線進入矢狀面 |
| 前進 | 輕輕前進，同時將DLT帶回中立位置 |
| 深度 | 前進至約29 cm |
| 確認 | 將DLT作為普通ETT處理；充盈氣管氣囊；檢查ETCO₂並確認放置在兩個肺 |

### DLT位置確認（左側）

| 步驟 | 內容 |
|-----|------|
| 夾鉗 | 夾鉗在Y-連接器上的氣管腔並打開端口感受洩漏 |
| 觀察 | 看胸部運動並檢查是否有適當的單側擴張 |
| 充盈氣囊 | 充盈支氣管氣囊直到洩漏消失 |
| 聽診 | 在您希望隔離的側邊進行聽診；放置夾鉗並充盈支氣管氣囊；如果正確，胸部聲音應消失 |
| 確認OLV | 確認可以通過夾鉗對側實現對側（手術側）肺的隔離和OLV |

### 光纖支氣管鏡檢查

| 建議 | 內容 |
|-----|------|
| 理想做法 | 每個DLT位置都應進行支氣管鏡檢查 |
| 最低要求 | 必須立即可用合適的支氣管鏡以評估DLT放置 |
| 右側管 | 應始終用支氣管鏡檢查以確保Murphy's eye位置正確 |
| 問題發生率 | 多項支氣管鏡研究顯示高達80%的DLTs在某种程度上位置不良 |

### 支氣管阻斷技術

**適應症：**
- 困難插管的患者
- 氣管支氣管解剖扭曲/氣管切開術
- 需要單個支氣管隔離（局灶性支氣管擴張或出血、肺膿腫、支氣管胸膜瘻、既往肺切除）

**技術：**
- 球囊尖端導管（阻斷器）通過SLT操縱進入適當的主（或肺葉）支氣管
- 使用窄光纖支氣管鏡輔助

**位置確認：**
- 患者的手術體位後重新檢查阻斷器位置
- 在側臥位時放置可能尷尬

**肺塌陷：**
- 通過在支氣管內充盈球囊從通氣隔離
- 肺慢慢塌陷（被困氣體被吸收或通過阻斷器的窄中心腔逃逸）
- 可通過先用100% O₂通氣几分鐘然後在呼氣末充盈阻斷器加速

## 20.6 單肺通氣管理

### V/Q不匹配

當患者接受OLV時會產生V/Q不匹配，導致分流。患者傾向於被側臥，允許縱隔和腹部內容物的重量減少FRC。

### 啟動OLV

| 步驟 | 內容 |
|-----|------|
| 開始 | 使用與雙肺通氣時相同的典型呼吸機設置 |
| 增加FiO₂ | 在啟動OLV之前增加至0.5-1.0 |
| 夾鉗 | 夾鉗Y-連接器至手術（非依賴）肺，打開該DLT腔上的密封蓋讓氣體逃逸 |
| 確認 | 如果正確隔離，V_T應降至未隔離時約三分之二 |
| 檢查 | 如果P_AW過高（>35 cmH₂O）或突然上升，排除機械原因和DLT位置不良或阻塞 |

**保護性通氣策略：**

| 參數 | 目標 |
|-----|------|
| P_AW | ≤35 cmH₂O，理想地≤30 cmH₂O |
| PEEP | 5-10 cmH₂O |
| 策略好處 | 減少ALI發生率 |

### OL V期間缺氧

**發生時機：**
- 通常在OLV開始后几分鐘發生
- （當非通氣肺中的O₂被吸收時）

**處理：通氣肺**

| 行動 | 內容 |
|-----|------|
| 增加FiO₂ | 增加到0.5-1.0並手動通氣100% O₂ |
| 增加分鐘通氣量 | - |
| 纖維支氣管鏡檢查 | 檢查管位置確保管沒有移動以阻塞上葉 |
| 吸引 | 清理任何分泌物 |
| 液體和血管加壓素 | 維持灌注 |
| PEEP | 應用到通氣肺，但注意這可能惡化分流 |

**處理：非通氣肺**

| 行動 | 內容 |
|-----|------|
| 吹入O₂ | 通過吸引導管吹入 |
| CPAP | 通過單獨回路應用到非通氣肺 |
| 夾鉗PA | 但這僅在肺葉切除術接近完成時真正可行 |

### 返回雙肺通氣

| 步驟 | 內容 |
|-----|------|
| 吸引 | 輕輕吸引非通氣肺以清理任何血液或膿——使用DLT提供的長吸引導管 |
| 關閉 | 關閉非通氣肺腔上的密封蓋並移除Y-連接器上的夾鉗 |
| 重新通氣 | 切換到手動通氣並在直接視覺下重新充盈塌陷的肺 |
| 充盈壓力 | 通常需要高達30 cmH₂O的充盈壓力以完全重新擴張所有肺部區域 |
| 返回機械通氣 | 除非顯著肺容積被切除，返回原始雙肺呼吸機設置和FiO₂ |
| 調整RR | 維持正常碳酸血症 |

## 20.7 硬性支氣管鏡和支架植入

### 程序概述

| 項目 | 內容 |
|-----|------|
| 程序 | 氣管支氣管樹的內鏡檢查±活檢、支架、異物取出 |
| 時間 | 10-20分鐘 |
| 疼痛 | + |
| 體位 | 仰臥，頭頸伸展 |
| 失血 | 通常最少 |
| 麻醉技術 | TCI propofol、alfentanil/remifentanil、rocuronium；高風險病例用sugammadex；通過支氣管鏡IPPV，O₂ via Venturi needle和Sanders injector；hFnO用於呼吸暫停氧合 |

### 術前考量

| 考量 | 內容 |
|-----|------|
| 氣道阻塞 | 檢查喘鳴、CT掃描上的氣管腫瘤或異物吸入史 |
| 門診患者 | 適當患者可作為門診程序 |
| 警告 | 告知術後咳嗽、咳血和喉嚨痛 |
| 聯合程序 | 可與縱隔鏡檢查結合以評估切除適應性 |
| 反流風險 | 有反流風險的患者可能需要術前用omeprazole或ranitidine |

### 圍手術期考量

| 考量 | 內容 |
|-----|------|
| 預充氧 | 充分預充氧並檢查Jet呼吸機在麻醉患者前正常工作 |
| 協調 | 與外科活動協調低頻率Jet通氣——外科醫生會說何時不通氣 |
| 地塞米松 | 可給予以減少氣道肿胀 |

### 術後考量

| 考量 | 內容 |
|-----|------|
| 體位 | 清醒後立即坐直 |
| 血塊 | 血塊可導致嚴重下氣道阻塞，需要立即插管、吸引和重複支氣管鏡 |

### 特殊考量

| 考量 | 內容 |
|-----|------|
| 刺激程序 | 可產生顯著高血壓反應 |
| CVS反應 | 需要抑制，提供深刻鬆弛但要有及時的喉反射和自發呼吸恢復 |
| 活檢風險 | 活檢可能引起危及生命的氣道出血 |
| 支架技術 | 可能困難，可能涉及週期性失去氣道控制 |

## 20.8 縱隔鏡檢查和超聲支氣管鏡

### 上/頸部縱隔鏡檢查

| 項目 | 內容 |
|-----|------|
| 程序 | 通過胸骨上或前肋間切口對上、前縱隔的腫瘤和淋巴結進行檢查和活檢 |
| 時間 | 20-30分鐘 |
| 疼痛 | + |
| 體位 | 仰臥或略微頭部抬高，手臂在兩側，頭環和肩下墊 |
| 失血 | 通常最少，但有大量出血可能性；G&S |
| 麻醉技術 | 通過SLT的IPPV；LA± propofol TCI用於超聲支氣管鏡 |

### 術前考量

| 考量 | 內容 |
|-----|------|
| 門診 | 適當患者可作為門診程序 |
| 檢查 | 檢查SVC阻塞和氣管偏移或大縱隔腫塊壓迫 |
| 聯合程序 | 有時在剛性支氣管鏡（"Bronch & Med"）之後進行 |

### 圍手術期考量

| 考量 | 內容 |
|-----|------|
| 氣道安全 | 確保氣道安全（頭部會被單覆蓋） |
| 鎮痛 | 手術期間給予IV芬太尼bolus |
| 靜脈通路 | 誘導後在下肢靜脈插入16-G套管 |
| 監測 | 注意外科對氣管的手術壓迫（監測V_T和P_AW） |

### 特殊考量

| 考量 | 內容 |
|-----|------|
| 出血風險 | 大血管有大量出血的可能性；SVC阻塞患者風險增加 |
| 其他風險 | 氣胸、喉返神經損傷、VAE |
| 需要 | 可能需要緊急胸骨切開術 |

### 超聲支氣管鏡（EBUS）

| 項目 | 內容 |
|-----|------|
| 用途 | 肺癌縱隔分期的替代方法 |
| 麻醉 | 可用LA±鎮靜或GA（TIVA） |
| 技術實質 | 通過LMA或單腔ETT進行的柔性支氣管鏡 |
| 準備 | 為通氣問題做準備；可能需要相對高流量手動通氣 |

## 20.9 肺手術：楔形切除、肺葉切除和全肺切除

### 程序概述

| 項目 | 內容 |
|-----|------|
| 程序 | 選擇性切除（楔形切除或肺葉切除）或全肺（全肺切除） |
| 時間 | 2-4小時 |
| 疼痛 | +++++ |
| 體位 | 側臥位，台「斷裂」，肘部彎曲使前臂與面部平行 |
| 失血 | 100-500 mL；偶爾顯著更多；G&S |
| 麻醉技術 | 通過DLT的IPPV使用OLV；椎旁區域麻醉帶導管用於術後鎮痛、动脉線 |

### 術前考量

| 考量 | 內容 |
|-----|------|
| 適應症 | 肺癌是最常見適應症；其他包括良性腫瘤、支氣管擴張和結核 |
| 術前評估 | 患者需要廣泛術前評估以評估心血管和功能 fitness |
| VATS | 大多數楔形切除和肺葉切除現在由VATS完成 |
| 氣道評估 | 評估DLT放置的氣道 |
| 教育 | 確保患者教育、啟動ERAS協議並計劃術後鎮痛方案 |

### 圍手術期考量

| 考量 | 內容 |
|-----|------|
| DLT選擇 | 選擇適當的DLT並在插管後仔細檢查肺隔離 |
| 动脉線 | 大多數涉及OLV的病例放置动脉線 |
| OLV | OLV便利手術並防止依賴肺污染 |
| 手術操作 | 常引起心臟和靜脈壓迫，減少CO/血壓並可能導致心律失常 |
| 支氣管縫合 | 在0.9%氯化鈉下用30 cmH₂O手動充盈測試「洩漏」 |

### 術後考量

| 考量 | 內容 |
|-----|------|
| 拔管 | 瞄準在程序結束時清醒坐著拔管 |
| 氧疗 | 處方持續加濕補充O₂ |
| 鎮痛 | 確保良好鎮痛；許多中心現在使用椎旁鎮痛而非硬膜外 |
| 胸部X光 | 通常需要 |

### 特殊考量

| 考量 | 內容 |
|-----|------|
| 非轉移性表現 | 支氣管肺癌患者可能有（「非轉移性」）表現（Lambert-Eaton肌無力綜合徵或異位激素產生） |
| 肺葉切除術死亡率 | 5-13% |
| ALI | 發生在4%的切除術，死亡率30-50% |
| 風險因素 | 手術的炎症反應、慢性酒精濫用、遺傳易感性、術中平臺壓>15 cmH₂O和24小時內>4000 mL IV液體 |

## 20.10 肺減容手術、肺大泡切除和支氣管內閥門

### 程序概述

| 項目 | 內容 |
|-----|------|
| 程序 | 非解剖性切除或封堵過度擴張和功能不良的肺組織區域 |
| 時間 | 2-5小時 |
| 疼痛 | +++/+++++ |
| 體位 | 正中胸骨切開術（雙側）：仰臥，手臂在兩側；胸廓切開術或VATS：側臥位（如同肺切除） |
| 失血 | 100-500 mL；G&S |
| 麻醉技術 | 仔細IPPV配合OLV；椎旁導管用於鎮痛 |

### 適應症

**患者選擇：**
- 嚴重呼吸衰竭继发於肺氣腫的手術治療
- 旨在通過切除最不健康的區域將總肺容積減少到更生理的水平

**其他適應症：**
- 肺大泡疾病和復發性氣胸

### 術前考量

| 考量 | 內容 |
|-----|------|
| 評估 | 患者需要強化評估、仔細選擇和優化 |
| 死亡率 | 這些是極高風險病例，由於相對較高的死亡率（約8%）現在做得越來越少 |
| 經驗 | 對病理生理學有清晰理解和足夠的胸腔經驗對安全麻醉管理至關重要 |

### 圍手術期考量

| 考量 | 內容 |
|-----|------|
| 入路 | 可通過胸骨切開術、胸廓切開術或VATS進行 |
| 監測 | 动脉線 required用於侵入性監測 |
| 預充氧 | 充分預充氧必要（去飽和可能很快） |
| 囊泡破裂風險 | IPPV有使肺氣腫囊泡破裂的嚴重風險，導致洩漏和張力性氣胸 |
| 禁用 | N₂O禁用 |
| 通氣策略 | 限制「氣體滯留」和動態肺過度擴張的風險：故意低通氣和允許性高碳酸血症（pH >7.2） |

**建議設置：**

| 參數 | 數值 |
|-----|------|
| V_T | 5-6 mL/kg |
| RR | 10-12 breaths/min |
| I:E比率 | 1:4 |
| P_AW | <30 cmH₂O |

### 術後考量

| 考量 | 內容 |
|-----|------|
| 拔管 | 瞄準在程序結束時拔管；大多數需要HDU或ICU護理 |
| 空氣洩漏 | 密切觀察並向外科醫生提醒任何擔憂 |
| 康復 | 需要出色的疼痛緩解、技能的物理治療和肺康復計劃 |

### 特殊考量

| 考量 | 內容 |
|-----|------|
| 最常見併發症 | 長期空氣洩漏：50%患者>7天 |
| 死亡率 | 最近系列為5-10% |
| 國家肺氣腫治療試驗 | 表明肺減容手術使主要為上葉疾病和低基線運動能力的患者受益 |

### 支氣管內閥門

**機制：**
- 單向閥門放置在特定肺段以防止在吸氣期間氣體進入但允許在呼氣期間空氣和黏液離開
- 旨在導致肺氣腫區域的肺不張

**程序：**
- 支氣管鏡操作，可用鎮靜或GA
- 與完全肺減容手術相比，發病率大大降低

**限制：**
- 高達三分之二的重度肺氣腫患者會發生葉間側支通氣
- 這將阻止支氣管內閥門的使用

## 20.11 膿胸引流和剝脫術

### 程序概述

| 項目 | 內容 |
|-----|------|
| 程序 | 手術去除膿（膿胸）和有組織的厚、纖維蛋白性胸膜膜（剝脫術） |
| 時間 | 引流20-40分鐘；剝脫術2-3小時 |
| 疼痛 | +++/+++++ |
| 體位 | VATS側臥位 |
| 失血 | 簡單引流：最少；剝脫術：500-2000 mL；G&S；X-match 2單位 |
| 麻醉技術 | GA、ETT和IPPV；剝脫術建議DLT（有空氣洩漏風險）；动脉線± CVP（如果敗血症且需要血管加壓素） |

### 術前考量

| 考量 | 內容 |
|-----|------|
| 敗血症 | 患者可能明顯敗血症，呼吸功能常已被肺炎或既往肺切除損害 |
| 檢查 | 檢查由糜爛進入肺部造成的支氣管胸膜瘻 |

### 圍手術期考量

| 考量 | 內容 |
|-----|------|
| 引流 | 通常通過肋骨切除和大孔徑肋間引流管引流 |
| VATS | 如果患者能耐受OLV |
| 硬膜外 | 如果需要胸廓切開術應考慮 |
| 出血 | 剝脫術常引起顯著出血 |
| 轉換 | 為轉換為胸廓切開術做準備 |

### 術後考量

| 考量 | 內容 |
|-----|------|
| 鎮痛 | 多模式鎮痛與serratus前或豎脊肌阻滯（用於VATS程序）；在開始時與肋間阻滯；椎旁導管不可能（由於胸膜丟失） |
| HDU | 建議虛弱患者的剝脫術 |

### 特殊考量

| 考量 | 內容 |
|-----|------|
| 手術目標 | 去除感染組織（包括胸膜「外皮」）、完全重新擴張肺部並消除感染的胸膜空間 |
| 空氣洩漏 | 剝脫術臟層胸膜後常見；大量空氣洩漏可能偶爾需要肺葉切除術 |
| 風險評估 | 剝脫術是重大程序，對老年、虛弱和患病患者需要仔細評估風險和益處 |

## 20.12 支氣管胸膜瘻修復

### 程序概述

| 項目 | 內容 |
|-----|------|
| 程序 | 關閉胸膜腔和氣管或支氣管之間的交通 |
| 時間 | 2-3小時（胸廓切開術入路） |
| 疼痛 | ++++/+++++ |
| 體位 | 保持坐直，受影響側向下直到好肺隔離，然後側臥位胸廓切開術 |
| 失血 | 300-800 mL；G&S；如貧血X-match |
| 麻醉技術 | RSI誘導和光纖引導支氣管內插管配合DLT |

### 臨床特點

**症狀嚴重程度與瘻大小成正比：**
- 大瘻伴大量氣漏導致嚴重呼吸困難
- 可能需要緊急呼吸支持

**特徵：**
- 產出性咳嗽、咳血、發熱、呼吸困難
- SC肺氣腫、持續空氣洩漏
- CXR上術後空間的液平面下降

### 術前考量

| 考量 | 內容 |
|-----|------|
| 患者狀態 | 患者常虛弱，呼吸功能被感染和既往肺切除損害 |
| 麻醉圖表 | 檢查既往麻醉圖表（插管難易度和使用的DLT類型） |
| 下氣道解剖 | 在CXR上仔細檢查（常被先前手術扭曲） |
| 復甦 | 患者需要在誘導前復甦並有功能性胸腔引流 |

### 圍手術期考量

| 考量 | 內容 |
|-----|------|
| 關鍵原則 | 保護「好」肺免受污染並控制通氣分佈 |
| 監測 | 誘導前開始侵入性动脉壓監測 |
| 隔離確認 | 在正壓通氣或重新定位患者之前必須確認肺隔離 |
| RSI | 許多胸腔麻醉師使用改良RSI並在直接視覺下用光纖支氣管鏡推進DLT |
| 人員 | 理想情況下需要三名麻醉師 |

**DLT放置：**
1. 一名進行喉鏡和插管
2. 一名操作光纖支氣管鏡
3. 一名觀察麻醉

**確認：**
- 通過光纖支氣管鏡確認管在瘻對側的正確支氣管中
- 首先充盈支氣管氣囊並直接在管的支氣管端口通氣

**風險：**
- IPPB增加氣體洩漏
- 導致V_T丟失和張力性氣胸風險
- 最小化P_AW

### 特殊考量

| 考量 | 內容 |
|-----|------|
| 病因 | 大多數瘻是肺葉切除術或肺葉切除術的手術後併發症，但有些是继发於肺炎、肺膿腫和膿胸 |
| 挑戰 | 修復支氣管胸膜瘻的麻醉具有挑戰性，不建議用於「偶爾」胸腔麻醉師 |

## 20.13 胸膜切除術/胸膜固定術

### 程序概述

| 項目 | 內容 |
|-----|------|
| 程序 | 從胸壁內側剝離壁層胸膜（胸膜切除術）；通過化學（滑石、四環素）或物理磨損在壁層和臟層胸膜之間產生粘連（胸膜固定術） |
| 時間 | 胸膜切除術1-2小時；胸膜固定術20-40分鐘 |
| 疼痛 | +++/++++ |
| 體位 | VATS或開放胸廓切開術側臥位 |
| 失血 | 可從剝離的胸膜出血；G&S |
| 麻醉技術 | 建議IPPV和OLV用於開放/VATS程序 |

### 術前考量

| 患者群體 | 特點 |
|---------|------|
| 年輕fit群體 | 復發性氣胸（檢查哮喘） |
| 老年群體 | COPD或復發性胸腔積液受損（檢查呼吸儲備） |

**檢查：**
- 近期CXR（氣胸和/或積液）
- 確保任何感染得到治療
- 術前胸腔引流（如有氣胸）
- 與外科醫生討論術後鎮痛和區域技術

### 圍手術期考量

| 考量 | 內容 |
|-----|------|
| P_AW | 有氣胸病史的患者保持盡可能低的P_AW |
| 警惕 | 警惕氣胸（在IPPV上即使有引流管到位也可能迅速張力；可能在「健康」側） |
| 禁用 | 避免N₂O |
| 目標 | 程序結束時肺部完全擴張以使壁層和臟層胸膜對合 |

### 術後考量

| 考量 | 內容 |
|-----|------|
| 拔管 | 拔管並在轉移到恢復室前坐直患者 |
| CXR | 需要檢查肺部完全擴張 |
| 鎮痛 | 多模式鎮痛再次必要；但避免NSAIDs（可能通過抗炎作用使胸膜固定術效果降低） |
| 區域技術 | 椎旁阻滯通常不合適（由於胸膜損傷）；可在程序開始時放置肋間阻滯 |

### 特殊考量

| 考量 | 內容 |
|-----|------|
| 胸膜切除術 | 通常用於復發性氣胸，結合釘合負責復發性空氣洩漏的肺組織（通常為尖部「肺大泡」或小囊泡） |
| 胸膜固定術 | 通常用於管理惡性胸腔積液（間皮瘤、轉移性癌） |
| 大量胸腔積液 | CXR上佔半胸以上或>2000 mL的患者應在手術前至少12小時「抽吸」並部分引流 |
| 循環崩潰 | 有大量積液的患者在的手術「積液側向上」轉動時也有循環崩潰風險 |

## 20.14 食道切除術

### 程序概述

| 項目 | 內容 |
|-----|------|
| 程序 | 食道全部或部分切除並動員胃（偶爾結腸）進入胸部 |
| 時間 | 3-6小時 |
| 疼痛 | +++++ |
| 體位 | 仰臥手臂在兩側，和/或胸廓切開側臥位 |
| 失血 | 500-1500 mL；X-match 2單位 |
| 麻醉技術 | IPPV、DLT或支氣管阻斷OLV；动脉/CVP線；尿導管；胸椎硬膜外或椎旁導管用於胸腹切口 |

### 手術類型

| 類型 | 描述 |
|-----|------|
| 微創 | 胸腔鏡食道動員、腹腔鏡胃動員和宮頸吻合術 |
| Ivor-Lewis | 腹腔鏡和右胸廓切開術，常腹腔鏡輔助 |
| 經裂孔 | 腹腔鏡和宮頸吻合術 |
| 胸腹 | 左胸廓切開術穿過肋緣和橫膈膜 |
| McKeown三期 | 腹腔鏡、右胸廓切開術和宮頸吻合術 |

### 術前考量

| 考量 | 內容 |
|-----|------|
| 適應症 | 通常為食道癌，但偶爾用於非惡性疾病（良性狹窄、失弛緩症） |
| 營養 | 術前營養不良或惡病質常見，與較高術後發病率和死亡率相關 |
| 化療 | 術前輔助化療可能留下殘留免疫抑制但可顯著改善吞嚥困難 |
| 單位 | 一些中心使用加速康復協議 |
| 護理 | 根據當地協議需要HDU或ICU |

### 圍手術期考量

| 考量 | 內容 |
|-----|------|
| 反流風險 | 考慮所有食道疾病患者有反流風險；有些患者可能需要改良RSI |
| DLT | 如果計劃胸廓切開術，使用DLT或支氣管阻斷 |
| 重新定位 | 準備長時間手術，有時涉及重新定位 |
| 鎮痛 | 多模式鎮痛必要，包括區域阻滯；避免NSAIDs；給予術中對乙醯氨基酚和鎂；考慮可樂定；肩尖疼痛考慮曲馬多 |
| 區域技術 | 根據手術入路計劃；胸腹入路用椎旁導管輸注配合嗎啡PCEA |
| nGT | 最初需要；用於切除並在吻合術後在外科教導下重新插入 |
| 溫度 | 監測核心溫度；對保持患者溫暖執著 |
| 液體 | 建議液體限制方案（通常2 mL/kg/h輸注，配合血流動力學參數引導的額外bolus用於低血容量）；必要時使用血管加壓素 |
| 監測 | 定期檢查血紅蛋白和血氣 |
| 心律失常 | 胸內食道動員期間可能發生心律失常和減少CO導致低血壓 |
| DLT更換 | 在宮頸吻合術（如進行）前為改善手術入路將DLT更换為SLT |

### 術後考量

| 考量 | 內容 |
|-----|------|
| 拔管 | 早期拔管理想；如冷（<35.5°C）或血流動力學不穩定，通氣直到狀況改善 |
| 護理 | 患者需要在專業病房、HDU或ICU進行強化和經驗豐富的術後護理 |
| 餵養 | 使用空腸造口術或鼻十二指腸管進行早期腸內餵養；對於術後譫妄傾向的患者，bridle可能有用 |

### 特殊考量

| 考量 | 內容 |
|-----|------|
| 死亡率 | 食道切除術是所有擇期程序中術後死亡率最高的之一（高達5%，即使在專業中心） |
| 死因 | 66%的死亡是继发於呼吸併發症或吻合口裂開的全身性敗血症 |
| 併發症 | 超過30%的患者遭受重大併發症 |
| 微創手術 | 大多數中心進行微創（內鏡）食道切除術；警惕在腹腔鏡裂孔解剖期間胸膜破裂時的「張力性二氧化碳胸」 |
| 警告 | 不建議用於食道切除術的偶爾麻醉（或手術）實踐 |

## 20.15 胸部損傷

### 肋骨骨折

| 項目 | 內容 |
|-----|------|
| 發病率 | 常見 |
| 風險 | 與呼吸併發症的高風險相關 |
| 患者群體 | 外傷性損傷越來越多地在>65歲有較高合併症負擔和多重用藥（包括抗凝）的患者中看到 |

**鎮痛：**

| 選項 | 內容 |
|-----|------|
| 藥物 | 對乙醯氨基酚；適當時NSAIDs；口服或PCEA阿片類藥物；區域阻滯 |
| 區域技術 | 椎旁導管插入或筋膜平面導管；超聲引導的豎脊肌或serratus前平面阻滯越來越多；可相對簡單進行，減少氣胸風險，可在凝血病患者中進行 |
| 平面阻滯導管 | 可用30-40 mL 0.25% levobupivacaine bolus，然後以10 mL/h 0.125% levobupivacaine輸注 |
| 椎旁導管 | 替代方案，提供可靠的單側阻滯，減少低血壓、運動阻斷和尿瀦留（與胸椎硬膜外相比） |
| 操作固定 | 可能需要，特別是有連枷段和無法控制的疼痛的患者 |

**支持治療：**
- 加濕O₂、霧化0.9%氯化鈉和呼吸物理治療
- 無創通氣或高流量O₂減少肺不張並改善連枷段的矛盾運動
- 如果可能避免IPPV

### 膈肌破裂修復

| 項目 | 內容 |
|-----|------|
| 表現 | 可作為慢性病或腸梗阻（腸突出）呈現 |
| 術前檢查 | 檢查術前液體和電解質狀態 |
| 封閉 | 應及時封閉缺陷，但很少緊急 |
| 入路 | 通過標準側胸廓切開術或胸腹切口；DLT便利手術入路 |
| 管理 | 如胃底折疊術所述 |
| 禁用 | 避免N₂O（會擴張腸道並可能使疝氣復位更困難） |
| nGT | 應使用以減壓胃 |

### 食道破裂修復

| 項目 | 內容 |
|-----|------|
| 病因 | 可由異物引起，但通常為醫源性（內鏡程序期間） |
| 病理 | 縱隔炎迅速發展為敗血症和全身炎症反應綜合徵（SIRS），伴有循環休克、腎衰竭和ARDS |
| 管理原則 | 最初為引流和防止進一步污染 |
| 術前穩定 | 胸腔引流、IV液體置換、鎮痛、侵入性監測和正性肌力支持 |
| 術後餵養 | 通過餵養空腸造口術或腸外早期術後餵養 |
| 裂開 | 有顯著發生率的裂開，導致食道胸膜皮膚瘻，高死亡率 |

### 氣管支氣管損傷修復

| 項目 | 內容 |
|-----|------|
| 初始管理 | 100% O₂和緩解張力性氣胸 |
| 氣道評估 | 如果通氣和氧合可接受，呼叫胸外科幫助並在插管前通過光纖支氣管鏡評估和識別氣道損傷部位 |
| 原則 | 氣道管理和麻醉原則適用如同大支氣管胸膜瘻 |
| 技術 | 撕裂的支氣管可通過光纖引導插管隔離到對側完整主支氣管（適當DLT） |
| 上氣管撕裂 | 未切割的SLT可在支氣管鏡引導下穿過上氣管撕裂，其氣囊在損傷遠端就位 |
| 修復 | 一旦氣道安全且通氣穩定，進行緊急胸廓切開術修復 |
| 嶺部分離 | 可能需要CPB在修復期間維持氧合 |
