## 總論（General Principles)

### 概述

多數婦科患者身體健康，接受相對較小的門診手術。部分是接受較大型手術的住院患者。老年人常需手術緩解骨盆底脫垂。

### 一般考量

- **焦慮**：多數患者焦慮，相對需要較高誘導劑量。
- **術後噁心嘔吐（PONV）**：是高風險問題。高風險患者使用適當技術；避免氧化亞氮（笑氣），給予預防性止吐藥，考慮全靜脈麻醉。
- **靜脈血栓栓塞（VTE）**：骨盆手術與VTE相關。確保採取了充分的預防措施。
- **抗生素預防**：可減少特定手術的術後傷口感染率。查看醫院方案。
- **迷走神經刺激**：可能發生於宮頸擴張、骨盆器官或腸繫膜拉扯、或腹腔鏡手術期間。
- **體位擺放**：患者常需在手術台上上下移動，此時氣道裝置可能脫落和脫離。現有背部或關節疼痛在截石位可能惡化。腿部以腳鐙支撐時，總腓神經有受傷風險。
- **直腸給藥**：可要求婦科醫師在麻醉下經直腸給予止痛藥。需確保已獲患者同意。
- **保暖**：確保患者在長時間手術中保持溫暖。
- **大量失血**：大型婦科手術可能造成大量失血且可能延長。自體血回收應備妥以供此類情況使用。
- **微創手術轉變**：許多先前以開放方式進行的婦科手術（如、子宮切除術、輸卵管妊娠修復）現在主要使用腹腔鏡技術。

## 妊娠物清除術（ERPC）/ 真空或吸刮終止妊娠（STOP/VTOP）

### 手術基本資料

| 項目 | 內容 |
|------|------|
| 手術方式 | 妊娠物清除術；吸刮或真空終止妊娠 |
| 手術時間 | 10 至 20 分鐘 |
| 疼痛程度 | + |
| 體位 | 截石位 |
| 失血量 | 通常輕微 |
| 實用技術 | 喉罩、自主換氣；門診手術 |

### 術前考量

ERPC：剩餘妊娠物可能需在不完全流產後手術清除。通常發生在妊娠 6 至 12 週之間。術前可能已發生大量失血並可能持續至術中。若出血看起來並非輕微，需建立靜脈通路並輸注晶體液。

STOP/VTOP：在妊娠達 12 週以內進行。

### 術中管理

使用喉罩或面罩。未禁食的緊急患者應進行氣管內插管。避免高濃度吸入麻醉劑，因對子宫有鬆弛作用。Propofol 誘導後間歇追加或 Propofol TCI 加鴉片類（Alfentanil 或 Fentanyl）是合適選擇。

外科醫師可能要求給予幫助宮縮和減少出血的藥物。可緩慢給予 Oxytocin 3 至 5 單位。

### 術後護理

口服止痛藥和止吐藥。

### 特殊考量

超過 12 週的妊娠可經由擴張和清除以手術方式終止。程序類似 STOP/VTOP，但失血潛力更大。

若有胃食道逆流症狀，需Ranitidine 或質子泵抑制劑術前給藥並進行氣管內插管。

若妊娠超過 16 週，可能需以前列腺素進行醫療終止。這些患者仍可能需要 ERPC，且更可能發生大量失血。

## 診斷性腹腔鏡/腹腔鏡絕育術

### 手術基本資料

| 項目 | 內容 |
|------|------|
| 手術方式 | 婦科器官的腹腔內檢查 +/- 治療 +/- 輸卵管夾 |
| 手術時間 | 30 至 60 分鐘 |
| 疼痛程度 | +/++ |
| 體位 | 截石位 |
| 失血量 | 無 |
| 實用技術 | 喉罩/氣管內管、正壓換氣；門診手術 |

### 術前考量

通常年輕健康。術前給予口服止痛藥。

### 術中管理

使用短效鴉片類（如 Fentanyl）。給予止吐藥以確保當日出院。鼓勵創口皮膚浸潤局部麻醉藥。

傳統氣道管理是氣管內插管，配合最小劑量的神經肌肉鬆弛劑，因為手術通常較快。使用神經刺激器監測，相應使用逆轉劑。

對於不複雜的短程序，聲門上呼吸道（LMA）是替代方案。這只適合非肥胖患者。必須仔細評估胃內容物逆流和吸入的風險。外科醫師可能因自主換氣困難而要求肌肉麻痺。

### 術後護理

可能需要追加鴉片類或止吐藥。

### 特殊考量

心搏過緩常見，由迷走神經刺激引起。應隨時備好 Atropine；要求外科醫師釋放氣腹。

肩痛是常見的術後問題，由於橫膈刺激。雖然是自限性，但可能難以治療。可在程序結束時盡量排出二氧化碳來減少。

若發現並切除嚴重內膜異位，術後疼痛可能嚴重，當日出院可能困難。

非常罕見地，二氧化碳可能意外注入血管內，導致靜脈空氣栓塞。這導致通氣/血流比例失衡，表現為呼氣末二氧化碳下降、心輸出量受損、低血壓、心律不整和心搏過速。通知外科醫師並復甦患者。

若使用聲門上呼吸道，考慮術前給予 H2 受體拮抗劑或質子泵抑制劑，並使用有胃通道的裝置（如 ProSeal）。

## 子宮切除術（Hysterectomy）

### 手術基本資料

| 項目 | 內容 |
|------|------|
| 手術方式 | 子宮切除（可能同時切除卵巢 +/- 輸卵管，即雙側輸卵管卵巢切除術） |
| 手術時間 | 1 至 3 小時，取決於手術方式 |
| 疼痛程度 | + 至 +++ |
| 體位 | 仰臥位，頭低腳高位或截石位 |
| 失血量 | 250 至 1500 mL；血型及交叉比對 |
| 實用技術 | 喉罩、自主換氣適合陰道、子宮切除術；腹腔鏡或開放手術需氣管內管、正壓換氣；脊髓 Diamorphine 可用於鎮痛 |

### 術前考量

適應症多樣，從陰道/子宫脫垂、子宮過多出血、子宮肌瘤到婦科惡性腫瘤。

患者若因子宮過多出血或停經後出血可能有貧血。若腹部腫塊壓迫輸尿管，腎功能可能異常。PONV 常見。

接受術前化學治療縮小腫瘤的患者常貧血、虛弱和營養不良。應視為高風險患者。

所有開放和根除性子宮切除術應使用自體血回收。確保已開始 VTE 預防。

### 術中管理

氣道管理如上述。抗生素預防通常需要。腹腔鏡方式需要陡的頭低腳高位。

失血量變異大；部分子宫切除術出血比預期多。若出血有問題及早備血。腹部切口散熱可觀。術中在上身使用加壓空氣保暖毯。

### 術後護理

陰道和腹腔鏡手術的疼痛通常以口服止痛藥能良好控制。這些患者通常在 48 小時內準備出院。

開放手術需要額外術後鎮痛。選項包括 PCIA、腹直肌鞘導管和中樞神經軸索阻斷。含 Diamorphine 的脊椎麻醉現在比硬膜外更受青睞。

### 陰道、子宮切除術（Vaginal Hysterectomy）

常配合前方 +/- 後方修補術，以減少膀胱和腸道經陰道脫垂。陰道、子宮切除術通常不可能經陰道移除輸卵管和卵巢，因為手術野受限。

腹腔鏡輔助陰道、子宮切除術（LAVH）設計目的是能夠經陰道移除子宫、輸卵管和卵巢。手術從腹腔鏡開始，識別並分離寬韌帶。一旦腹腔鏡下達到婦科器官滿意的活動度，經陰道移除。

### 腹腔鏡和腹部子宫切除術

隨著腹腔鏡技術進步，開放手術數量已減少。現在通常保留給晚期惡性腫瘤和子宫太大無法經腹腔鏡移除的案例。

腹腔鏡和開放手術都需要良好的肌肉鬆弛。為宮頸或子宫惡性腫瘤進行的腹腔鏡子宫切除術越來越多地結合哨兵淋巴結活檢。藍色染料在宮頸周圍註射，允許術中哨兵淋巴結採樣。有過敏反應風險。

根除性子宮切除術適用於宮頸和子宫惡性腫瘤患者。除子宫外，輸卵管和通常卵巢也被切除，並進行骨盆腔淋巴結清掃。這些手術時間更長，有大量失血的潛力。應考慮动脉導管和目標導向液體治療。脊髓麻醉用於術後鎮痛有用。

## 子宮外孕（Ectopic Pregnancy）

### 手術基本資料

| 項目 | 內容 |
|------|------|
| 手術方式 | 腹腔鏡（較少用剖腹探查術）止血破裂的輸卵管妊娠 |
| 手術時間 | 60 分鐘 |
| 疼痛程度 | ++/+++（取決於手術方式） |
| 體位 | 截石位 |
| 失血量 | 可能大量；備血 2 單位 |
| 實用技術 | 氣管內管、正壓換氣 |

### 術前考量

表現多樣。穩定患者可能有模糊的腹痛和停經；其他可能以危及生命的腹腔出血表現。至少應在手術前建立一個大口徑靜脈針，並根據臨床情況輸注晶體液或血液製品。

入院時應要求血液常規、備血和可能的凝血篩檢。若患者不穩定，尋求第二麻醉醫師的幫助。

### 術中管理

快速順序插管。誘導要謹慎，因為大量失血可能隱蔽。持續靜脈液體復甦並積極為患者保暖。

### 術後護理

若失血量大，凝血異常並不少見。送凝血篩檢或進行床邊凝血測試。鎮痛需求視手術方式而異。可能需要 PCIA。

### 特殊考量

患者年輕，通常對大量失血有代償能力，術前臨床表徵可能錯誤地令人安心。

多數中心的手術以腹腔鏡進行，只在有併發症時才轉為剖腹探查術。注意氣腹可能阻礙靜脈回流，導致低血壓。
