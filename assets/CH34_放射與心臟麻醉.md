## 診斷影像麻醉（Anaesthesia for Diagnostic Imaging）

### 總論

麻醉醫師越來越多地在醫學影像部門工作。然而，環境仍然有潛在危險，設備也不熟悉。確保有經驗、受訓的協助和完整監測。熟悉工作人員和環境。找到最近的復甦設施。

### 適應症

- 無法合作或焦慮的兒童
- 心理、行為或運動障礙的較大兒童或成人
- 急性創傷患者或加護病房患者
- 需要止痛、鎮靜或麻醉的超聲、CT 或 MRI 引導介入手術

### 麻醉要點

- 檢查麻醉機使用管路氣體或鋼瓶
- 與放射技師事先計劃麻醉機、抽吸、監測和呼吸系統的配置和路由
- 決定誘導地點——可能在專用誘導區或非常小的空間
- 確保有合適的恢復設施

## CT 麻醉（Anaesthesia for CT）

### 總論

CT 掃描不限制使用的設備類型，但空間可能有限，因此緊湊的機器和監測儀是理想的。

患者頭部在 CT 掃描期間通常可及，因此若不需要氣道保護，可使用聲門上氣道。

### 危害

- CT 掃描使用游離輻射，麻醉醫師最好在掃描室外監測患者
- 移動患者通過掃描器時，針頭、導管、引流管和氣管內管可能拉出——事先檢查無物品絆腳

### 對比劑

- 靜脈對比劑通常為碘基、非離子、水溶性化合物
- 可能觸發過敏反應
- 自動化對比注射器不應連接到標準中心靜脈線
- 對比劑可能對脫水或腎功能受損患者造成腎損傷
- 服用雙胍類（二甲雙胍）的患者可能誘發乳酸酸中毒——理想上在掃描前後 48 小時避免

## MRI 麻醉（Anaesthesia for MRI）

### 概述

MRI 使用超導電磁體產生強大磁場。電腦從氫原子核產生的射頻信號創建圖像。MRI 產生詳細圖像，特別是軟組織，且無游離輻射危險。

### 危害

| 危害類型 | 說明 |
|----------|------|
| 靜磁場位移 | 鐵磁性材料可變為拋射物；外來物（某些心臟瓣膜、動脈瘤夾、眼部鋼碎片）可移動；植入裝置（心臟起搏器）可能移動或失效 |
| 噪音 | 快速磁場變化可產生 >80 dB 噪音並損害聽力 |
| 射頻場加熱 | 可導致嚴重燒傷；起搏器導線可能加熱 |
| 氦氣洩漏 | 突然關閉磁場時，液態氦冷卻劑迅速膨脹為氣體 |
| 對比劑 | 高達 30% 的掃描使用 Gadolinium，副作用包括頭痛、噁心、頭暈；過敏反應罕見 |

### 設備安全分類

| 分類 | 說明 |
|------|------|
| MRI 安全 | 不含任何可能造成危害的材料 |
| MRI 條件性 | 在製造商指定的條件下安全使用 |
| MRI 不安全 | 嚴禁带入 MRI 掃描器 |

### 氣道管理

- 氣道和血管通路要牢固固定
- 插管和換氣：嬰兒和小兒童（<10 kg）、顱內壓升高或有此懷疑的患者，或需要保護氣道的患者
- 自主換氣經聲門上氣道：較大兒童和成人，無顱內壓升高風險時可考慮
- 不要使用含金屬線圈的靈活聲門上氣道

### 重症監護患者

- 需要詳細計劃
- 避免掃描血流動力學或不穩定患者
- 傳統監測（包括顱內壓換能器和臨時起搏導線）應移除或更換為「MRI 安全」設備
- 輸液管線要足夠長，使泵可在安全距離從磁鐵放置

## 介入手術放射麻醉（Anaesthesia for Interventional Radiology）

### 總論

介入手術放射使用多種影像模式指導微創診斷或治療程序。

### 適應症

- 需要長時間保持不動的患者
- 疼痛程序
- 兒童患者

### 氣道與麻醉方式

- 選項包括監測護理、鎮靜、全身麻醉和區域技術或其組合
- 清理系統可能不可能，可能需要活性炭吸附器或全靜脈麻醉

### 血管程序：血管成形術/支架置入

- 氣球擴張/臨時阻斷某些血管結構可引起迷走神經反應
- 腹主動脈瘤的血管內修復與較低死亡率相關

### 栓塞治療

- 用於或預防出血、阻止腫瘤生長
- 涉及酒精的栓塞治療程序或動靜脈畸形的程序可能非常疼痛，需要鎮靜或全身麻醉

### 射頻消融

- 通過加熱破壞組織（如腫瘤）
- 常用於治療肝臟和腎臟腫瘤

###血栓溶解和血栓切除

- 微創溶解或移除血塊以改善血流供應
- 可用於急性腦血管意外、冠狀動脈栓塞等的急性管理

## 心臟程序麻醉（Anaesthesia for Cardiology Procedures）

### 總論

這些程序通常在心導管室（「cath lab」）進行，許多放射麻醉的額外考量同樣適用。

### 適應症

- 心房顫動消融
- 經導管主動脈瓣置入術（TAVI）
- 心血管重建術
- 心臟電擊復律
- 心臟裝置置入

### 麻醉要點

| 項目 | 內容 |
|------|------|
| 藥物準備 | 準備好預期使用的麻醉藥物和緊急用藥（如 Metaraminol、Ephedrine、Atropine、Suxamethonium） |
| 設備 | 可能需要多個輸液泵、麻醉深度監測或噴射換氣機 |
| 監測 | 鉛圍裙和甲狀腺屏蔽是強制性的 |
| 誘導 | 在實驗室本身最容易，但有噪音和干擾 |

## 心血管電擊復律麻醉（Anaesthesia for Cardioversion）

### 概述

心血管電擊復律可以是選擇性或緊急的。

### 考量

| 項目 | 內容 |
|------|------|
| 患者特徵 | 通常為門診患者，常有記錄的不良心室功能和心房顫動 |
| 抗凝 | 檢查抗凝合規性或 INR 充足性 |
| 氣道管理 | 若需要相關的經食道心臟超聲（檢查心房附屬物血塊），可能需要明確氣道管理；否則頭高位面罩和預充氧 |
| 麻醉方式 | Propofol +/- Midazolam 通常足夠 |

### 體外電擊復律

- 墊放置從前到後最有效
- 若有植入式心律去顫器（ICD）在場，應有電生理學家在場

## 心導管檢查麻醉（Anaesthesia for Angiography）

### 考量

- 心導管檢查和血管成形術最常在局部麻醉下由心臟病專家進行
- 部分情況需要全身麻醉或鎮靜

### 特定情況

| 情況 | 麻醉考量 |
|------|----------|
| 院外心臟驟停後倖存者 | 可能不穩定但痛苦和躁動，需要仔細鎮靜 |
| 無反應患者 | 需要插管和換氣，術後轉ICU |
| 不穩定患者 | 平衡安全監測與快速干預的需求 |

## 心臟裝置置入麻醉（Anaesthesia for Cardiac Device Insertion）

### 裝置類型

| 裝置 | 說明 |
|------|------|
| 永久起搏器 | 單腔或雙腔 |
| ICD | 植入式心律去顫器 |
| CRT | 心臟再同步治療裝置（CRT-P 或 CRT-D）|
| S-ICD | 皮下植入式心律去顫器 |

### 程序與麻醉方式

| 程序 | 麻醉方式 | 備註 |
|------|----------|------|
| 永久起搏器置入 | 局部麻醉 +/- 鎮靜 | 通常由心臟病專家進行 |
| ICD 置入 | 局部麻醉 +/- 鎮靜 | 可為一級或二級預防 |
| ICD 置入（需測試）| 全身麻醉（聲門上氣道/氣管內管）| 需动脉導管 |
| S-ICD 置入 | 全身麻醉（配合局部浸潤 +/- 區域阻斷）| - |
| CRT 置入 | 全身麻醉或局部麻醉 + 鎮靜 | 心室功能不良 |
| 起搏器盒更換 | 局部麻醉 +/- 鎮靜 | 深部/傷口修補時全身麻醉 |
| 導線/系統拔除 | 全身麻醉（氣管內管）| 可能感染；大出血風險 |

### 導線拔除

- 所有導線拔除案例都應備好大出血搶救
- 困難案例應有心臟外科醫師和體外循環在場

## 電生理學程序麻醉（Anaesthesia for Electrophysiology Procedures）

### 概述

消融手術現在常作為陣發性和持續性心房顫動的二線治療。常用技術包括射頻能量（燒灼）或冷凍療法（冷凍）。

### 程序要點

| 項目 | 內容 |
|------|------|
| 手術時間 | 2-4 小時 |
| 患者移動 | 需要最小化 |
| 心房消融目標 | 四條肺靜脈的分離 |
| 輻射 | 過程中使用游離輻射 |

### 通氣與入路

- 整個過程可用低潮氣量和高呼吸頻率的間歇性正壓換氣
- 或配合 Jet 換氣（頻率 150，30-60 分鐘）
- 若使用 Jet 換氣，使用 TIVA 配合 EEG/BIS 監測

### 考量

- 程序桌窄且不能傾斜或頭高位
- 使用體下加壓空氣保暖器和溫度監測
- 0.9% 氯化鈉沖洗消融導管尖端導致大量液體負荷（1.5 L）
- 平衡限制液體 +/- 血管收縮劑

## 經導管主動脈瓣置入術麻醉（Anaesthesia for TAVI）

### 概述

經導管主動脈瓣置入術是越來越常見的手術，用於避免手術主動脈瓣置換。最初在高風險患者（虛弱/老年/共病症）中進行，現在適應症擴展到較低風險患者。

### 方式

| 方式 | 說明 |
|------|------|
| 經股骨 | 最常見，經皮 |
| 經心尖 | 小型開胸術 |
| 其他 | 經腋下、經主動脈、經腔靜脈 |

### 經股骨方式

- 通常在局部麻醉配合鎮靜下進行
- 需要良好的周邊通路

### 經心尖/經主動脈方式

- 按體外循環設置：动脉導管、中心靜脈壓、大口徑靜脈通路、導尿管、體外電擊復律墊
- 通常 TCI 全身麻醉（如 Remifentanil、Propofol、鬆弛劑；插管和換氣）
- 為所有 TAVI 程序備好緊急藥物

### 程序要點

| 項目 | 內容 |
|------|------|
| 快速心房起搏 | 程序需要快速心房起搏以允許瓣膜切開和放置 |
| 血壓維持 | 維持收縮壓 >100 mmHg；備好腎上腺素（1:10,000）|
| 低血壓管理 | 考慮硝酸甘油輸注 |

### 術後考量

| 項目 | 內容 |
|------|------|
| 鎮痛 | 浸潤所有傷口和靜脈 Acetaminophen；考慮開胸術的椎旁阻斷 |
| 恢復 | 仰臥位（股骨入路關閉），補充氧氣 |
| 併發症 | 心律不整/心臟傳導阻滯/心包填塞/破裂/瓣膜錯位或遷移/冠狀動脈阻塞/腦血管意外 |
