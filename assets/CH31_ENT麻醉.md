## 總論（General Principles）

### 概述

耳鼻喉科手術的主要關注點是氣道問題，與潛在臨床問題和共用氣道相關。

呈現的病理可能：
- 造成氣道阻塞
- 使入路困難或不可能

在氣道內或附近工作的外科醫師可能：
- 位移、阻塞或損壞氣道裝置
- 遮擋麻醉醫師對患者的視線
- 限制術中對患者的入路
- 造成氣道內出血（術中及術後）
- 潛在術後氣道水腫/阻塞加劇

外科醫師和麻醉醫師應共同計劃，使用能提供良好手術條件同時維持安全、穩固氣道的技術。當術中懷疑氣道問題，校正問題是首要優先，必要時停止手術。頭部周圍的其他結構在手術期間無法觸及，需要保護，特別是眼睛。確保眼睛保持閉合並以適當膠帶固定，必要時墊好，並防止設備造成壓力，特別是長時間手術。

### 氣道/通氣管理

#### 氣管內管或聲門上氣道

傳統上，氣管內管（ETT）用於多數耳鼻喉科工作的氣道保護。預成型 Rae 管提供極佳保護且對手術野干擾最小。

口腔 south-facing Rae 管用於鼻腔和大口腔手術，雖然鼻管（north-facing）允許更好的口腔手術入路。

聲門上氣道（LMA），通常是加強型，提供對血液或手術碎屑吸入的充分保護，並避免氣管內插管/拔管的併發症。然而，它在更大程度上限制手術入路，且在手術期間更容易位移（可能帶來災難性結果）。

#### 自主換氣或間歇性正壓換氣

多數耳鼻喉科手術不需要持續神經肌肉阻斷，且面神經監測常用。換氣技術的進步使許多耳鼻喉科麻醉醫師偏好自主換氣或壓力支援自主換氣。這允許滴定鎮痛，並提高深拔管技術的成功率。

Remifentanil/鴉片類輸注的使用優化手術條件（並避免神經肌肉阻斷劑）通常需要經 ETT 或 LMA 的間歇性正壓換氣。靜脈麻醉技術配合自主換氣的患者現在很常見。描述有多種誘導技術。

### 深拔管或淺拔管

許多耳鼻喉科手術在氣道內造成出血。術前在直視下徹底抽吸（和紗布移除）對此類案例至關重要。

一個特別的危險部位是軟腭後方的鼻咽，這區域不易看見。血液在此滯留可在拔管後被吸入，導致致命結果（「驗屍官的血塊」）。最好使用鼻抽吸導管或將彎頭抽吸管旋轉使其成角尖端置於懸雍垂後方來清除。

過度咳嗽可能加劇出血風險並影響拔管技術選擇。喉痙攣可隨拔管發生，特別是兒童，因近期氣道器械操作或血液刺激。風險透過在深或淺（而非之間）的狀態下拔管來最小化。

#### 深拔管

深拔管最適合自主換氣/支援自主換氣。手術結束時，持續甚至增加麻醉劑濃度，但將氣體更換為 100% 氧氣以增加功能殘氣量。仔細抽吸後，插入 Guedel 气道；將患者翻身至左側臥/頭低腳高（扁桃體位）；檢查呼吸規律（翻身可能產生短暫咳嗽/屏氣），然後拔管。

檢查氣道/呼吸完整，並在此體位保持患者直到氣道反射恢復。由於患者在恢復室最初仍無意識，訓練有素的恢復室工作人員的護理至關重要，並有麻醉醫師在場以應對氣道併發症。

#### 淺拔管

淺拔管推薦用於所有氣道困難或顯著呼吸功能受損的患者。

仔細抽吸後，任何殘餘神經肌肉阻斷劑逆轉，停止麻醉藥物，並在喉反射恢復後拔管。淺拔管最初常產生短暫的咳嗽/躁動。鴉片類使用可減少此情況。

### 喉部紗布

喉部紗布（濕紗布或棉塞）常用於 ETT/LMA 周圍，以吸收可能在上呼吸道滯留的血液。在鼻腔手術（出血可能大量且手術中不清除）和牙科手術中特別有用。

紗布必須在拔管前移除，否則可導致災難性氣道阻塞。確認移除的系統包括：
- 將紗布綁或貼於 ETT
- 在 ETT 或患者額頭上貼識別貼紙
- 將紗布計入洗手護士的清點
- 標準化喉部紗布移除是外科還是麻醉任務，但無論如何拔管前都要進行直視喉鏡檢查

### 鼻腔血管收縮劑

局部血管收縮劑常規用於減少鼻腔手術出血，可經噴霧、凝膠或浸泡紗布給予。可卡因溶液（如 Moffett 溶液：可卡因、碳酸氫鈉和腎上腺素）使用減少，主要是由於藥物儲存/取得的疑慮和全身毒性（高血壓、心律不整、欣快感）。外用可卡因最大劑量為 3 mg/kg。

專有減充血劑是常用替代，如偽麻黃鹼或苯腎上腺素，效果相當。全身吸收可導致短暫的擬交感神經反應。

浸潤含腎上腺素溶液可能作為補充，但全身效應風險更大。

### Remifentanil

Remifentanil 的強效鴉片類作用，加上其快速恢復特性，使其在耳鼻喉科，特別是大型案例中廣泛使用。通常以輸注給予，臨床應用包括：

- 中耳手術/大型頭頸部切除（控制性低血壓減少出血）
- 腮腺切除術（方便不使用鬆弛劑的間歇性正壓換氣）
- 喉鏡/咽鏡檢查（減弱高血壓反應）
- 需要神經監測的程序（神經肌肉阻斷劑禁忌）

注意在誘導時，特別是老年患者，可能引起心搏過緩/低血壓。患者間變異性大大限制了預定輸注方案的价值。

大型手術時，為防止術後恢復室反彈性高血壓/躁動，持續低速率 Remifentanil 輸注或在手術結束前 15 至 20 分鐘給予長效鴉片類（嗎啡）；Clonidine 達 2 mcg/kg IV 或 Dexmedetomidine 也可能有用。

## 術前氣道阻塞（Preoperative Airway Obstruction）

### 評估

術前氣道阻塞患者通常為確診或緩解阻塞而接受手術。

阻塞可在聲門上、聲門或聲門下。阻塞最常見的部位是喉，典型產生喘鳴（高音調、吸氣性）。

成人中，腫瘤是急性上氣道阻塞的最常見原因，儘管血腫或感染（包括會厭炎）也可能。在兒童中，感染（哮吼）或異物更可能；在英國，Hib 疫苗已實際消除了兒童會厭炎。

阻塞逐漸發生時，患者可非常有效地代償，嚴重阻塞可能發展而無明顯身體徵象。識別顯著上氣道阻塞的特徵包括：

- 吸氣長而慢，說話時暫停
- 近期運動耐量明顯下降
- 睡眠時喘鳴惡化（從配偶處獲得病史）

口咽病變很少以氣道阻塞表現，術前檢查通常直接明瞭。重要特徵是張口受限和舌頭突出，以及任何腫塊對氣道的影響。

放射學檢查（X光片、CT/MRI）或耳鼻喉科門診柔性或間接喉鏡檢查可能提供有用資訊。

### 初步處理

對於危及生命的氣道阻塞，可能需要緊急干預，但通常手術將是計劃程序。確定氣道管理的時間緊急性。是否有時間進行進一步檢查？

#### 高流量鼻導管氧氣

經高流量鼻導管的氧氣輸送革新了氣道管理——多種裝置，包括 Optiflow，提供低水準正壓，改善氣道開放性，改善氧合並減少呼吸功和窘迫。可整合入清醒插管技術並延長窒息性氧合。

#### 類固醇

如果與水腫/炎症相關有用，在急性環境中不太可能造成傷害且可能有益。成人氣道阻塞直接證據較少，但改善兒童哮吼症狀，並改善成人喉水腫和拔管後喘鳴。

#### 噴霧腎上腺素

病例報告中似乎有助減少各種阻塞病因相關的喘鳴（較少更好證據）。廣泛使用，1 至 5 mg 腎上腺素噴霧於氧氣中。

#### 氦氣

醫療 Heliox（混合物可為 21/70 或 30/70 氧/氦）。黏度類似空氣但較低密度增加層流的傾向並減少湍流的阻力。受混合物中對流動特性有用的氧含量減少所限制。

#### 持續性正壓氣道

支撐任何塌陷的氣道段。在緊急環境中可用性不定，高流量鼻導管可能更好耐受。

### 確立的氣道管理方法

確立的氣道入路的主要問題是：

- 氣道阻塞可能因平躺、全身麻醉（所有技術）或器械操作氣道（喉痙攣、出血）而加劇
- 由於解剖扭曲（特別是聲門上病變）、分泌物或血液，識別聲門入口可能困難
- 嚴重狹窄可能使管道通過困難（特別是聲門或聲門下腫瘤）

沒有證據支持任何一個特定麻醉技術。傳統上，確立阻塞氣道安全入路的三個主要選項是：

1. 直視喉鏡檢查並在深吸入麻醉下插管（自主換氣）
2. 局部麻醉下的前頸部氣道異物取出
3. 局部麻醉下的氣管切開術（或在較不嚴重案例中，深吸入麻醉配合面罩或聲門上氣道）

無論使用何種技術，都應準備全套設備，包括不同喉鏡、氣管切開套件和各尺寸氣管內管。在冰中保存的小 ETT 會較硬，可用於通過緊狹窄變。

## 噴射換氣（Jet Ventilation）

### 概述

噴射換氣描述從高壓源傳送氣體射流至開放氣道，通常依賴被動呼氣。噴射換氣有顯著風險，主要是由於對技術的不熟悉。經錯誤放置/方向錯誤的套管進行噴射輸送可能帶來災難性後果。氣體交換不足常見。

### 低頻率噴射換氣

通常使用注射器系統實現，如手動可調流量 Manujet。連接方式包括：

- 經皮甲狀腺膜套管置入，瞄準氣管隆突
- 噴射導管（硬性、不壓縮），尖端置於氣管中段
- 連接至手術喉鏡/硬性氣管鏡的近端注射針頭

### 高頻率噴射換氣

需要能執行高頻率（通常 1 至 10 Hz）的專業換氣機。溫暖和濕化的吸入噴射由電子控制。適應症包括：

- 經氣管/環甲膜套管的緊急氣道管理
- 氣道手術（包括雷射）
- 胸腔手術，包括肺隔離技術期間的非依賴肺換氣

## 通氣管置入（Grommet Insertion）

### 手術基本資料

| 項目 | 內容 |
|------|------|
| 手術方式 | 鼓膜切開術和通氣管置入，通常為雙側 |
| 手術時間 | 5 至 15 分鐘 |
| 疼痛程度 | + |
| 體位 | 仰臥，頭側傾，頭環 |
| 失血量 | 無 |
| 實用技術 | 面罩或聲門上氣道；自主換氣使用 T-piece 或兒科迴路 |

### 術前考量

通常為兒童（1 至 8 歲），通常為門診手術。檢查近期上呼吸道感染。口服 Acetaminophen/NSAID。

### 術中管理

聲門上氣道常用。若外科醫師願意繞過面罩工作，面罩適合。鋪單前插入 Guedel 气道，並確保整個過程中儲液囊可見。

### 術後護理

額外鎮痛需求不太可能。

### 特殊考量

若面罩氣道困難，早更換聲門上氣道。與鼓膜的部分迷走神經支配相關，偶爾可見反射性心搏過緩。

## 扁桃體切除術/腺樣體切除術：兒童

### 手術基本資料

| 項目 | 內容 |
|------|------|
| 手術方式 | 從口咽部（扁桃體）或鼻咽（腺樣體）切除淋巴組織 |
| 手術時間 | 20 至 30 分鐘 |
| 疼痛程度 | +++ |
| 體位 | 仰臥，肩墊 |
| 失血量 | 通常少量，可能術後出血 |
| 實用技術 | South-facing Rae 管或置於 Boyle-Davis 張口器槽中的加強型聲門上氣道；自主換氣或間歇性正壓換氣 |

### 術前考量

仔細病史以排除/定義阻塞性睡眠呼吸中止或活動性感染。若病史/睡眠研究顯示中/重度阻塞性睡眠呼吸中止，考慮過夜住院。

考慮口服 Acetaminophen/NSAID。如使用直腸鎮痛，請同意。

### 術中管理

靜脈或吸入誘導（七氟烷 +/- 笑氣）。口咽氣道在腺樣體大阻塞鼻咽時有用。

使用鬆弛劑或深吸入/靜脈麻醉進行插管（Rae 管），或使用 Propofol/鴉片類或深吸入麻醉插入聲門上氣道。在中線固定，無紗布（遮擋手術野）。

注意外科醫師術中可能位移/阻塞管道，特別是在插入或打開 Boyle-Davis 張口器後。

可靠靜脈針至關重要，靜脈輸液是常規。嗎啡或 Fentanyl 滴定靜脈給予，加 Acetaminophen/NSAID（如術前未給）。考慮 Tramadol。至少推薦一種止吐藥——Dexamethasone 或 Ondansetron。

術末在直視下徹底抽吸口咽和鼻咽（通常由外科醫師執行）。在左側臥/頭低腳高位（扁桃體位）拔管，置入口咽氣道。

### 術後護理

在扁桃體位保持患者直到氣道反射恢復。高品質恢復護理至關重要。

最初以靜脈嗎啡/Fentanyl 鎮痛，然後口服 Acetaminophen/NSAID/嗎啡。Dexamethasone 有使用。保留靜脈針（沖洗）以防出血。

### 特殊考量

小兒童可用枕頭/滾筒墊於肩膀下以提供所需傾斜。避免使用硬抽吸管盲目抽吸咽部，因為可能從扁桃體床引發出血。

最佳可用證據無法證明 NSAID 增加出血風險。恢復中持續吞嚥是警訊，可能是扁桃體/腺樣體床出血的徵兆。

腺樣體切除/扁桃體切除現在廣泛作為門診手術，術後留觀 5 至 6 小時；嗎啡仍適合鎮痛。

## 扁桃體切除術：成人

與兒童相同，但注意：

- 成人術後通常更疼痛：在手術室給予嗎啡/長效鴉片類。考慮 Tramadol。
- 更常使用神經肌肉阻斷劑技術配合間歇性正壓換氣。深拔管可能更依賴氣道輔助。
- 偶爾患者以扁桃體周圍膿腫（quinsy）就診。現在通常以抗生素治療，稍後進行扁桃體切除術。若因氣道水腫必須引流，通常在局部浸潤下以注射器和大針頭抽吸膿液。

## 鼓室成形術（Myringoplasty）

### 手術基本資料

| 項目 | 內容 |
|------|------|
| 手術方式 | 以自體移植（通常為顳肌筋膜）重建穿孔鼓膜 |
| 手術時間 | 60 至 90 分鐘 |
| 疼痛程度 | ++ |
| 體位 | 仰臥，頭側傾，頭環，頭高位 |
| 失血量 | 微量 |
| 實用技術 | South-facing Rae 管或聲門上氣道（通常加強型）；自主換氣或間歇性正壓換氣 |

### 術中管理

確保手術期間避免咳嗽；局部麻醉噴霧至喉；如使用間歇性正壓換氣-鬆弛劑技術，監測神經肌肉阻斷。乾燥的手術野改善視野，但對鼓室成形術不如鎚骨切除術重要；頭高位和避免高血壓/心搏過速通常足夠。Remifentanil 輸注適合。常规止吐有用。

### 術後護理

需要時口服或靜脈 Acetaminophen 或 NSAID；可能需要嗎啡。需要時止吐。

### 特殊考量

笑氣擴散入中耳可能導致移植片浮起；避免或在個案結束前 20 分鐘停止。

## 鎚骨切除術/鼓室成形術（Stapedectomy/Tympanoplasty）

### 手術基本資料

| 項目 | 內容 |
|------|------|
| 手術方式 | 切除/重建損壞的中耳結構 |
| 手術時間 | 2 至 4 小時 |
| 疼痛程度 | ++/+++ |
| 體位 | 仰臥，頭側傾，頭環，頭高位 |
| 失血量 | 微量 |
| 實用技術 | South-facing Rae 管或聲門上氣道（通常加強型）。通常間歇性正壓換氣。常使用动脉導管 |

### 術前考量

檢查心血管疾病，因為這將限制可達到的低血壓程度。口服術前用藥選項包括苯二氮平類、β-受體阻斷劑和 Clonidine。

### 術中管理

若外科使用神經完整性監測，避免神經肌肉阻斷劑。

無血手術野使手術更精確。簡單措施包括：誘導前強效鴉片類；確保插管時避免咳嗽（喉局部麻醉噴霧有幫助）；頭高位減少靜脈壓。

進一步降低血壓的好處可達到（健康患者平均 50 至 60 mmHg）和心率（<60 bpm）。Remifentanil 輸注是實現此目標的理想選擇。替代方案是靜脈 Labetalol（複合 α/β-阻斷劑，5 mg 增量）或靜脈 β-阻斷劑（Metoprolol 1 mg 增量，Esmolol 輸注）加血管擴張劑（異氟烷，Hydralazine 5 mg 增量）。有心血管疾病或使用強效血管擴張劑時強烈建議动脉導管；頭高位進一步減少腦灌注壓力。

常規給予至少一種止吐藥。考慮 Prochlorperazine（與外科醫師討論眩暈風險）。

### 術後護理

24 至 48 小時規律止吐。需要時口服或靜脈/直腸 Acetaminophen 或 NSAID；可能需要嗎啡。

### 特殊考量

笑氣擴散入中耳可能干擾手術，雖然不如鼓室成形術重要。避免或在個案結束前 20 分鐘停止。

## 鼻腔手術（Nasal Cavity Surgery）

### 手術基本資料

| 項目 | 內容 |
|------|------|
| 手術方式 | 鼻中隔黏膜下切除、整型術、鼻甲切除術、息肉切除術、功能性內視鏡鼻竇手術 |
| 手術時間 | 20 至 100 分鐘 |
| 疼痛程度 | ++ |
| 體位 | 仰臥，頭環，頭高位 |

在息肉切除術時不要貼眼（視神經可能靠近，外科醫師需檢查眼球運動）。全靜脈麻醉和 Remifentanil 是理想選擇，可改善手術條件；目標控制血壓（健康個體平均 50 至 60 mmHg）。

拔管前抽吸咽部（特別是軟腭後方以防「驗屍官血塊」）。

### 術後護理

盡快讓患者坐起以減少出血。

### 特殊考量

保留靜脈針過夜，因為可能術後出血。

## 微喉鏡檢查（Microlaryngoscopy）

### 手術基本資料

| 項目 | 內容 |
|------|------|
| 手術方式 | 使用手術顯微鏡檢查喉（加切除/活檢；可能使用雷射） |
| 手術時間 | 10 至 30 分鐘 |
| 疼痛程度 | +/++ |
| 體位 | 仰臥，墊肩，頭過伸 |
| 失血量 | 無 |
| 實用技術 | 微喉鏡管和常規間歇性正壓換氣。全靜脈麻醉和經注射器系統的噴射換氣（氧氣加夾帶空氣）|

### 換氣技術選項

#### 自主換氣/窒息性氧合，氣道未固定

與外科醫師討論是否需看到喉功能（不常見）；需要自主換氣技術。高流量鼻導管和全靜脈麻醉技術允許不中斷的手術入路。氣道保護不良。

若要進行切除，血液/病原體的霧化是考量因素。

#### 微喉鏡管和常規間歇性正壓換氣

微喉鏡管是較長的氣管內管，內徑和外徑較小，高容量/低壓氣囊，通常有 4 號及以上尺寸。比傳統氣管內管允許更大的手術入路。

允許以吸入劑維持麻醉、常規間歇性正壓換氣策略和氣體交換及氣道壓力監測。防止血液或手術碎屑吸入，但遮擋約三分之一後側聲門的視野。

### 噴射換氣

微喉鏡結束時，可在自主換氣重新建立前繼續噴射換氣，或更常見停止並以替代氣道換氣直到自主換氣重新開始。

### 術前考量

患者常為老年人，通常吸菸者；心血管/呼吸系統問題常見。仔細評估氣道是否有阻塞證據。確保誘導前所有設備就緒，包括環甲膜切開套件，且外科醫師準備好在需要時進行緊急氣管切開術。

### 術中管理

如懷疑氣道阻塞，最初使用概述原則穩固氣道。插入環甲膜套管在局部麻醉下誘導前提供在完全阻塞事件中的換氣路由。

給予短效鴉片類（Alfentanil、Remifentanil）減弱高血壓反應。高劑量輸注可能防止需要神經肌肉鬆弛劑以獲得不動的手術野。

### 術後護理

需要時口服或靜脈/直腸 Acetaminophen 或 NSAID 鎮痛。可能因已受損氣道的水腫而發生術後喘鳴。Dexamethasone 8 至 12 mg 靜脈有時用於預防。考慮噴霧腎上腺素。

### 特殊考量

使用雷射需仔細計劃。氣道雷射手術有顯著的氣道火災風險，以及對患者和工作人員的附帶組織損傷/傷害。微喉鏡檢查可用於注射惰性物質（如矽膠）入癱瘓聲帶以改善發音，雖然過量可導致氣道阻塞。

## 氣管切開術（Tracheostomy）

### 手術基本資料

| 項目 | 內容 |
|------|------|
| 手術方式 | 經頸部切口插入氣管內管 |
| 手術時間 | 30 分鐘 |
| 疼痛程度 | ++ |
| 體位 | 仰臥，墊肩，頭環，頭高位 |
| 失血量 | 通常少量，雖然甲狀腺血管可能出血 |
| 實用技術 | 間歇性正壓換氣，氣管內管管徑「朝北」，術中改為氣管切開管 |

### 術前考量

通常為長期重症監護或氣道阻塞。重症監護患者几乎肯定已插管。如換氣困難且氧合關鍵，在手術室使用全靜脈麻醉設置重症監護換氣機，而非吸入劑。

如氣管切開術是為氣道阻塞，根據概述原則最初穩固氣道。誘導前確保所有設備準備就緒（包括環甲膜切開套件），且外科醫師準備好在需要時進行緊急氣管切開術。

### 術中管理

將氣管內管以膠帶固定以允許術中輕易移除，氣囊測試閥門易於觸及。鋪單前抽吸鼻胃管（如有）和清除口咽分泌物。

更換為氣管切開管前，預充氧 3 至 4 分鐘（必要時增加吸入劑），並檢查神經肌肉阻斷是否足夠。

在外科醫師切開氣管前，先將氣管內管氣囊洩氣，如此可重新充氣並在有問題時繼續換氣。將氣管內管緩慢退出至上部氣管（不要從氣管移除直至氣管切開管安全確定），並經無菌導管座將呼吸回路和二氧化碳圖連接到新的氣管切開管。

### 術後護理

定期抽吸新的氣管切開管（血液、分泌物）。濕化吸入氣體。恢復室以 Acetaminophen 或 NSAID 靜脈/直腸或嗎啡靜脈鎮痛。通常此後不需要太多鎮痛。

新的氣管切開管常產生長時間咳嗽——嗎啡、苯二氮平類或低劑量 Propofol 可用於控制。需要時止吐。

### 氣管切開管特點

- 內管（如 Shiley）：可移除清潔
- 可調法蘭：長度可調整以適應短氣管或深造口
- 套管內引導絲通道
- 開窗：允許患者說話（用手指堵住管腔並經管後壁孔呼氣）

## 喉切除術（Laryngectomy）

| 項目 | 內容 |
|------|------|
| 手術方式 | 切除喉（會厭和聲門）並建立氣管造口 |
| 手術時間 | 3 至 4 小時 |
| 疼痛程度 | +++ |
| 體位 | 仰臥，墊肩，頭環，頭高位 |
| 失血量 | 中等至大量；備血 2 單位 |
| 實用技術 | 間歇性正壓換氣，氣管內管管徑「朝北」，術中改為氣管切開管。动脉導管、導尿管、長期/複雜手術或心臟病時的中心靜脈壓管 |

### 術前考量

可能有某種程度的氣道阻塞。患者可能近期因確診接受過全身麻醉以引導氣道管理；如已有一段時間，要注意。

通常吸菸者；心血管/呼吸系統問題和營養不良常見。術前討論氣管切開術的影響（溝通、分泌物、咳嗽）。語言治療師將做大部分工作。

### 術中管理

誘導時插入細鼻胃餵養管並牢固固定（可縫至鼻中隔）。保暖毯和液體加溫器。

可觀的血量可能堆積在頸後方的單子下，可能直到個案結束都不明顯。中心靜脈壓入路，所有頸部管路阻礙手術；股靜脈最好，雖然肘窩或鎖骨下可用。

更換為氣管切開管時，見氣管切開術預防措施。

### 術後護理

重症監護是理想選擇。濕化和定期抽吸至關重要（血液、分泌物）。新的氣管切開管產生長時間咳嗽——嗎啡、苯二氮平類或低劑量 Propofol 可用於控制。

需要時口服嗎啡/鼻胃、口服或靜脈/直腸 Acetaminophen 或 NSAID 需要時鎮痛。鎮痛需求通常驚人地低。需要時止吐。

### 特殊考量

注意解剖過程中的空氣栓塞——早期檢測依賴呼氣末二氧化碳突然下降。

需要時口服或靜脈/直腸 Acetaminophen 或 NSAID，口服/靜脈嗎啡鎮痛。通常鎮痛需求驚人地低。需要時止吐。

### 特殊考量

注意解剖過程中的空氣栓塞——早期檢測依賴呼氣末二氧化碳突然下降。頸竇的手術操作可產生顯著心搏過緩。

## 腮腺切除術（Parotidectomy）

### 手術基本資料

| 項目 | 內容 |
|------|------|
| 手術方式 | 切除腮腺，通常保留面神經 |
| 手術時間 | 2 至 5 小時 |
| 疼痛程度 | ++/+++ |
| 體位 | 仰臥，頭環，頭側傾並適度過伸，頭高位 |
| 失血量 | 少量/中等，血型及交叉比對。惡性腫瘤時更大 |
| 實用技術 | South-facing Rae 管和通常間歇性正壓換氣，或加強型聲門上氣道和間歇性正壓換氣或自主換氣。在面神經周圍解剖時不使用神經肌肉阻斷劑 |

### 術前考量

檢查是否適合自主換氣——如老年、肥胖或呼吸系統疾病則不適合。檢查張口，特別是惡性腫瘤時。

### 術中管理

保暖毯、液體加溫器 +/- 導尿管。避免初始劑量後的神經肌肉阻斷（以神經刺激器檢查恢復）以允許外科醫師測試面神經。Remifentanil 輸注是理想選擇，可在不使用神經肌肉阻斷劑下實現間歇性正壓換氣並減少失血。

### 術後護理

為減少恢復室躁動/反彈性高血腫和高血壓的機會，持續低速率 Remifentanil 輸注或在手術結束前 15 至 20 分鐘給予嗎啡；保持頭高位，早期治療高血壓；Clonidine 達 2 mcg/kg 靜脈非常有用。需要時止吐。需要時嗎啡靜脈/口服、Acetaminophen 或 NSAID 口服/靜脈/直腸鎮痛。

### 特殊考量

外科醫師通常使用神經刺激器識別解剖過程中的面神經，可能希望保留同側眼睛暴露以監測反應。開始時建立大口徑靜脈通路，因為偶爾大量出血（特別是惡性腫瘤）。
