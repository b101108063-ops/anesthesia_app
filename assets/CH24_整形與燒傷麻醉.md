# Chapter 24: Plastic and Burns Surgery
# 整形與燒傷麻醉

*Source: book2.pdf, pages 668-689*

## 24.1 一般原則

整形與燒傷手術麻醉的複雜程度從常規到具有挑戰性都有。一些廣泛的程序（如游離皮瓣修復、顱頜面重建）可能涉及侵入性監測、廣泛失血和術後重症監護支持。

### 區域技術和鎮靜

| 技術 | 適用情況 |
|-----|----------|
| 局部浸潤 | 微小體表程序 |
| 廣泛浸潤 + IV鎮靜 | 不適合GA的重大體表程序（如皮膚腫瘤切除和移植） |
| 區域麻醉 | 上下肢手術特別適合 |

### 鎮靜技術

| 藥物 | 劑量/方法 |
|-----|-----------|
| Propofol | 0.5-1.0 mcg/mL TCI 或 10-15 mL/h的1%溶液 |
| 咪達唑侖 | 1-2 mg（補充用） |
| Remifentanil | 0.5-1.0 ng/mL TCI 或 5 mL/h（1 mg在20 mL溶液中） |

### 困難氣道

頭頸病理導致氣道困難的患者經常遇到。氣道困難可能由腫瘤、創傷、感染、先前手術、放療或疤痕引起的解剖變形引起。

**共享氣道：** 頭頸手術中常規。與外科醫生討論：
- 使用的管子類型
- 路線（口腔、鼻腔、轉換氣管切開術）
- 如何固定管子

### 患者訪問受限

| 問題 | 影響 |
|-----|------|
| 保溫 | 可能難以獲得足夠的體表面積來維持溫度 |
| 監測 | ECG導線、血氧計探頭和血壓袖帶可能難以適當放置 |
| 血管通路 | 將套管放置在手術區域之外；必要時使用股血管或腳部 |

### 長時間手術考量

| 考量 | 內容 |
|-----|------|
| 血管通路 | 確保管路放置不會干擾手術部位；侵入性动脉監測是理想的 |
| 失血 | 確保血液已X-matched；初始解剖通常是失血最多的時期 |
| 液體平衡 | 導尿管插入是必不可少的；仔細監測液體平衡 |
| 體溫 | 監測核心溫度；使用低FGFs、HME過濾器、加溫IV液體、保溫環境溫度 |
| 體位 | 確保結構（如頸椎和臂叢）不在應力位置 |
| VTE | 術中通常開始DVT預防；所有患者應有血栓栓塞壓縮襪和間歇性小腿壓迫 |
| 眼護理 | 輕輕粘貼和墊護眼睛；簡單眼膏可能有幫助 |
| ETT氣囊壓力 | 如果使用N₂O，氣囊壓力會逐漸增加；可能時在手術期間間隔重新檢查氣囊壓力 |

### 術後照護

| 考量 | 內容 |
|-----|------|
| 護理地點 | 外科醫生通常偏好患者返回整形病房 |
| ICU/HDU | 更密切的患者觀察、侵入性監測和定期血氣估計可能更容易實現 |

## 24.2 乳房手術

### 一般考量

英國現在乳房癌是最常見的癌症，過去25年發病率增加>20%。

### 術前考量

| 考量 | 內容 |
|-----|------|
| 焦慮 | 通常很高；在術前訪視時獲得患者信心，討論鎮痛，必要時開處方抗焦慮 |
| 新輔助治療 | 考慮任何新輔助治療的不良影響；最近化療的患者可能免疫抑制 |
| 重建程序 | 化療後放射治療、乳房大放射治療後的乳房切除術和乳房縮小手術增加失血風險 |

### 圍手術期考量

| 考量 | 內容 |
|-----|------|
| 氣道 | 短至中等程序LMA和SV通常合適；長時間程序、肥胖和誤吸風險患者使用插管和機械通氣 |
| 監測 | 大多數程序標準監測；長時間程序需要主動加溫和溫度測量 |
| 藍染料 | 警惕用於前哨淋巴結活檢的藍染料與過敏反應相關 |

### 區域鎮痛

| 技術 | 說明 |
|-----|------|
| 椎旁阻滯 | - |
| 胸椎硬膜外 | - |
| 筋膜平面（胸神經和前鋸肌平面）阻滯 | - |
| 肋間阻滯 | - |
| 膜內阻滯 | - |

### 術後考量

| 考量 | 內容 |
|-----|------|
| 輕微程序 | 可能適合門診手術 |
| 廣泛程序 | 術後可能需要HDU |

### 特殊考量

| 考量 | 內容 |
|-----|------|
| 既往手術 | 患者可能曾有乳房手術和腋窩清除；患側手臂靜脈穿刺應避免（感染和淋巴水腫風險） |
| 慢性疼痛 | 乳房手術後可能發生，發生率高達20-50% |
| 乳房重建 | 兩種主要方法：基於植入物或自體組織瓣重建 |

## 24.3 乳房縮小術

| 項目 | 內容 |
|-----|------|
| 程序 | 通過腺體切除縮小乳房尺寸；通常為雙側 |
| 時間 | 3小時 |
| 疼痛 | ++ |
| 體位 | 仰臥，30°頭高腳低，間歇deck-chair；手臂在板上或彎曲在臀部後面 |
| 失血 | 500 mL；G&S |
| 實用技術 | IPPV via ETT或LMA |

### 術前考量

| 考量 | 內容 |
|-----|------|
| 患者特點 | 通常fit；年齡20-40歲；許多外科醫生排除BMI >30的患者 |
| 貧血 | 檢查FBC和G&S；X-matching通常不需要 |
| 月經週期 | 時間不重要 |
| VTE預防 | 所有患者應接受DVT預防（彈力襪、每日LMWh） |

### 圍手術期考量

| 考量 | 內容 |
|-----|------|
| IPPV | 外科醫生常在手術期間對胸壁施加壓力；IPPV維持滿意的胸部擴張和控制PaCO₂ |
| 監測放置 | 將ECG电极放在患者背部；將患者躺在失血吸收墊上 |
| 體位護理 | 確保對稱；確認血管套管固定良好並墊好 |
| Deck-chair位置 | 手術期間多次短暫放置以檢查乳房對稱；需要護理氣道、血管通路、監測和CPP |
| 失血 | 取決於外科技術；稀釋腎上腺素LA浸潤有助於減少失血 |

### 術後考量

| 考量 | 內容 |
|-----|------|
| 疼痛 | 雙側乳房縮小不會引起顯著術後疼痛；手術結束時給予嗎啡劑量，然後 regular簡單鎮痛藥和NSAID通常足夠 |
| 血腫形成 | 早期並發症；偶爾乳頭灌注可能受損需要減壓 |
| 輸血 | <5%患者需要；這個年輕患者群體輕度Hb下降耐受良好 |

## 24.4 乳房增大術

| 項目 | 內容 |
|-----|------|
| 程序 | 雙側或單側增大乳房尺寸 |
| 時間 | 90分鐘 |
| 疼痛 | ++/+++ |
| 體位 | 仰臥，30°頭高腳低 |
| 失血 | 最少 |
| 實用技術 | SV或IPPV via LMA |

### 術前考量

| 適應症 | 說明 |
|--------|------|
| 化療後重建 | - |
| 乳房不對稱矫正 | - |
| 美容雙側增大 | - |

### 術後考量

| 考量 | 內容 |
|-----|------|
| 疼痛 | 可能與植入物大小相關；大植入物引起更多組織拉伸和術後疼痛 |
| 血腫 | 可能需要早期返回手術室 |

## 24.5 游離皮瓣手術

| 項目 | 內容 |
|-----|------|
| 程序 | 從供體部位轉移組織並進行顯微外科吻合至遠端接受部位 |
| 時間 | 可變，取決於程序，通常6-8小時或更長 |
| 疼痛 | ++/+++ |
| 體位 | 可變；通常仰臥；可能需要手術期間體位改變 |
| 失血 | 可變；G&S至X-match 4-6單位 |
| 實用技術 | ETT + IPPV；arterial線；CO監測；尿導管；患者加溫 |

### 生理學目標

優化移植皮瓣的循環流量。管理主要基於第一原理和Hagen-Poiseuille方程應用。

### 變量控制

| 變量 | 控制方法 |
|-----|---------|
| 壓力梯度 | 輸液和血管加壓素 |
| 粘度 | 與Hct密切相關；瞄準30-35% |
| 血管半徑 | 通過麻醉和手術技術影響 |

### 圍手術期考量

| 考量 | 內容 |
|-----|------|
| 誘導和維持 | Propofol降低SVR，快速代謝，是止吐藥；避免術後顫抖；可能通過避免揮發性對紅細胞膜剛度的作用更有利於微循環 |
| 吸入維持 | Isoflurane維持微循環血流；Sevoflurane可能減弱缺血-再灌注損傷；Desflurane有更快消退 |
| 阿片類藥物 | Remifentanil提供術中鎮痛和顯著血管擴張控制血壓；否定對NMBAs的需要 |
| 液體管理 | 瞄準正常血容量血液稀釋在SVR降低的患者中；游離皮瓣的破坏淋巴系統增加間質水腫風險；游離皮瓣中過量血漿擴張可中斷內皮糖萼層與水腫和皮瓣失敗相關 |
| 血管收縮劑 | 在初始解剖階段控制性低血壓可減少手術失血；然而，在皮瓣吻合期間瞄準恢復正常血壓以確保通過組織的充足灌注壓力 |

### 術後考量

| 考量 | 內容 |
|-----|------|
| 甦醒 | 瞄準平穩甦醒，避免對縫合線的任何張力 |
| 血管收縮 | 冷、疼痛、循環量低、低血壓和低碳酸血症引起的血管收縮將威脅皮瓣；處理並排除低體溫並考慮用加壓空氣加溫設備、0.5-3 mg/kg IV曲馬多或20-50 mg IV哌替啶治療術後顫抖 |
| 鎮痛 | 除了區域麻醉外，口服阿片類藥物通常足夠；吞嚥受損（如頭頸部皮瓣）考慮IV PCA |

### 特殊考量

| 考量 | 內容 |
|-----|------|
| 指/肢體再植入 | 游離皮瓣處理；臂叢導管有助於管理再植入指（有助於持續血管擴張肢體） |
| 帶蒂皮瓣 | 構建當動靜脈連接保持完整時；需要與優化皮瓣循環相同的麻醉原則 |

## 24.6 抽脂術

| 項目 | 內容 |
|-----|------|
| 程序 | 通過小皮膚切口和專用鈍頭套管真空吸引皮下脂肪 |
| 時間 | 可變 30-90分鐘 |
| 疼痛 | + |
| 體位 | 可變，取決於部位；通常仰臥 |
| 失血 | 浸潤液體吸引量的1% |
| 實用技術 | 局部浸潤配合IV鎮靜/LMA和SV |

### 技術類型

| 技術 | 說明 |
|-----|------|
| 超濕技術 | 1 mL浸潤/1 mL預期吸引量 |
| 腫脹技術 | 大量LA/腎上腺素浸潤產生組織腫脹；作為門診技術開發，無需額外麻醉或鎮靜 |

### 圍手術期考量

| 考量 | 內容 |
|-----|------|
| 失血 | 取決於使用的LA/腎上腺素浸潤量；超濕和腫脹技術為吸引量的7% |
| 液體替代 | 將吸引量1:1與IV晶體替代；對心臟或腎衰竭患者謹慎 |
| 利多卡因劑量 | 限制總劑量至6 mg以避免毒性；傳統利多卡因/腎上腺素安全劑量為7 mg/kg |

## 24.7 皮膚移植和燒傷重建手術

| 項目 | 內容 |
|-----|------|
| 程序 |，游離皮膚移植應用於清創後的手術創建的裸露表面或肉芽創傷 |
| 時間 | 可變 30分鐘至2小時 |
| 疼痛 | ++/+++（特別是供體部位） |
| 體位 | 可變；取決於要移植的區域；通常仰臥 |
| 失血 | 簡單移植無；廣泛清創和燒傷移植可能需要6-8單位 |
| 實用技術 | GA/LMA和SV（如果大腿供體部位則配合大腿外側皮神經或髂筋膜阻滯）；下肢手術用脊髓 |

### 皮膚移植類型

| 類型 | 說明 |
|-----|------|
| 全層皮膚移植（FT SG） | 由表皮和真皮組成；用於小區域；可用SC LA浸潤29G針頭收縮 |
| 裂層皮膚移植（SSG） | 由表皮和可變部分真皮組成；比FT SG使用更廣泛；用皮膚移植刀或電動真皮刀收縮 |

### 急性燒傷手術

| 考量 | 內容 |
|-----|------|
| 氣道評估 | 插管幾乎對所有燒傷清創都需要；面部燒傷和餵養管的存在可能使面罩通氣困難 |
| 通氣 | 使用肺保護性通氣；通氣需求將很高（高代謝率）；可能必須容忍高碳酸血症 |
| 失血 | 確保6-8單位紅細胞X-matched；去除了的組織自由出血 |
| 體溫控制 | 大面積暴露體表面積將通過輻射和蒸發快速散熱；使用所有可用方法保溫 |
| 監測 | 實用主義；动脉線有助於血壓測量和血液採樣；中心靜脈線提供可靠靜脈通路 |
| 琥珀膽鹼 | 僅在燒傷後前48小時內使用；大量K⁺釋放可能導致心臟驟停 |

### 慢性燒傷手術

| 考量 | 內容 |
|-----|------|
| 氣道 | 面部和頸部燒傷後疤痕可能導致張口和頸部活動的重大限制；考慮AT OI；LMA可能是解剖學上具有挑戰性的氣道的有用救援設備 |
| 訪問 | 常規靜脈和动脉穿刺部位的疤痕使穿刺困難；超聲可能有幫助 |
| 體位 | 疤痕可能阻止在手術台上的常規體位；仔細注意墊壓力區域和支撐肢體和頭部 |
| 心理 | 患者通常在最初受傷後有長時間創傷性住院經歷；創傷後應激障礙常見 |
