# Chapter 26: Bariatric Surgery
# 減肥手術麻醉

*Source: book2.pdf, pages 706-715*

---

## 26.1 簡介

減肥手術描述了對肥胖患者進行的各種程序。減肥通過以下方式實現：
- 用胃束帶縮小胃部
- 移除部分胃部
- 繞過部分小腸道

### 麻醉考量

在為減肥手術進行麻醉時，應考慮第2章討論的圍手術期因素。

### 團隊設置

減肥手術應在多學科團隊環境中進行，包括：
- 嚴格的術前評估
- 術中路徑
- 術後位置和管理的考慮

---

## 26.2 手術適應症

### NICE指南2014

減肥手術是以下人群的治療選擇：

**體重指數（BMI）標準：**
- BMI ≥40 kg/m²
- 或 BMI 35-40 kg/m² 並有其他可通過減肥改善的重大疾病（如2型糖尿病或高血壓）

**其他標準：**
- 已嘗試所有適當的非手術措施，但未達到或維持臨床上有益的體重減輕
- 正在接受或將要接受第3層級服務的評估
- 通常適合麻醉和手術
- 承諾長期的術後隨訪

### 2014年新增的重點

**近期發病的2型糖尿病患者：**
- BMI ≥35 kg/m² 的患者應提供快速減肥手術評估
- BMI 30-34.9 kg/m² 的患者應考慮評估（亞洲血統患者可能在較低BMI時考慮）

## 26.3 減肥手術風險評分

### 肥胖手術死亡率風險評分（OS-MRS）

| 風險因素 | 分數 |
|---------|------|
| 年齡 >45歲 | 1 |
| 高血壓 | 1 |
| 男性 | 1 |
| VTE風險因素* | 1 |
| BMI ≥50 kg/m² | 1 |

*VTE風險因素包括：既往VTE、肺高壓、術前腔靜脈過濾器或肥胖導致的通氣不足

### 風險分組

| 風險分組（分數） | 術後死亡率 |
|-----------------|------------|
| A級（0-1分） | 0.2% |
| B級（2-3分） | 1.1% |
| C級（4-5分） | 2.4% |

## 26.4 胃內球囊插入/移除

| 項目 | 內容 |
|-----|------|
| 程序 | 通過胃鏡將700 mL矽膠球囊放入胃中，充滿700 mL 0.9%氯化鈉（用亞甲藍染色） |
| 時間 | 15-30分鐘 |
| 疼痛 | 無 |
| 體位 | 左側臥或坐起 |
| 失血 | 無 |
| 實用技術 | IV鎮靜、局部麻醉或GA、ETT |

### 適應症

| 患者類型 | 說明 |
|---------|------|
| 低BMI患者 | BMI 25-35 kg/m²（通常低風險） |
| 極高BMI患者 | 通常 >60-70 kg/m²（通常有許多顯著合併症） |

### 特殊考量

| 考量 | 內容 |
|-----|------|
| 嘔心 | 術後立即嘔心常見；應給予止吐藥並開處方帶回家 |
| 移除 | 有時在技術上困難且可能花費更長時間 |
| 門診 | 插入和移除通篇可在門診進行 |

## 26.5 胃束帶術

| 項目 | 內容 |
|-----|------|
| 程序 | 在胃上部放置矽膠可調節束帶，形成一個小囊；通過皮下注射端口連接管道 |
| 時間 | 45-90分鐘 |
| 疼痛 | + |
| 體位 | 仰臥，頭高腳低 |
| 失血 | 最少 |
| 實用技術 | GA、ETT、IPPV（配合PEEP） |

### 術前考量

| 考量 | 內容 |
|-----|------|
| 程序特點 | 相對簡單的腹腔鏡程序，死亡率非常低 |
| 患者選擇 | 通常用於較低BMI和較低風險患者；但一些中心對大多數病例使用束帶 |

### 圍手術期考量

| 考量 | 內容 |
|-----|------|
| 設備 | 確保設備（如手術台、hover床墊）適合體重 |
| IV通路 | 插入兩個IV套管 |
| 體位 | 極端小心以避免壓力或懸垂組織造成的損傷；防止頭高位時患者滑落 |
| 監測 | 標準圍手術期監測；上臂過大或形狀不合適用前臂血壓袖帶 |
| 預充氧 | 在頭高位充分預充氧 |
| 氣道 | 面罩通氣可能困難；預期呼吸暫停期間快速去飽和；氣管內插管和通氣是強制性的 |
| 麻醉技術 | 使用短效藥物如七氟醚；TIVA和propofol是良好技術 |

### 術後考量

| 考量 | 內容 |
|-----|------|
| 體位 | 在恢復室中護理坐起 |
| 阿片類藥物 | 通常需要；限制術中阿片類藥物並在恢復室滴定劑量 |
| HDU | 大多數患者可安全管理；但有顯著OSA患者應考慮 |
| 抗血栓 | 按當地方案 |

## 26.6 胃繞道術

| 項目 | 內容 |
|-----|------|
| 程序 | Roux-en-Y胃繞道；幾乎總是腹腔鏡 |
| 時間 | 90-200分鐘 |
| 疼痛 | ++ |
| 體位 | 仰臥，頭高腳低 |
| 失血 | >500 mL（偶爾更多由於脾損傷或胃滲血）；需要G&S |
| 實用技術 | GA、ETT、IPPV（配合PEEP） |

### 程序特點

**組分：**
- 小腸吻合
- Roux肢形成
- 胃囊形成
- 胃空腸吻合

### Bougie/大鼻胃管使用

- 外科醫生可能要求在囊形成期間經口插入大（通常34 Fr） bougie
- 這有助於識別囊並防止食道被縫合
- 在胃空腸吻合期間將bougie推入Roux肢

### 特殊考量

| 考量 | 內容 |
|-----|------|
| 嘔心 | 術後立即重要；推薦兩種止吐藥 |
| 術後CPAP | 安全；沒有證據顯示對胃吻合口造成損傷 |

### 術後併發症

| 併發症 | 體徵 |
|--------|------|
| 吻合口漏 | 心動過速（術後心動過速是漏直到證明其他）、過度疼痛、進食時疼痛 |
| 出血 | 嚴重出血的體徵；24小時內可能出現黑便或嘔血 |

## 26.7 袖狀胃切除術

| 項目 | 內容 |
|-----|------|
| 程序 | 通過縫合將胃縮小至約25%原始大小；移除沿大彎的大部分胃 |
| 時間 | 90-150分鐘 |
| 疼痛 | ++ |
| 體位 | 仰臥，頭高腳低 |
| 失血 | >500 mL（偶爾更多由於胃縫合線滲血）；需要G&S |
| 實用技術 | GA、ETT、IPPV（配合PEEP） |

### 程序特點

- 越來越常見的減肥程序
- 經常在非常高風險患者中進行（作為胃繞道的替代）
- 比胃繞道更簡單、更快的手術

### Bougie使用

- 外科醫生可能要求在程序期間經口插入大（通常34 Fr） bougie
- 這允許外科醫生沿著bougie縫合，識別解剖並防止食道被縫合

### 特殊考量

| 考量 | 內容 |
|-----|------|
| 嘔心 | 術後立即重要；推薦兩種止吐藥 |
| 避免低血壓 | 在胃縫合後避免低血壓；有助於識別縫合線出血 |
| 術後CPAP | 安全 |
