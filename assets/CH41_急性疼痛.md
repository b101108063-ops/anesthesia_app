## 總論（Introduction）

### 急性疼痛管理的重要性

嚴重術後疼痛和手術壓力反應會導致：
- 心血管：心跳過速、高血壓、心肌缺氧
- 呼吸：橫膈固定、肺容積減少、肺部感染
- 胃腸：胃排空延遲、腸道蠕動減少
- 代謝：皮質醇、血糖升高
- 慢性疼痛：急性疼痛患者更可能發展為慢性疼痛

### 疼痛評估工具

| 工具 | 說明 |
|------|------|
| 語言評分量表 | 輕度、中度、重度 |
| 數字評分量表 (NRS) | 0-10 分 |
| 視覺類比量表 (VAS) | 100 mm 線 |

## 止痛藥物（Analgesic Drugs）

### Paracetamol

| 特性 | 說明 |
|------|------|
| 作用機制 | 中樞前列腺素、5-羥色胺、阿片類、一氧化氮、內源性大麻素途徑 |
| 劑量 | 成人 4 g/天 |
| 給藥途徑 | 口服、直腸、靜脈 |

### NSAIDs

| 特性 | 說明 |
|------|------|
| 作用機制 | COX 抑制，减少前列腺素合成 |
| opioid 節省效應 | 20-40% |
| 副作用 | 胃腸潰瘍、腎功能損害、心血管事件 |

### 鴉片類藥物

| 藥物 | 等效劑量 (IM/IV) | 等效劑量 (口服) |
|------|-------------------|-------------------|
| Morphine | 10 mg | 25 mg |
| Diamorphine | 5 mg | - |
| Fentanyl | 0.15-0.2 mg | - |
| Pethidine | 100 mg | 250 mg |
| Codeine | - | 175 mg |
| Tramadol | 100 mg | 100 mg |

### 鴉片類副作用

- 呼吸抑制
- 鎮靜
- 噁心嘔吐
- 便秘
- 瘙癢

## 患者自控止痛（PCA）

### 常用方案

| 設定 | 藥物和劑量 |
|------|-------------|
| 追加劑量 | Morphine 1 mg、Fentanyl 20 mcg、Diamorphine 0.5 mg |
| 鎖定間隔 | 5 分鐘 |
| 背景輸注 | 通常為 0 mg/h |

### 鎮靜評分量表

| 分數 | 狀態 |
|------|------|
| 0 | 完全清醒 |
| 1 | 輕度嗜睡 |
| 2 | 中度嗜睡 |
| 3 | 重度嗜睡，難以喚醒 |
| 4 | 睡眠狀態但易喚醒 |

## 硬膜外止痛（Epidural Analgesia）

### 適應症

- 胸腹部大手術
- 肺部疾病患者
- 下肢血管手術

### 常用配方

| 配方 | 說明 |
|------|------|
| Bupivacaine 0.125% + Fentanyl | 5 mcg/mL |
| Bupivacaine 0.1% + Fentanyl | 5 mcg/mL |
| 輸注速率 | 8-15 mL/h（成人）|

### 併發症

| 併發症 | 發生率 | 處理 |
|---------|---------|------|
| 硬膜穿刺後頭痛 | 16-86% | 臥床休息、止痛、血液貼片 |
| 神經損傷 | 0.016-0.56% | 立即神經學評估 |
| 呼吸抑制 | 0.13-0.4% | 減少opioid濃度 |
| 低血壓 | 3-30% | 輸液、血管收縮劑 |

## 非鴉片類輔助止痛藥

### 常用藥物

| 藥物 | 作用 | 說明 |
|------|------|------|
| Gabapentin/Pregabalin | 抑制神經元鈣離子通道 | 術後使用有爭議 |
| Ketamine | NMDA 受體拮抗劑 | 減少疼痛強度達 25% |
| Lidocaine | 抗心律不整、抗炎、抗痛覺過敏 | 靜脈輸注對胃腸手術有益 |
| Clonidine/Dexmedetomidine | α2-腎上腺素受體激動劑 | 鎮靜、抗焦慮 |

## 特定患者群體

### 長期鴉片類使用患者

| 考量 | 說明 |
|------|------|
| 耐受性 | 對opioid敏感性降低 |
| 身體依賴 | 停藥時戒斷症狀 |
| 劑量需求 | 通常比一般患者高 3 倍 |

### 物質濫用障礙患者

- 心理和行為特徵
- 藥物依賴和戒斷風險
- 多模式止痛策略
- 區域麻醉技術
